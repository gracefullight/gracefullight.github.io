Prototype

  • 4.3.5 prototype мӮ¬м–‘м„ңмқҳ лӮҙмҡ©мқҖ лӢӨмқҢкіј к°ҷлӢӨ.

лӢӨлҘё к°қмІҙм—җ кіөмң  н”„лЎңнҚјнӢ°лҘј м ңкіөн•ҳлҠ” к°қмІҙ

  • NOTE: мғқм„ұмһҗк°Җ к°қмІҙлҘј л§Ңл“Ө л•Ң к·ё к°қмІҙлҠ” м•”л¬өм ҒмңјлЎң мғқм„ұмһҗмқҳ prototype мҶҚм„ұмқ„ м°ёмЎ°н•ңлӢӨ.
  • мғқм„ұмһҗмқҳ prototype н”„лЎңнҚјнӢ°лҠ” constructor.prototype м—җ мқҳн•ҙ м°ёмЎ°лҗ  мҲҳ мһҲкі  к°қмІҙмқҳ н”„лЎңнҶ нғҖмһ…м—җ м¶”к°Җлҗң н”„лЎңнҚјнӢ°лҠ” мғҒмҶҚмқ„ нҶөн•ҙ н”„лЎңнҶ нғҖмһ…мқ„ кіөмң н•ҳлҠ” лӘЁл“  к°қмІҙм—җ кіөмң лҗңлӢӨ.
  • Object.create н•ЁмҲҳлҘј мӮ¬мҡ©н•ҳм—¬ лӘ…мӢңм ҒмңјлЎң м§Җм •лҗң н”„лЎңнҶ нғҖмһ…мңјлЎң к°қмІҙлҘј л§Ңл“Ө мҲҳ мһҲлӢӨ.

new мӢӨн–ү лӢЁкі„вҖӢ

  • 9.2.2 [[Construct]] мӮ¬м–‘м„ңмқҳ лӮҙмҡ©мқҖ лӢӨмқҢкіј к°ҷлӢӨ.

ECMAScriptмқҳ function к°қмІҙ Fм—җ лҢҖн•ң лӮҙл¶Җ л©”мҶҢл“ң [[Construct]] к°Җ argumentsList мҷҖ newTarget нҢҢлқјлҜён„°мҷҖ н•Ёк»ҳ нҳём¶ңлҗңлӢӨ. argumentsList лҠ” л№Ҳ List к°Җ к°ҖлҠҘн•ҳлӢӨ.

  1. Assert: FлҠ” ECMAScript function objectмқҙлӢӨ.
  2. Assert: newTargetмқҖ ObjectмқҙлӢӨ.
  3. Let callerContextлҘј нҳ„мһ¬ execution contextлЎң м„Өм •н•ңлӢӨ.
  4. kindлҠ” F.[[ConstructorKind]]лЎң м„Өм •н•ңлӢӨ.
  5. if kindк°Җ baseлқјл©ҙ
    • a. thisArgumentлҘј newTargetмқҳ prototypeмқ„ к°Җм§ҖлҠ” мғҲлЎңмҡҙ к°қмІҙлЎң м„Өм •н•ңлӢӨ.
  6. calleeContextлҠ” PrepareForOrdinaryCall(F, newTarget)лЎң м„Өм •н•ңлӢӨ.
  7. Assert: calleeContextлҠ” мқҙм ң нҳ„мһ¬ мӢӨн–ү мӨ‘мқё execution contextмқҙлӢӨ.
  8. If kindк°Җ baseлқјл©ҙ, OrdinaryCallBindThis(F, calleeContext, thisArgument)лЎң this л°”мқёл”©мқ„ н•ңлӢӨ.
  9. constructorEnvлҘј calleeContextмқҳ LexicalEnvironmentлЎң м„Өм •н•ңлӢӨ.
  10. envRecлҘј constructorEnvмқҳ EnvironmentRecordлЎң м„Өм •н•ңлӢӨ.
  11. н•ЁмҲҳ FлҘј argumentListлЎң мӢӨн–үн•ң кІ°кіјлҘј resultлЎң м„Өм •н•ңлӢӨ.
  12. calleeContextлҘј execution context stackм—җм„ң мӮӯм ңн•ҳкі  callerContextлҘј нҳ„мһ¬ execution contextлЎң ліөкө¬н•ңлӢӨ.
  13. If result.[[Type]]мқҙ returnмқҙл©ҙ
    • a. result.[[Value]]к°Җ к°қмІҙмқҙл©ҙ result.[[Value]]лҘј л°ҳнҷҳн•ңлӢӨ.
    • b. If kindк°Җ baseмқҙл©ҙ thisArgumentлҘј л°ҳнҷҳн•ңлӢӨ.
    • c. If result.[[Value]] к°Җ undefined к°Җ м•„лӢҲлқјл©ҙ TypeErrorлҘј л°ңмғқмӢңнӮЁлӢӨ.
  14. м•„лӢҲлқјл©ҙ, resultлҘј л°ҳнҷҳн•ңлӢӨ.
  15. EnviromnetRecordмқҳ thisлҘј л°ҳнҷҳн•ңлӢӨ.

5.aвҖӢ

  • м—¬кё°м„ң thisлҘј newTargetмқҳ prototypeмқ„ к°Җм§ҖлҠ” к°қмІҙлЎң мғқм„ұн•ңлӢӨ.
  • мқҙ мӮ¬м–‘мңјлЎң н”„лЎңнҶ нғҖмһ… мІҙмқёмқҙ мӢӨн–үлҗңлӢӨ.

8вҖӢ

  • мң„м—җм„ң мғқм„ұлҗң к°қмІҙлҘј thisлЎң м„Өм •н•ңлӢӨ.

11вҖӢ

  • argumentsListлҘј нҢҢлқјлҜён„°лЎң л„ҳкёҙ мғқм„ұмһҗ н•ЁмҲҳ FлЎң мғҲлЎңмҡҙ к°қмІҙлҘј мҙҲкё°нҷ”н•ңлӢӨ.

15вҖӢ

  • мҙҲкё°нҷ”лҗң thisлҘј н•ЁмҲҳ нҳём¶ңмқҳ кІ°кіјлЎң л°ҳнҷҳн•ңлӢӨ.
  • мқҙ мӮ¬м–‘мңјлЎң new нӮӨмӣҢл“ңлЎң мӢӨн–үмӢң return this; кө¬л¬ёмқҙ м—ҶлҠ”лҚ°лҸ„ мқёмҠӨн„ҙмҠӨк°Җ л°ҳнҷҳлҗңлӢӨ.

кё°нғҖ нҠ№м„ұвҖӢ

  • н•ЁмҲҳ к°қмІҙмқҳ prototypeмқҖВ Object.prototypeм—җ лҢҖн•ң delegation linkмҷҖВ constructorВ мҶҚм„ұмқ„ к°Җм§„ к°қмІҙмқҳ м°ёмЎ°лҘј к°Җм§„лӢӨ.
  • constructorВ мҶҚм„ұмқҖ н•ЁмҲҳ к°қмІҙм—җ лҢҖн•ң м—ӯм°ёмЎ°лҘј к°Җм§Җкі  мһҲлӢӨ. (мһ¬к·Җм Ғ)
  • н•ЁмҲҳлҠ”В Function.prototypeВ м—җ лҢҖн•ң лҚёлҰ¬кІҢмқҙм…ҳ л§ҒнҒ¬лҘј к°Җм§Җкі  мһҲм–ҙВ applyмҷҖВ callмқ„ мғҒмҶҚл°ӣлҠ”лӢӨ.

  • н•ЁмҲҳ к°қмІҙл§Ңмқҙ к°Җм§Җкі  мһҲлҠ” prototype н”„лЎңнҚјнӢ°лҠ” мғқм„ұмһҗ н•ЁмҲҳк°Җ мғқм„ұн•  мқёмҠӨн„ҙмҠӨмқҳ н”„лЎңнҶ нғҖмһ…мқ„ к°ҖлҰ¬нӮЁлӢӨ.
