The Deutsch Problem​

  • Given a boolean function f:{0,1}n→{0,1}f:\{0,1\}^{n} \to \{0,1\}f:{0,1}n→{0,1}, determine if fff is constant or balanced.
  • Constant: same output for all inputs.
  • Balanced: outpus 0 for half the inputs and 1 for the other half.

  1. Prepare ∣0⟩∣1⟩\ket{0}\ket{1}∣0⟩∣1⟩.
  2. Apply HHH to both qubits →∣+⟩∣−⟩\to \ket{+}\ket{-}→∣+⟩∣−⟩.
  3. Apply oracle UfU_fUf​.
  4. Phase kickback encodes fff in the input phase.
  5. Apply HHH to input qubit, then measure.
  • One query to the oracle is sufficient to determine if fff is constant or balanced.
  • if measure 0: fff is constant.
  • if measure 1: fff is balanced.

Deustch-Jozsa Algorithm​

The direct generalization for any nnn-bit boolean function.

  1. Input register nnn qubits: Initialize qubits in ∣0⟩\ket{0}∣0⟩ and apply HHH gate to each one.
  2. Scratch Qubit 111 qubit: Initialize in ∣1⟩\ket{1}∣1⟩ with XXX and then apply an HHH gate.
  3. Oracle: Apply UfU_fUf​ to the input and scratch registers (All qubits).
  4. Final Hadamards: Apply HHH to input qubits.
  5. Measurement: Measure the input register.
  • if all qubits returned 000: fff is constant.
  • if any qubit returned 111: fff is balanced.

How it works​

  • The initial state is

∣00⋯0⟩∣1⟩\ket{00 \cdots 0} \ket{1}∣00⋯0⟩∣1⟩

  • After applying HHH gates to the input and scratch registers, we get

∣00⋯0⟩→12n(∣00⋯0⟩+∣00⋯1⟩+⋯+∣11⋯1⟩)\ket{00 \cdots 0} \rightarrow \frac{1}{\sqrt{2^n}} \left( \ket{00 \cdots 0} + \ket{00 \cdots 1} + \cdots + \ket{11 \cdots 1} \right)∣00⋯0⟩→2n​1​(∣00⋯0⟩+∣00⋯1⟩+⋯+∣11⋯1⟩)

  • The input register is in a superposition of a computational states containing all possible inputs to nnn-bit string.
  • The last qubit hasn't changed from n=1n = 1n=1 case, so it is in the state ∣−⟩\ket{-}∣−⟩.
    • H∣1⟩=∣−⟩H\ket{1} = \ket{-}H∣1⟩=∣−⟩
  • The definition of the oracle was generic for any bit string: Uf∣x⟩∣−⟩=(−1)f(x)∣x⟩∣−⟩U_f \ket{x}\ket{-} = (-1)^{f(x)} \ket{x} \ket{-}Uf​∣x⟩∣−⟩=(−1)f(x)∣x⟩∣−⟩
  • The phase-kickback puts a phase in front of each term in the input register that depends on the output of the function fff: 12n(  (−1)f(00…0)∣00…0⟩+⋯+(−1)f(11…1)∣11…1⟩  )\frac{1}{\sqrt{2^n}}\big(\;(-1)^{f(00\ldots0)}|00\ldots 0\rangle+ \cdots +(-1)^{f(11\ldots1)}|11\ldots 1\rangle\;\big)2n​1​((−1)f(00…0)∣00…0⟩+⋯+(−1)f(11…1)∣11…1⟩)
    • The scratch qubit remains in ∣−⟩\ket{-}∣−⟩, so we can ignore it for the rest of the algorithm.
  • Constant case
    • if fff is constant, then all the phases are the same, either +1+1+1 or −1-1−1:
    • f(00⋯0)=f(00⋯1)=⋯=f(11⋯1)=0f(00 \cdots 0) = f(00 \cdots 1) = \cdots = f(11 \cdots 1) = 0f(00⋯0)=f(00⋯1)=⋯=f(11⋯1)=0
    • f(00⋯0)=f(00⋯1)=⋯=f(11⋯1)=1f(00 \cdots 0) = f(00 \cdots 1) = \cdots = f(11 \cdots 1) = 1f(00⋯0)=f(00⋯1)=⋯=f(11⋯1)=1
    • The phse in front of every computational state is the same, Either +1+1+1 or −1-1−1.
    • Before the second application of the HHH gates, the state of the input register is:
    • ±12n(∣00⋯0⟩+∣00⋯1⟩+⋯+∣11⋯1⟩)\pm \frac{1}{\sqrt{2^n}} \left( \ket{00 \cdots 0} + \ket{00 \cdots 1} + \cdots + \ket{11 \cdots 1} \right)±2n​1​(∣00⋯0⟩+∣00⋯1⟩+⋯+∣11⋯1⟩)
    • In measurement, in either case, the probability to obtain P(00⋯0)=1P(00 \cdots 0) = 1P(00⋯0)=1
    • A constant function deterministically returns all zeros with a single query.
  • Balanced case
    • we are promised the function is either constant or balanced, so there are equal number of +1+1+1 and −1-1−1 phases. so we don't need to consider this case.
    • If the measurement produces anything but all zeros, we know with certainty the function is not constant, so it must be balanced.
    • There are a lot of balanced function, but half the terms in the superposition will exactly have a −1-1−1 phase.
    • It's clearly orthogonal to the state with all ones in superposition.
    • Appllying HHH's will change the state to some other superpostiion, or perhaps a unique computational state.
    • But, orthogonality to ∣00⋯0⟩\ket{00 \cdots 0}∣00⋯0⟩ must remain.
    • A balenced function deterministically returns a state with at least one entry as 111, with a single query.

One quantum query vs 2n−1+12^{n-1} + 12n−1+1 classical queries. It is an algorithm that puts all inputs into superposition at once, encodes the function values as phases, and then uses interference to distinguish between constant and balanced functions.

Buildling Oracles​

Multi-Controlled and Anti-Controlled Gates​

  • build UfU_fUf​ that applies XXX to the output qubit exactly when f(x)=1f(x) = 1f(x)=1.
    • if n=2n=2n=2, the multi-controlled XXX gate is a Toffoli gate.
    • x=11x=11x=11 is the only input that gives f(x)=1f(x) = 1f(x)=1, so we can use a Toffoli gate with controls on the first two qubits and target on the output qubit.
      • This is because the Toffoli gate will flip the output qubit if and only if both control qubits are ∣1⟩\ket{1}∣1⟩, which corresponds to the input x=11x=11x=11.
    • if x=00x=00x=00 is the only input that gives f(x)=1f(x) = 1f(x)=1, we can use an anti-controlled Toffoli gate, which applies XXX to the target qubit if both control qubits are ∣0⟩\ket{0}∣0⟩.
      • This is because the anti-controlled Toffoli gate will flip the output qubit if and only if both control qubits are ∣0⟩\ket{0}∣0⟩, which corresponds to the input x=00x=00x=00.
      • Applying XXX gates to the two control qubits, then applying a Toffoli gate, and then applying XXX gates again to the control qubits will effectively create an anti-controlled Toffoli gate.

  • A multi-controlled XXX gates (CnXC^nXCnX) flips the target only all control qubits are ∣1⟩\ket{1}∣1⟩.
  • To target a specific input xxx:
    • Place XXX gates on each qubit iii where xi=0x_i = 0xi​=0. (anti-control)
    • Apply CnXC^nXCnX.
    • Undo the XXX gates.
  • For nnn qubits, it can be generalized to perform an XXX gate (or any UUU) with nnn control qubits, requireing n−1n-1n−1 extra scratch qubits and 2(n−1)2(n-1)2(n−1) Toffoli gates.
  • Whenever scratch qubits are invoked, always see a symmetric pattern of gates.
  • Uf8Uf1∣x⟩∣y⟩=∣x⟩∣y⊕f1(x)⊕f8(x)⟩=∣x⟩∣y⊕f9(x)⟩U_{f8} U_{f1} \ket{x}\ket{y} = \ket{x}\ket{y \oplus f_1(x) \oplus f_8(x)} = \ket{x}\ket{y \oplus f_9(x)}Uf8​Uf1​∣x⟩∣y⟩=∣x⟩∣y⊕f1​(x)⊕f8​(x)⟩=∣x⟩∣y⊕f9​(x)⟩

  1. The computation is done using the scratch qubits.
  2. The answer is copied to the target or output register
  3. The computation is inverted to reset the scratch qubits to ∣0⟩\ket{0}∣0⟩.
  • called "uncomputation", ensures the scratch qubits are returned to their initial state.
  • no input or output qubits are entangled with the scratch qubits at the end of the algorithm.

def random_oracle(n):
    # Circuit object to hold the gates
    circuit = qiskit.QuantumCircuit(n + 1)

    # With 50% probability, return a constant oracle
    if np.random.randint(0, 2):
        qasm = ""
        # another 50:50 chance of it being a 1 instead of 0 oracle
        if np.random.randint(0, 2):
            qasm += f"x q[{n-1}];"
        return qasm, "constant"

    # A balanced function has half the inputs as 0
    # Randomly select where those are
    zero_strings = np.random.choice(range(2**n),int(2**(n-1)),replace=False)

    for string in zero_strings:

        # Convert base 10 to 2
        bitstring= f"{string:0b}"

        # X gates for 0 locations
        for xi, bit in enumerate(reversed(bitstring)): # enumerate iterates through the list as well as the index in the list
            if bit == "1":
                circuit.x(xi)

        # C^n X gate
        circuit.mcx(list(range(n)), n)

        # X gates for 0 locations
        for xi, bit in enumerate(reversed(bitstring)):
            if bit == "1":
                circuit.x(xi)

    transpiled_circuit = qiskit.transpile(circuit, basis_gates=["u1", "u3", "u2", "cx"])
    qasm = qiskit.qasm2.dumps(transpiled_circuit)[47:]
    return qasm, "balanced"

import numpy as np
from qiskit import QuantumCircuit, transpile
from qiskit_aer import AerSimulator


def bv_query(n, secret=None):
    # Build oracle for f_s(x) = s · x
    # q[0]..q[n-1] = input register
    # q[n] = scratch/output qubit

    if secret is None:
        value = np.random.randint(0, 2 ** n)
        secret = format(value, f"0{n}b")
    else:
        secret = secret.zfill(n)

    oracle = QuantumCircuit(n + 1, name="Uf")
    for index, bit in enumerate(reversed(secret)):
        if bit == "1":
            oracle.cx(index, n)

    return oracle, secret


def bernstein_vazirani_circuit(n, secret=None):
    # Build full Bernstein-Vazirani circuit
    oracle, secret = bv_query(n, secret)

    qc = QuantumCircuit(n + 1, n)

    # Prepare scratch qubit in |->
    qc.x(n)

    # Apply Hadamards to all qubits
    for i in range(n + 1):
        qc.h(i)

    # Apply oracle
    qc.compose(oracle, inplace=True)

    # Apply final Hadamards to input register
    for i in range(n):
        qc.h(i)

    # Measure input register
    for i in range(n):
        qc.measure(i, i)

    return qc, secret


def run_bernstein_vazirani(n, secret=None, shots=1):
    qc, secret = bernstein_vazirani_circuit(n, secret)

    simulator = AerSimulator()
    compiled = transpile(qc, simulator)
    result = simulator.run(compiled, shots=shots).result()
    counts = result.get_counts()

    measured = max(counts, key=counts.get)
    recovered = measured[::-1]

    return qc, secret, recovered, counts


# Example
qc, secret, recovered, counts = run_bernstein_vazirani(5, "01101")
print(qc.draw())
print("Secret   :", secret)
print("Measured :", recovered)
print("Counts   :", counts)
