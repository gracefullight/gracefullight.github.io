동시사용자 계산​

Little's Law

CU=λ∗W CU = \lambda * WCU=λ∗W
  • 동시사용자 = TPS * (Response Time + Think Time)
  • 웹 시스템에선 동시사용자와 활성사용자 간 차이가 있으므로 TPS 가정 하에 활성 사용자 계산
