ΈΞ░Ήζ┤ΊΕ░ ΈςρΈΞ╕ΈπΒ ΉΜε Ά░εΉ▓┤ΉβΑ Ά░εΉ▓┤ ΉΓυΉζ┤ΉΩΡ Έ╢ΑΉΩυΊΧαΈΛΦ Ά┤ΑΆ│ΕΉΕ▒ ΉπΣΊΧσΉζα ΉζαΈψ╕Ά░Α ΈςρΊα╕ΊΧαΉΩυ ΉδΡΊΧαΈΛΦ Ά▓░Ά│╝Έξ╝ ΉΨ╗ΉζΕ ΉΙα ΉΩΗΆ▒░ΈΓα ΊΨξΊδΕ ΉΩΖΈυ┤Ή▓αΈουΉΩΡ ΉαΒΊΨξΉζΕ Έψ╕Ή╣αΆ▓Ν ΈΡαΈΛΦ ERΈςρΈΞ╕Ήζα Έυ╕ΉιεΉιΡ

  • Έ╢ΑΉ▒ΕΆ╝┤ ΊΧρΉιΧ: Fan Trap, Ά░εΉ▓┤ΉπΣΊΧσ(Entity Set) ΉΓυΉζ┤ΉΩΡ Ά┤ΑΆ│ΕΉΕ▒ ΉπΣΊΧσ(Relation Set)Ήζ┤ ΉιΧΉζαΈΡαΉΨ┤ ΉηΙΆ╕░ΈΛΦΊΧαΉπΑΈπΝ Ά┤ΑΆ│ΕΉΕ▒ ΉαΙΉΜεΆ░Α ΈςρΊα╕ΊΧε Ά▓╜Ήγ░
  • Ά╖ιΉΩ┤ ΊΧρΉιΧ: Chasm Trap, Ά░εΉ▓┤ΉπΣΊΧσ ΉΓυΉζ┤ΉΩΡ Ά┤ΑΆ│ΕΉΕ▒ ΉπΣΊΧσΉζ┤ ΉιΧΉζαΈΡαΉΨ┤ ΉηΙΆ╕░ΈΛΦ ΊΧαΉπΑΈπΝ Ήζ╝Έ╢Α Ά░εΉ▓┤ΉπΣΊΧσΆ│╝ Ά░εΉ▓┤ΉπΣΊΧσ ΉΓυΉζ┤ΉΩΡ Ά┤ΑΆ│ΕΉΕ▒Ήζ┤ Ήκ┤ΉηυΊΧαΉπΑ ΉΧΛΈΛΦ Ά▓╜Ήγ░

Έ░αΉιΧΆ╖εΊβΦέΑΜ

ΉιΧΆ╖εΊβΦΈΡε ΉΩΦΊΜ░ΊΜ░ ΊΔΑΉηΖ, ΉΗΞΉΕ▒, Ά┤ΑΆ│ΕΈξ╝ ΉΜεΉΛνΊΖεΉζα ΉΕ▒ΈΛξΊΨξΉΔΒ, Ά░εΈ░εΆ│╝ Ήγ┤ΉαΒΉζα ΊΟ╕ΉζαΈξ╝ ΉεΕΊΧ┤ ΈΞ░Ήζ┤ΊΕ░ ΈςρΈΞ╕ΉζΕ ΊΗ╡ΊΧσΊΧαΈΛΦ ΊΦΕΈκεΉΕ╕ΉΛν

DB Ήγ┤ΉαΒέΑΜ

ΊΛ╕ΈηεΉηφΉΖαέΑΜ

ΈΞ░Ήζ┤ΊΕ░Ήζα ΉιΧΊβΧΊΧε Ήζ╝Ά┤ΑΉΕ▒Ά│╝ Έυ┤Ά▓░ΉΕ▒ΉζΕ Έ│┤ΉηξΊΧαΆ╕░ ΉεΕΊΧ┤ ΉβΕΉιΕΊηΙ ΉλΖΈμΝΊΧ┤ΉΧ╝ΊΧαΈΛΦ ΈΞ░Ήζ┤ΊΕ░Έ▓ιΉζ┤ΉΛν Ή▓αΈουΉζα ΈΖ╝ΈουΉιΒ ΉηΣΉΩΖ ΈΜρΉεΕ(Logical Unit of Work)

  • ΉδΡΉηΡΉΕ▒: Atomicity, Έ╢ΕΊΧ┤Ά░Α Έ╢ΙΆ░ΑΈΛξΊΧε ΉΙαΊΨΚ ΈΜρΉεΕΈκε ΉβΕΉιΕΊηΙ ΉΙαΊΨΚ ΈαΡΈΛΦ ΉΙαΊΨΚΈΡαΉπΑ ΉΧΛΉζΝ
  • Ήζ╝Ά┤ΑΉΕ▒: Consistency, ΊΛ╕ΈηεΉηφΉΖαΉζ┤ ΉΕ▒Ά│╡ΉιΒΉε╝Έκε ΉβΕΈμΝΈΡαΈσ┤ ΉΨ╕ΉιεΈΓα ΈςρΉΙεΉζ┤ ΉΩΗΈΛΦ ΉΔΒΊΔε ΉειΉπΑ, Έυ┤Ά▓░ΉΕ▒ ΉιεΉΧ╜ Ήκ░Ά▒┤
  • Ά│ιΈο╜ΉΕ▒: Isolation, ΊΛ╕ΈηεΉηφΉΖα ΉΜνΊΨΚ ΉνΣ ΉνΣΆ░Ε ΉΩ░ΉΓ░ Ά▓░Ά│╝ΈΛΦ ΈΜνΈξ╕ ΊΛ╕ΈηεΉηφΉΖα ΉιΣΆ╖╝ Έ╢ΙΆ░Α
  • ΉαΒΉΗΞΉΕ▒: Durability, ΉΕ▒Ά│╡ΉιΒΉε╝Έκε ΉβΕΈμΝΈΡε ΊΛ╕ΈηαΉηφΉΖαΉζΑ ΉΨ┤ΈΨν Ά│ιΉηξΉΩΡΈΠΕ ΉΗΡΉΜν ΈΡαΉΨ┤ΉΕεΈΛΦ ΉΧΙΈΡρ

ΊΛ╕ΈηεΉηφΉΖα ΉΔΒΊΔε ΉιΕΉζ┤έΑΜ

  • ΉΜνΊΨΚ ΉΔΒΊΔε: Active
  • Έ╢ΑΈ╢Ε ΉβΕΈμΝ: Partially Committed
  • ΉβΕΈμΝ ΉΔΒΊΔε: Committed
  • ΉΜνΊΝρ ΉΔΒΊΔε: Failed
  • Ή▓ιΊγΝ ΉΔΒΊΔε: Aborted

ΈΠβΉΜεΉΕ▒ ΉιεΉΨ┤έΑΜ

ΈΜνΉνΣ ΉΓυΉγσΉηΡ ΊβαΆ▓╜ΉζΕ ΉπΑΉδΡΊΧαΈΛΦ ΈΞ░Ήζ┤ΊΕ░Έ▓ιΉζ┤ΉΛν ΉΜεΉΛνΊΖεΉΩΡΉΕε ΉΩυΈθυ ΊΛ╕ΈηεΉηφΉΖαΈΥνΉζ┤ ΉΕ▒Ά│╡ΉιΒΉε╝Έκε ΈΠβΉΜεΉΩΡ ΉΜνΊΨΚΈΡι ΉΙα ΉηΙΈΠΕΈκζ ΉπΑΉδΡΊΧαΈΛΦ Ά╕░ΈΛξ

  • ΊΛ╕ΈηεΉηφΉΖα ΉΛνΉ╝ΑΉξ┤: Ά░Β ΊΛ╕ΈηεΉηφΉΖαΉζΕ Ά╡υΉΕ▒ΊΧαΈΛΦ ΉΩ░ΉΓ░ΈΥνΉζ┤ ΉΜεΉΛνΊΖεΉΩΡΉΕε ΉΜεΆ░ΕΉΩΡ ΈΦ░Έζ╝ ΉΜνΊΨΚΈΡαΈΛΦ ΉΙεΉΕε
  • ΉπΒΈιυΆ░ΑΈΛξΉΕ▒: Serializability, ΊΛ╕ΈηεΉηφΉΖαΈΥνΉζΕ Έ│ΣΊΨΚΉ▓αΈουΊΧε Ά▓░Ά│╝Ά░Α ΊΛ╕ΈηεΉηφΉΖαΈΥνΉζΕ ΉΙεΉ░ρΉιΒΉε╝Έκε Ή▓αΈουΊΧε Ά▓░Ά│╝ΉβΑ Ά░βΉζΑ Ά▓Δ
  • Ά░▒ΉΜιΈ╢ΕΉΜν Έυ╕Ήιε: Lost update problem
    • ΊΛ╕ΈηεΉηφΉΖαΉζ┤ ΈΠβΉζ╝ ΈΞ░Ήζ┤ΊΕ░Έξ╝ ΈΠβΉΜεΉΩΡ Ά░▒ΉΜιΊΧι Ά▓╜Ήγ░ Έ░εΉΔζΊΧαΈΛΦ Έυ╕Ήιε
    • Ήζ┤ΉιΕ ΊΛ╕ΈηεΉηφΉΖαΉζ┤ ΈΞ░Ήζ┤ΊΕ░Έξ╝ Ά░▒ΉΜιΊΧε ΊδΕ ΊΛ╕ΈηεΉηφΉΖαΉζ┤ ΉλΖΈμΝΊΧαΆ╕░ ΉιΕΉΩΡ Ήζ┤ΊδΕ ΊΛ╕ΈηεΉηφΉΖαΉζ┤ Ά░▒ΉΜι Ά░ΤΉζΕ ΈΞχΉΨ┤ ΉΥ░ΈΛΦ Ά▓╜Ήγ░ ΉανΈξα Έ░εΉΔζ
  • Έ╢ΙΉζ╝Ή╣α Έυ╕Ήιε: Inconsistency analysis problem
    • ΈΜνΉνΣ ΉΓυΉγσΉηΡ ΊΛ╕ΈηεΉηφΉΖαΈΥνΉζ┤ ΈΠβΉΜεΉΩΡ ΉΜνΊΨΚΈΡι ΈΧΝ DBΆ░Α Ήζ╝Ά┤ΑΉΕ▒ ΉΩΗΈΛΦ ΈςρΉΙεΈΡε ΉΔΒΊΔεΈκε ΈΓρΈΛΦ Έυ╕Ήιε
  • ΉΩ░ΉΘΕΈ│╡Ά╖Α Έυ╕Ήιε: Cascading rollback problem
    • Έ│╡ΉΙαΉζα ΊΛ╕ΈηεΉηφΉΖαΉζ┤ ΈΞ░Ήζ┤ΊΕ░ Ά│╡Ήει ΉΜε ΊΛ╣ΉιΧ ΊΛ╕ΈηεΉηφΉΖαΉζ┤ Ή▓αΈουΈξ╝ Ή╖ρΉΗΝΊΧι Ά▓╜Ήγ░ ΈΜνΈξ╕ ΊΛ╕ΈηεΉηφΉΖαΉζ┤ Ή▓αΈουΊΧε Έ╢ΑΈ╢ΕΉΩΡ ΈΝΑΊΧ┤ Ή╖ρΉΗΝ Έ╢ΙΆ░ΑΈΛξ

ΈΠβΉΜεΉΕ▒ ΉιεΉΨ┤ Ά╕░Έ▓ΧέΑΜ

  • Locking
    • ΊΛ╕ΈηεΉηφΉΖαΉζ┤ ΉΓυΉγσΊΧαΈΛΦ ΉηΡΉδΡΉΩΡ ΈΝΑΊΧ┤ ΉΔΒΊα╕Έ░░Ήιε(Mutual Exclusive) Ά╕░ΈΛξΉζΕ ΉιεΆ│╡ΊΧαΈΛΦ Ά╕░Έ▓Χ
    • ΉΔΒΊα╕Έ░░ΉιεΈΛΦ ΊΛ╣ΉιΧ ΊΛ╕ΈηεΉηφΉΖαΉζ┤ ΈΞ░Ήζ┤ΊΕ░ ΊΧφΈςσΉΩΡ ΈΝΑΊΧ┤ Έζ╜ΉζΕ ΉΕνΉιΧΊΧαΈσ┤ ΉηιΆ╕ΙΉζΕ ΉΕνΉιΧΊΧε ΊΛ╕ΈηεΉηφΉΖαΉζ┤ ΉΨ╕Έζ╜ΈΡι ΈΧΝΆ╣ΝΉπΑ ΈΞ░Ήζ┤ΊΕ░Έξ╝ ΈΠΖΉιΡΉιΒΉε╝Έκε ΉΓυΉγσ
  • Time-stamp Ordering
    • ΊΛ╕ΈηεΉηφΉΖαΉζ┤ ΉΜεΉΛνΊΖεΉΩΡ ΉηΖΈιξΈΡι ΈΧΝ ΊΔΑΉηΕΉΛνΊΔυΊΦΕ Έ╢ΑΉΩυ, ΊΛ╕ΈηζΆ░εΉΖαΉζα ΉΙαΊΨΚ ΉΙεΉΕεΈξ╝ ΉιεΉΨ┤
    • ΊΔΑΉηΕΉΛνΊΔυΊΦΕ Ά░ΤΉζ┤ ΈΓαΉνΣΉζ╕ ΊΛ╕ΈηεΉηφΉΖαΉζ┤ ΊΔΑΉηΕΉΛνΊΔυΊΦΕ Ά░ΤΉζ┤ Ήγ░ΉΕιΉζ╕ ΊΛ╕ΈηεΉηφΉΖαΉΩΡ ΉιΣΆ╖╝ΊΧαΆ│ι ΉηΙΈΛΦ ΈΞ░Ήζ┤ΊΕ░ ΊΧφΈςσΉΩΡ ΈΝΑΊΧε ΉιΣΆ╖╝ΉζΕ ΉγΦΉ▓φΊΧαΈΛΦ Ά▓╜Ήγ░ ΈκνΈ░▒ ΈΜ╣ΊΧρ
    • ΉπΒΈιυΉΕ▒ Έ│┤Ήηξ
    • ΈΞ░ΈΥεΈζ╜ Έ░σΉπΑ
  • ΈΓβΆ┤ΑΉιΒ Ά╕░Έ▓Χ
    • ΊΛ╕ΈηεΉηφΉΖαΉζ┤ ΉΙαΊΨΚΈΡαΈΛΦ ΈΠβΉΧΙ ΉΨ┤ΈΨιΊΧε Ά▓ΑΉΓυΈΠΕ ΉΜνΊΨΚΊΧαΉπΑ ΉΧΛΉζΝ
    • ΊΛ╕ΈηεΉηφΉΖα ΉΙαΊΨΚ ΈπΙΉπΑΈπΚΉΩΡ Ά░▒ΉΜι ΉΓυΊΧφΈΥνΉζ┤ ΉπΒΈιυΆ░ΑΈΛξΉΕ▒ΉζΕ ΉεΕΈ░αΊΧαΈΛΦ Ά▓╜Ήγ░Ά░Α ΉηΙΈΛΦΉπΑΈπΝ Ά▓ΑΉΓυΊΧαΉΩυ Ήζ┤ΉΔΒΉζ┤ ΉΩΗΉζΕ Ά▓╜Ήγ░ ΊΛ╕ΈηεΉηφΉΖαΉζΕ ΊβΧΉιΧ
  • ΈΓβΆ┤ΑΉιΒ Ά╕░Έ▓Χ ΈΜρΆ│Ε
    • Read Phase: DBΈξ╝ Ήζ╜Ά│ι Ά░εΉζ╕ΉιΒΉζ╕ Έ│╡ΉΓυΈκε Ά░▒ΉΜι, ΊΛ╕ΈηεΉηφΉΖα Ά░▒ΉΜιΉζΑ ΈςρΈΣΡ ΉηΕΉΜε Ά░▒ΉΜιΊΝΝΉζ╝ΉΩΡ Ά╕░Έκζ, ΈΜνΈξ╕ ΊΛ╕ΈηεΉηφΉΖαΉΩΡ ΉζαΊΧ┤ ΉιΣΆ╖╝ Έ╢ΙΆ░ΑΈΛξ
    • Validation Phase: Ά░ΑΊΧ┤ΉπΕ Έ│ΑΊβΦΆ░Α ΈΞ░Ήζ┤ΊΕ░Έ▓ιΉζ┤ΉΛνΉζα Έυ┤Ά▓░ΉΕ▒Ά│╝ Ήζ╝Ά┤ΑΉΕ▒ΉΩΡ ΉαΒΊΨξΉζΕ Ήμ╝ΉπΑ ΉΧΛΉζΕ Ά▓ΔΉζ┤Έζ╝ΈΛΦ Ά▓ΔΉζΕ Έ│┤ΉηξΊΧαΆ╕░ ΉεΕΊΧ┤ Ά▓ΑΉοζΈΡρ, Ήζ┤ΉΔΒ ΉηΙΉζΕ Ά▓╜Ήγ░ ΊΛ╕ΈηεΉηφΉΖα ΉηυΉΜεΉηΣ Έ░Π ΈκνΈ░▒.
    • Write Phase: Έ│ΑΆ▓╜ΉΓυΊΧφΉζ┤ ΉαΒΆ╡υΉιΒΉε╝Έκε DBΉΩΡ ΉιΒΉγσΈΡρ

Locking Ά╕░Έ▓ΧέΑΜ

  • Shared Lock: Ά│╡Ήει ΉηιΆ╕ΙΊΧε ΊΛ╕ΈηεΉηφΉΖαΉζΑ ΈΞ░Ήζ┤ΊΕ░ ΊΧφΈςσΉΩΡ ΈΝΑΊΧ┤ Ήζ╜Ά╕░ΈπΝ Ά░ΑΈΛξ, ΈΜνΈξ╕ ΊΛ╕ΈηεΉηφΉΖαΈΠΕ Ήζ╜Ά╕░ ΈπΝ ΉΜνΊΨΚ Ά░ΑΈΛξ
  • Exclusive Lock: ΉιΕΉγσ ΉηιΆ╕ΙΊΧε ΊΛ╕ΈηεΉηφΉΖαΉζΑ ΈΞ░Ήζ┤ΊΕ░ ΊΧφΈςσΉΩΡ ΈΝΑΊΧ┤ Ήζ╜Ά╕░ΉβΑ ΉΥ░Ά╕░ ΈςρΈΣΡ Ά░ΑΈΛξ, ΈΜνΈξ╕ ΊΛ╕ΈηεΉηφΉΖαΉζΑ Ήζ╜Ά╕░/ΉΥ░Ά╕░ Έ╢ΙΆ░Α.
  • DB -> Table -> Row -> Column: Locking overhead ΉοζΆ░Α, ΉηΡΈμΝ Ά│╡ΉειΉΕ▒ ΈΗΤΉζΝ.

2 Phase LockingέΑΜ

ΈςρΈΥι ΊΛ╕ΈηεΉηφΉΖαΉζ┤ lockΆ│╝ unlock ΉΩ░ΉΓ░ΉζΕ ΊβΧΉηξΈΜρΆ│ΕΉβΑ ΉΙαΉ╢ΧΈΜρΆ│ΕΈκε Ά╡υΈ╢ΕΊΧαΉΩυ ΉΙαΊΨΚ

  • ΊβΧΉηξΈΜρΆ│Ε: ΊΛ╕ΈηεΉηφΉΖαΉζΑ lockΈπΝ ΉΙαΊΨΚΊΧι ΉΙα ΉηΙΆ│ι, unlockΉζΑ ΉΙαΊΨΚΊΧι ΉΙα ΉΩΗΈΛΦ ΈΜρΆ│Ε
  • ΉΙαΉ╢ΧΈΜρΆ│Ε: ΊΛ╕ΈηεΉηφΉΖαΉζΑ unlockΈπΝ ΉΙαΊΨΚΊΧι ΉΙα ΉηΙΆ│ι, lockΉζΑ ΉΙαΊΨΚΊΧι ΉΙα ΉΩΗΈΛΦ ΈΜρΆ│Ε
  • ΉπΒΈιυΆ░ΕΉε╝ΉΕ▒ΉζΕ Έ│┤ΉηξΊΧι ΉΙα ΉηΙΈΛΦ Ά╖εΉΧ╜Ήε╝Έκε Ά░ΑΉηξ ΈπΟΉζ┤ ΉΓυΉγσΈΡρ.
  • Ά╡ΡΉ░σΉΔΒΊΔε Έ░εΉΔζΈΡι ΉΙα ΉηΙΉε╝ΈΓα ΉαΙΈ░σΆ│╝ ΊΔΡΉπΑΈκε ΊΧ┤Ά▓░.

2 Phase CommitέΑΜ

Έ╢ΕΉΓ░ ΈΞ░Ήζ┤ΊΕ░Έ▓ιΉζ┤ΉΛν ΊβαΆ▓╜ΉΩΡΉΕε ΉδΡΉηΡΉΕ▒ΉζΕ Έ│┤ΉηξΊΧαΆ╕░ ΉεΕΊΧ┤ Έ╢ΕΉΓ░ ΊΛ╕ΈηεΉηφΉΖαΉΩΡ ΊΠυΊΧρΊΧαΈΛΦ ΈσΦΉ╗νΈΜΙΉοα PrepareΉβΑ Commit ΈΜρΆ│ΕΈξ╝ ΊΗ╡ΊΧ┤ ΈςρΈΥι ΈΖ╕ΈΥεΆ░Α Ή╗νΈ░ΜΊΧαΉπΑ ΉΧΛΉε╝Έσ┤ ΈκνΈ░▒ΊΧαΈΛΦ Ά╕░Έ▓Χ

  • Phase 1: ΉνΑΈ╣Ε ΈΜρΆ│Ε
    • ΊΧε ΉπΑΉΩφΈΖ╕ΈΥεΉΩΡΉΕε Ή╗νΈ░Μ ΉγΦΆ╡υ, Global CoordinatorΆ░Α Commit Point Site Ά▓░ΉιΧ
    • Prepare Message Έξ╝ ΉιΕΉΗκΊΧαΆ│ι ΉδΡΆ▓σ ΈΖ╕ΈΥεΈΛΦ Prepare Message Reply
  • Phase 2: Ή╗νΈ░Μ ΈΜρΆ│Ε
    • ΈςρΈΥι ΉπΑΉΩφ ΈΖ╕ΈΥεΆ░Α Ή╗νΈ░Μ ΉνΑΈ╣ΕΈΡαΉΩΙΈΜνΈΛΦ Ά▓ΔΉζΕ Έ░δΉζΑ Ά▓╜Ήγ░ Ά░Β ΈΖ╕ΈΥεΉΩΡ CommitΉζΕ ΈςΖΈι╣ΊΧρ
