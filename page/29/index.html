nodejsлЎң м •л§җ к°„лӢЁн•ң oauth2, jwt л“ұмқҳ мқёмҰқмқ„ мӮ¬мҡ©н•ҳм§Җ м•ҠлҠ” APIлҘј л§Ңл“Өкё°м—җлҠ” expressк°Җ м •л§җ л”ұмқҙлӢӨ.

н•ҳм§Җл§Ң мЎ°кёҲ лҚ” к№”лҒ”н•ң мҪ”л“ңлҘј мӣҗн•ҳкұ°лӮҳ мғҲлЎӯм§Җл§Ң л°ҳліөм Ғмқё кё°лҠҘмқ„ л„Јкё° мң„н•ҙм„  лҚ” нҒ° н”„л Ҳмһ„мӣҢнҒ¬к°Җ н•„мҡ”н–ҲлӢӨ.

мЎ°кұҙвҖӢ

м–ём ңлӮҳ к°ңл°ңмӢңк°„мқҖ л¶ҖмЎұн•ҳкё°м—җ лӮҙк°Җ н•„мҡ”н•ң кё°лҠҘл“Өмқ„ м •лҰ¬н•ҙ ліҙм•ҳлӢӨ.

  • await, asyncлҘј м•„л¬ҙ м„ёнҢ… м—Ҷмқҙ мӮ¬мҡ©н•ҙ Promise chainingм—җм„ң лІ—м–ҙлӮҳкі  мӢ¶лӢӨ. (Babelкіј Webpackмқ„ м§Ғм ‘ м„ёнҢ…н•ҳлҠ” мӢңк°„ кұёлҰ¬лҠ” м§“мқ„ м•Ҳн•ҳкі  мӢ¶лӢӨ.)
  • кё°ліём Ғмқё мӣ№ ліҙм•Ҳкё°лҠҘ (validate, csrf, xss, injection) л°Ҹ API Throttle Request кё°лҠҘмқҙ мһҲм–ҙм•јн•ңлӢӨ.
  • мҶҢм…ң лЎңк·ёмқёмқ„ passportлҘј мӮ¬мҡ©н•ҳм§Җ м•Ҡкі  н•ң мӨ„лЎң лҒқлӮҙкі  мӢ¶м—ҲлӢӨ.
  • JWT мқёмҰқ, Log кё°лҠҘ, Bcrypt м•”нҳёнҷ”лҘј м§Ғм ‘ л¶ҷнһҲкі  мӢ¶м§Җ м•ҠлӢӨ.
  • ORMм—җ л§һкІҢ E-commerce DBлҘј кө¬мЎ°нҷ”н–Ҳкё° л•Ңл¬ём—җ ORMмқҙ кјӯ н•„мҡ”н•ҳлӢӨ.
  • м§ҒкҙҖм Ғмқҙм—¬м•ј н•ңлӢӨ. (Modelкіј Controllerк°Җ мҷ„лІҪнһҲ лӮҳлүҳлҠ” кө¬мЎ°лҘј мӣҗн–ҲлӢӨ.)
  • л¬ём„ңнҷ”к°Җ мҷ„лІҪн•ҙм•јн•ңлӢӨ.
  • м»Өл®ӨлӢҲнӢ°лӮҳ Github Issueк°Җ нҷңм„ұнҷ” лҗҳм–ҙмһҲм–ҙм•јн•ңлӢӨ.
  • кҙҖлҰ¬мһҗ кё°лҠҘмқ„ SSRлЎң н•ҙм•јн•  к°ҖлҠҘм„ұмқҙ мһҲм–ҙ н…ңн”ҢлҰҝ кё°лҠҘмқҙ мһҲм–ҙм•јн•ҳл©°, ejsлҠ” м ҲлҢҖ м“°кі  мӢ¶м§Җ м•Ҡкі  лӢӨлҘё н…ңн”ҢлҰҝмқ„ м§Ғм ‘ м„ёнҢ…н•ҙм•јн•ҳлҠ” мқјмқҖ лҚ”лҚ”мҡұ м—Ҷм–ҙм•јн•ңлӢӨ.

мқҙ мЎ°кұҙмқ„ кұ°лҘҙлӢҲ adonisjs лқјлҠ” мІҳмқҢ ліҙлҠ” н”„л Ҳмһ„мӣҢнҒ¬к°Җ лҲҲм•һм—җ мһҲм—ҲлӢӨ.
