function pkcs5_unpad($text) {
  $pad = ord($text{strlen($text)-1});
  if ($pad > strlen($text)) {
    return $text;
  }
  if (!strspn($text, chr($pad), strlen($text) - $pad)) {
    return $text;
  }
  return substr($text, 0, -1 * $pad);
}

PKCS5 vs PKCS7вҖӢ

  • openssl_encryptмқҳ default padding мқҖ PKCS7 padding мқёлҚ°, м–ҙл–»кІҢ нҳёнҷҳмқҙ лҗҳлҠ” кІғмқјк№Ң?
  • м—¬кё°м„ң н•ҙлӢөмқ„ м°ҫм•ҳлӢӨ.

The difference between the PKCS#5 and PKCS#7 padding mechanisms is the block size; PKCS#5 padding is defined for 8-byte block sizes, PKCS#7 padding would work for any block size from 1 to 255 bytes. So fundamentally PKCS#5 padding is a subset of PKCS#7 padding for 8 byte block sizes. so, data encrypted with PKCS#5 is able to decrypt with PKCS#7, but data encrypted with PKCS#7 may not be able to decrypt with PKCS#5.

php artisan лӘ…л №м–ҙлҠ” laravelм—җм„ң мӮ¬мҡ©н•ҳлҠ” м»Өл§Ёл“ң мүҳмқҙлӢӨ. php artisanлЎң лӘ…л №м–ҙ лҰ¬мҠӨнҠёк°Җ лӮҳмҳӨкі  php artisan help лӘ…л №м–ҙ лЎң н•ҙлӢ№ лӘ…л №м–ҙмқҳ мҳөм…ҳмқ„ ліј мҲҳ мһҲлӢӨ.

н”„лЎңм қнҠё кө¬мЎ°вҖӢ

лқјлқјлІЁ н•ңкёҖ л©”лүҙм–јмқҙлӮҳ кіөмӢқ л©”лүҙм–јмқ„ ліҙл©ҙ лҗҳлҠ”лҚ° л§Ҳм№ҳ мӮ¬м „мқ„ нҺҙлҶ“м•ҳлҠ”лҚ° м–ҙл–Ө лӢЁм–ҙлҘј м°ҫлҠ”м§Җ к№ҢлЁ№мқҖ лҠҗлӮҢмқҙ л“Ө кІғмқҙлӢӨ.

лӘЁл“  н”„л Ҳмһ„мӣҢнҒ¬лҠ” Model, View, Controller, Routingл§Ң м•Ңл©ҙ лҒқмқҙлқјлҠ” кІғмқ„ мһ мӢң кё°м–ө мҶҚм—җм„ң кәјлӮҙліҙкі  л”ұ 3к°Җм§Җ кІҪлЎңл§Ң кё°м–өн•ҳмһҗ.

  • Route: routes/
  • Controller: app/Http/Controllers/
  • View : resources/views/

лӘЁлҚёмқҖ м–ҙл”Ём–ҙ? лқјкі  мғқк°Ғн•  кІғ к°ҷмқҖлҚ° Modelмқ„ мҷ„лІҪн•ҳкІҢ мӮ¬мҡ©н•ҳл Өл©ҙ ORMмқ„ мҚЁм•јн•ҳкі  DBлҘј ORMм—җ л§һкІҢ м •к·ңнҷ”н•ҙм•јн•ҳл©° мқҙ мһ‘м—…мқҖ мІҳмқҢ м§„н–үн•ҳкё°м—җ кі нҶөмҠӨлҹҪлӢӨ. лӢӨмқҢ нҸ¬мҠӨнҠём—җ м§„н–үн•ҙліҙмһҗ.

RouteвҖӢ

лӘЁл“  мӣ№ н”„л Ҳмһ„мӣҢнҒ¬мқҳ кё°ліёмқҖ RoutingмқҙлӢӨ. routes нҸҙлҚ”м—җ кё°ліёмңјлЎң api, channels, console, web нҢҢмқјмқҙ ліҙмқҙлҠ”лҚ° к·ё мӨ‘ 2к°Җм§Җл§Ң м•Ңл©ҙ лҗңлӢӨ.

  • api.php: Token мқёмҰқмқҙ н•„мҡ”н•ң лқјмҡ°н„°лЎң /api/{route} лЎң м ‘к·јмқҙ к°ҖлҠҘн•ҳлӢӨ.
  • web.php: кё°ліём Ғмқё RouteмқҙлӢӨ. GETмқҙ м•„лӢҢ лӢӨлҘё л©”мҶҢл“ңлҠ” CSRF Tokenмқҙ мһҲм–ҙм•јл§Ң нҳём¶ңмқҙ к°ҖлҠҘн•ҳлӢӨ.

ControllerвҖӢ

Routing мҡ”мІӯмқ„ мІҳлҰ¬н•ҳлҠ” лЎңм§Ғмқҙ л“Өм–ҙмһҲлҠ” л¶Җл¶„мқҙлӢӨ.

к°қмІҙм§Җн–ҘвҖӢ

к°қмІҙм§Җн–Ҙ кё°лІ•вҖӢ

  • нҳ„мӢӨ м„ёкі„мқҳ к°ңмІҙлҘј кё°кі„мқҳ л¶Җн’ҲмІҳлҹј н•ҳлӮҳмқҳ к°қмІҙлЎң л§Ңл“Өм–ҙ кё°кі„м Ғмқё л¶Җн’Ҳл“Өмқ„ мЎ°лҰҪн•ҳм—¬ м ңн’Ҳмқ„ л§Ңл“Ө л“Ҝ мҶҢн”„нҠёмӣЁм–ҙлҘј к°ңл°ңн•  л•ҢлҸ„ к°қмІҙл“Өмқ„ мЎ°лҰҪн•ҙ мһ‘м„ұн•  мҲҳ мһҲлҸ„лЎқ н•ҳлҠ” кё°лІ•

к°қмІҙвҖӢ

  • лҚ°мқҙн„°мҷҖ лҚ°мқҙн„°лҘј мІҳлҰ¬н•ҳлҠ” н•ЁмҲҳлҘј л¬¶м–ҙ лҶ“мқҖ н•ҳлӮҳмқҳ мҶҢн”„нҠёмӣЁм–ҙ лӘЁл“Ҳ
  • лҚ°мқҙн„°: к°қмІҙк°Җ к°Җм§Җкі  мһҲлҠ” м •ліҙ
  • н•ЁмҲҳ: к°қмІҙк°Җ мҲҳн–үн•ҳлҠ” кё°лҠҘ, к°қмІҙк°Җ к°–лҠ” лҚ°мқҙн„°лҘј мІҳлҰ¬н•ҳлҠ” м•Ңкі лҰ¬мҰҳ

нҒҙлһҳмҠӨвҖӢ

  • кіөнҶөлҗң мҶҚм„ұкіј м—°мӮ°мқ„ к°–лҠ” к°қмІҙмқҳ м§‘н•©
  • к°қмІҙмқҳ мқјл°ҳм Ғмқё нғҖмһ…
  • к°Ғк°Ғмқҳ к°қмІҙл“Өмқҙ к°–лҠ” мҶҚм„ұкіј м—°мӮ°мқ„ м •мқҳн•ҳкі  мһҲлҠ” нӢҖ
  • мқёмҠӨн„ҙмҠӨ: нҒҙлһҳмҠӨм—җ мҶҚн•ң к°Ғк°Ғмқҳ к°қмІҙ

л©”м„ём§ҖвҖӢ

  • к°қмІҙл“Ө к°„м—җ мғҒнҳёмһ‘мҡ©мқ„ н•ҳлҠ” лҚ° мӮ¬мҡ©лҗҳлҠ” мҲҳлӢЁ
  • к°қмІҙм—җкІҢ м–ҙл–Ө н–үмң„лҘј н•ҳлҸ„лЎқ м§ҖмӢңн•ҳлҠ” лӘ…л № лҳҗлҠ” мҡ”кө¬мӮ¬н•ӯ

к°қмІҙм§Җн–Ҙ кё°ліё мӣҗм№ҷвҖӢ

нҶөн•© н…ҢмҠӨнҠёвҖӢ

  • мӮ¬мҡ© кё°л°ҳ н…ҢмҠӨнҠё: лҸ…лҰҪ нҒҙлһҳмҠӨлҘј н…ҢмҠӨнҠён•ң нӣ„ лҸ…лҰҪ нҒҙлһҳмҠӨлҘј мӮ¬мҡ©н•ҳлҠ” лӢӨмқҢ кі„мёөмқҳ мў…мҶҚ нҒҙлһҳмҠӨлҘј н…ҢмҠӨнҠё

IEEE 1471вҖӢ

  • мҶҢн”„нҠёмӣЁм–ҙ м§‘м•Ҫм Ғмқё мӢңмҠӨн…ңм—җм„ң м•„нӮӨн…ҚмІҳк°Җ н‘ңнҳ„н•ҙм•ј н•ҳлҠ” мҡ”мҶҢмҷҖ лӮҙмҡ©л“Ө, мқҙл“Ө к°„мқҳ кҙҖкі„лҘј к·ңм •н•ҳкі  мһҲлҠ” көӯм ң н‘ңмӨҖ
  • н‘ңмӨҖнҷ”, мӨ‘лҰҪм„ұ, мң м—°м„ұ, мқҳмӮ¬мҶҢнҶө

м ҖмһҘмҶҢ кө¬мЎ°вҖӢ

  • мӨ‘м•ҷмһҗлЈҢкө¬мЎ°мҷҖ лҸ…лҰҪлҗң м»ҙнҸ¬л„ҢнҠёлЎң кө¬м„ұлҗң м•„нӮӨн…ҚмІҳ
  • нҒ° лҚ°мқҙн„°мқҳ мқҙлҸҷ л°Ҹ кіөмң м—җ м Ғн•©н•ҳл©° м»ҙнҸ¬л„ҢнҠё к°„ нҶөмӢ мқҖ мқҙлӨ„м§Җм§Җ м•ҠлҠ”лӢӨ.
  • лҢҖлҹүмқҳ лҚ°мқҙн„°лҘј м ҖмһҘн•ҳлҠ”лҚ° нҡЁкіјм ҒмқҙлӢӨ.
  • м»ҙнҸ¬л„ҢнҠёмқҳ м¶”к°Җ мӮӯм ңк°Җ нҺёлҰ¬н•ҳлӢӨ.
  • мӨ‘м•ҷ м§‘мӨ‘нҷ”лҘј нҶөн•ҙ лҚ°мқҙн„° кҙҖлҰ¬к°Җ мҡ©мқҙн•ҳкі  ліҙм•Ҳмқҙ лӣ°м–ҙлӮҳлӢӨ.

лӘЁлҚёвҖӢ

  • м• н”ҢлҰ¬мјҖмқҙм…ҳмқҳ н•өмӢ¬ кё°лҠҘмқ„ нҸ¬н•Ё
  • мғҒнғң ліҖнҷ”мӢң м»ЁнҠёлЎӨлҹ¬мҷҖ л·°м—җ м „лӢ¬

л·°вҖӢ

  • м •ліҙ н‘ңмӢңлҘј кҙҖлҰ¬
  • кІ°кіјл¬ј мғқм„ұмқ„ мң„н•ҙ лӘЁлҚёлЎңл¶Җн„° м •ліҙлҘј мҲҳмӢ 

м»ЁнҠёлЎӨлҹ¬вҖӢ

  • мӮ¬мҡ©мһҗлЎңл¶Җн„° мһ…л Ҙмқ„ л°ӣм•„ лӘЁлҚёкіј л·°м—җ лӘ…л №мқ„ м „лӢ¬
  • лӘЁлҚём—җ лӘ…л №мқ„ м „лӢ¬н•ҙ мғҒнғңлҘј ліҖкІҪн•ҳкі  л·°м—җ лӘ…л №мқ„ ліҙлӮҙ н‘ңмӢң л°©лІ•мқ„ ліҖкІҪ

нҒҙлқјмқҙм–ёнҠё/м„ңлІ„ кө¬мЎ°вҖӢ

  • нҒҙлқјмқҙм–ёнҠёмҷҖ м„ңлІ„лЎң лӮҳлүҳлҠ” м•„нӮӨн…ҚмІҳ
  • нҒҙлқјмқҙм–ёнҠё: мӮ¬мҡ©мһҗлЎңл¶Җн„° мһ…л Ҙмқ„ л°ӣм•„ м„ңлІ„м—җ мҡ”мІӯмқ„ м „лӢ¬
  • м„ңлІ„: мҲҳмӢ лҗң мҡ”мІӯмқ„ мҲҳн–үн•ҳкі  лҚ°мқҙн„°мқҳ мқјкҙҖм„ұмқ„ мң м§Җ
  • мғҲлЎңмҡҙ м„ңлІ„мқҳ м¶”к°Җ л°Ҹ м—…к·ёл Ҳмқҙл“ңк°Җ мҡ©мқҙ
  • лҚ°мқҙн„°к°Җ м„ңлІ„м—җ м§‘мӨ‘лҗҳм–ҙ лҚ°мқҙн„° кҙҖлҰ¬ мҡ©мқҙ
  • м„ңлІ„м—җ л„ӨнҠёмӣҢнҒ¬ нҠёлһҳн”Ҫкіј лҚ°мқҙн„°к°Җ м§‘мӨ‘лҗҳ мІҳлҰ¬ л№„мҡ©мқҙ кёүмҰқн•  мҲҳ мһҲлӢӨ.

кі„мёө кө¬мЎ°вҖӢ

  • н”„лЎңм„ёмӢұмқ„ мң„н•ң мӢңмҠӨн…ңмқҙ к°Ғ н•„н„°м—җ мәЎмҠҗнҷ”лҗҳм–ҙ мһҲмңјл©° лҚ°мқҙн„°лҠ” мқём ‘ н•„н„° мӮ¬мқҙмқҳ нҢҢмқҙн”„лҘј нҶөн•ҙ м „лӢ¬лҗҳлҠ” нҳ•нғң
