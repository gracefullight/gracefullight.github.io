  • RUN: мқҙлҜём§Җ л№Ңл“ң мӢң м»Ён…Ңмқҙл„Ҳм—җм„ң мӢӨн–үн•  лӘ…л №м–ҙ м •мқҳ
  • COPY: нҳёмҠӨнҠём—җм„ң м»Ён…Ңмқҙл„ҲлЎң нҢҢмқј л°Ҹ л””л үнҶ лҰ¬ ліөмӮ¬
  • ADD: COPY + м••м¶• н•ҙм ң + URL лӢӨмҡҙлЎңл“ң
    • мҡҙмҳҒ мІҙм ңлҘј лӢҙмқҖ кё°л°ҳ мқҙлҜём§ҖлҘј л§Ңл“ңлҠ” кІҪмҡ°мІҳлҹј нҠ№мҲҳн•ң кІҪмҡ°м—җл§Ң мӮ¬мҡ©н•ҳл©ҙ лҗңлӢӨ.
    • м•Ҳм •м„ұ ліҙмһҘмқҙ лҗҳм§Җ м•ҠмңјлҜҖлЎң COPY лҘј мӮ¬мҡ©н•ҳмһҗ.
  • CMD: м»Ён…Ңмқҙл„Ҳм—җм„ң foregroundлЎң мӢӨн–үн•  лӘ…л №м–ҙ м •мқҳ
  • ENTRYPOINT: м»Ён…Ңмқҙл„ҲлҘј мӢӨн–ү к°ҖлҠҘ нҢҢмқјлЎң мӮ¬мҡ©н•  л•Ң м •мқҳн•ҳлҠ” лӘ…л №
    • CMD мҷҖ ENTRYPOINT л‘ҳ лӢӨ мӮ¬мҡ© к°ҖлҠҘ
  • ARG: docker image buildлҘј мӢӨн–үн•  л•Ң мӮ¬мҡ©н•ҳлҠ” ліҖмҲҳ
  • ENV: м»Ён…Ңмқҙл„Ҳ м•Ҳмқҳ нҷҳкІҪліҖмҲҳ м •мқҳ
  • EXPOSE: м»Ён…Ңмқҙл„Ҳк°Җ л…ём¶ңн•ҳлҠ” нҸ¬нҠё
  • VOLUME: нҳёмҠӨнҠёлӮҳ лӢӨлҘё м»Ён…Ңмқҙл„Ҳм—җм„ң л§ҲмҡҙнҠён•  мҲҳ мһҲлҠ” нҸ¬мқёнҠё мғқм„ұ
  • LABEL: мқҙлҜём§Җм—җ м¶”к°Җн•ҳлҠ” л©”нғҖлҚ°мқҙн„°
  • STOPSIGNAL: м»Ён…Ңмқҙл„Ҳм—җ м „лӢ¬лҗҳл©ҙ м»Ён…Ңмқҙл„ҲлҘј мў…лЈҢн•ҳлҠ” мӢңмҠӨн…ң мӢңк·ёл„җ м„Өм •
  • HEALTHCHECK: м»Ён…Ңмқҙл„Ҳ м•Ҳм—җм„ң лӘ…л №мқ„ мӢӨн–ү нӣ„ кІ°кіјлҘј н—¬мҠӨ мІҙнҒ¬м—җ мӮ¬мҡ©
  • USER: м»Ён…Ңмқҙл„Ҳ мӢӨн–ү мӢң м»Ён…Ңмқҙл„Ҳ мӮ¬мҡ©мһҗ
    • мқҙлҜём§Җ л№Ңл“ңмӢң USER л’Өм—җ лӮҳмҳӨлҠ” RUN мқёмҠӨнҠёлҹӯм…ҳлҸ„ н•ҙлӢ№ мӮ¬мҡ©мһҗмқҳ к¶Ңн•ңмңјлЎң мӢӨн–үлҗңлӢӨ.
  • WORKDIR: м»Ён…Ңмқҙл„Ҳмқҳ мһ‘м—… л””л үнҶ лҰ¬
  • ONBUILD: м»Ён…Ңмқҙл„Ҳ м•Ҳм—җм„ң мӢӨн–үлҗҳлҠ” лӘ…л № м •мқҳ, мқҙлҜём§Җм—җм„ң мӢӨн–үлҗҳм§Җ м•ҠлҠ”лӢӨ.
    • ONBUILD лҘј м •мқҳн•ң мқҙлҜём§ҖлҘј кё°л°ҳ мқҙлҜём§ҖлЎң мӮјм•„ лӢӨлҘё мқҙлҜём§ҖлҘј л№Ңл“ңн•  л•Ң мӢӨн–үлҗңлӢӨ.

  • busybox: кё°ліё мң нӢёлҰ¬нӢ° (echo, ls л“ұ) мқҙ мһҲлҠ” мқҙлҜём§Җ
    • нҢЁнӮӨм§Җ кҙҖлҰ¬мһҗк°Җ м—ҶлӢӨ.
    • л””лІ„к№…мқҖ мўҖ лӮ«лӢӨ.
  • alpine: busybox кё°л°ҳмңјлЎң 4MB м§Җл§Ң apk нҢЁнӮӨм§Җ л§ӨлӢҲм Җк°Җ мһҲлӢӨ.
    • glibc лҢҖмӢ  muslмқ„ м“ҙлӢӨ.
    • apk add --no-cache package
    • apk add --no-cache --virtual=ailas package && apk del --no-cache ailas

л©ҖнӢ°мҠӨн…Ңмқҙм§Җ л№Ңл“ңвҖӢ

golangкіј к°ҷмқҖ л№Ңл“ңк°Җ н•„мҡ”н•ң мқҙлҜём§Җм—җм„ңлҠ” л©ҖнӢ°мҠӨн…Ңмқҙм§Җ л№Ңл“ңлҘј мқҙмҡ©н•ҙ л№Ңл“ң нҷҳкІҪкіј н”„лЎңлҚ•м…ҳ нҷҳкІҪмқ„ лӢӨлҘҙкІҢ к°Җм ёк°Ҳ мҲҳ мһҲлӢӨ.

FROM golang:1.9 AS build

WORKDIR /
COPY . /go/src/github.com/...
RUN go get gokpg.in/gorp.v1
RUN cd /go/src/github.com/... && go build -o bin/start main.go

FROM alpine:3.7
COPY --from=build /go/src/.../bin/start /usr/local/bin/
CMD ["start"]
