해결방안​

overflow: hidden​

래퍼에 이 속성을 넣는 것으로 해결 된다.

body {
  width: 1000px;
  margin: 0 auto;
}
#aside {
  float: left;
  width: 200px;
}
#section {
  float: left;
  width: 800px;
}
#wrap {
  overflow: hidden;
}

clear: both​

또는 래퍼를 삭제하고 구획을 나누는 부분에 sibling 노드로 clear: both 속성을 주면 된다.

  body {
    width: 1000px;
    margin: 0 auto;
  }
