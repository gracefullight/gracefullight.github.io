AWS 용어 정리

2017년 7월 16일 · 약 2분
Eunkwang Shin
Owner

AWS의 생태계에 들어가기 위해선 먼저 용어에 익숙해져야 될 것 같았다. 사용법은 쉬운데.. 약어가 처음 접하는 사람에게 조금 부담스러웠다.
