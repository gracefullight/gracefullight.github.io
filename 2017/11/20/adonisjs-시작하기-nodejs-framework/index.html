adonisjs ΉΜεΉηΣΊΧαΆ╕░ (nodejs framework)

nodejsΈκε ΉιΧΈπΡ Ά░ΕΈΜρΊΧε oauth2, jwt ΈΥ▒Ήζα Ήζ╕ΉοζΉζΕ ΉΓυΉγσΊΧαΉπΑ ΉΧΛΈΛΦ APIΈξ╝ ΈπΝΈΥνΆ╕░ΉΩΡΈΛΦ expressΆ░Α ΉιΧΈπΡ ΈΦ▒Ήζ┤ΈΜν.

ΊΧαΉπΑΈπΝ Ήκ░Ά╕Ι ΈΞΦ Ά╣ΦΈΒΦΊΧε Ή╜ΦΈΥεΈξ╝ ΉδΡΊΧαΆ▒░ΈΓα ΉΔΙΈκφΉπΑΈπΝ Έ░αΈ│╡ΉιΒΉζ╕ Ά╕░ΈΛξΉζΕ ΈΕμΆ╕░ ΉεΕΊΧ┤ΉΕι ΈΞΦ ΊΒ░ ΊΦΕΈιΙΉηΕΉδΝΊΒυΆ░Α ΊΧΕΉγΦΊΨΙΈΜν.

Ήκ░Ά▒┤έΑΜ

ΉΨ╕ΉιεΈΓα Ά░εΈ░εΉΜεΆ░ΕΉζΑ Έ╢ΑΉκ▒ΊΧαΆ╕░ΉΩΡ ΈΓ┤Ά░Α ΊΧΕΉγΦΊΧε Ά╕░ΈΛξΈΥνΉζΕ ΉιΧΈουΊΧ┤ Έ│┤ΉΧαΈΜν.

  • await, asyncΈξ╝ ΉΧΕΈυ┤ ΉΕ╕ΊΝΖ ΉΩΗΉζ┤ ΉΓυΉγσΊΧ┤ Promise chainingΉΩΡΉΕε Έ▓ΩΉΨ┤ΈΓαΆ│ι ΉΜ╢ΈΜν. (BabelΆ│╝ WebpackΉζΕ ΉπΒΉιΣ ΉΕ╕ΊΝΖΊΧαΈΛΦ ΉΜεΆ░Ε Ά▒╕ΈουΈΛΦ ΉπΥΉζΕ ΉΧΙΊΧαΆ│ι ΉΜ╢ΈΜν.)
  • Ά╕░Έ│╕ΉιΒΉζ╕ Ήδ╣ Έ│┤ΉΧΙΆ╕░ΈΛξ (validate, csrf, xss, injection) Έ░Π API Throttle Request Ά╕░ΈΛξΉζ┤ ΉηΙΉΨ┤ΉΧ╝ΊΧεΈΜν.
  • ΉΗΝΉΖε ΈκεΆ╖╕Ήζ╕ΉζΕ passportΈξ╝ ΉΓυΉγσΊΧαΉπΑ ΉΧΛΆ│ι ΊΧε ΉνΕΈκε ΈΒζΈΓ┤Ά│ι ΉΜ╢ΉΩΙΈΜν.
  • JWT Ήζ╕Ήοζ, Log Ά╕░ΈΛξ, Bcrypt ΉΧΦΊα╕ΊβΦΈξ╝ ΉπΒΉιΣ Έ╢βΊηΙΆ│ι ΉΜ╢ΉπΑ ΉΧΛΈΜν.
  • ORMΉΩΡ ΈπηΆ▓Ν E-commerce DBΈξ╝ Ά╡υΉκ░ΊβΦΊΨΙΆ╕░ ΈΧΝΈυ╕ΉΩΡ ORMΉζ┤ Ά╝φ ΊΧΕΉγΦΊΧαΈΜν.
  • ΉπΒΆ┤ΑΉιΒΉζ┤ΉΩυΉΧ╝ ΊΧεΈΜν. (ModelΆ│╝ ControllerΆ░Α ΉβΕΈ▓╜ΊηΙ ΈΓαΈΚαΈΛΦ Ά╡υΉκ░Έξ╝ ΉδΡΊΨΙΈΜν.)
  • Έυ╕ΉΕεΊβΦΆ░Α ΉβΕΈ▓╜ΊΧ┤ΉΧ╝ΊΧεΈΜν.
  • Ή╗νΈχνΈΜΙΊΜ░ΈΓα Github IssueΆ░Α ΊβεΉΕ▒ΊβΦ ΈΡαΉΨ┤ΉηΙΉΨ┤ΉΧ╝ΊΧεΈΜν.
  • Ά┤ΑΈουΉηΡ Ά╕░ΈΛξΉζΕ SSRΈκε ΊΧ┤ΉΧ╝ΊΧι Ά░ΑΈΛξΉΕ▒Ήζ┤ ΉηΙΉΨ┤ ΊΖεΊΦΝΈο┐ Ά╕░ΈΛξΉζ┤ ΉηΙΉΨ┤ΉΧ╝ΊΧαΈσ░, ejsΈΛΦ ΉιΙΈΝΑ ΉΥ░Ά│ι ΉΜ╢ΉπΑ ΉΧΛΆ│ι ΈΜνΈξ╕ ΊΖεΊΦΝΈο┐ΉζΕ ΉπΒΉιΣ ΉΕ╕ΊΝΖΊΧ┤ΉΧ╝ΊΧαΈΛΦ Ήζ╝ΉζΑ ΈΞΦΈΞΦΉγ▒ ΉΩΗΉΨ┤ΉΧ╝ΊΧεΈΜν.

Ήζ┤ Ήκ░Ά▒┤ΉζΕ Ά▒░Έξ┤ΈΜΙ adonisjs Έζ╝ΈΛΦ Ή▓αΉζΝ Έ│┤ΈΛΦ ΊΦΕΈιΙΉηΕΉδΝΊΒυΆ░Α ΈΙΙΉΧηΉΩΡ ΉηΙΉΩΙΈΜν.

ΉΕ▒ΈΛξέΑΜ

ΈΓαΉΒαΉπΑ ΉΧΛΈΜν. ΉΨ┤Ή░ρΊΦ╝ nginxΉΩΡΉΕε reverse proxy Ή▓αΈουΉΩΡ, auto scaling Ά▒╕Ά║╝Ά│ι Ά╖╕ΈηαΈΠΕ ΈΞΦ Έ╣ιΈξ╕ ΉΗΞΈΠΕΈξ╝ ΉδΡΊΧαΈσ┤ golangΉε╝Έκε Ά░ΑΉΧ╝ΉπΑ Ήβε Ήζ┤ ΊΠυΉΛνΊΛ╕Έξ╝ Έ│┤Ά│ιΉηΙΈΓα!

  • ΉΩυΆ╕░ΉΕε ΉηΡΉΕ╕ΊΧε ΈΓ┤ΉγσΉζΕ ΊβΧΉζ╕ΊΧι ΉΙα ΉηΙΈΜν.

ΊΠΚΊΝΡέΑΜ

Ήζ┤ ΊΦΕΈιΙΉηΕΉδΝΊΒυΈξ╝ ΉΥ░ΈΛΦ ΈΜνΈξ╕ ΉΓυΈηΝΉζΑ ΉΨ┤ΈΨ╗Ά▓Ν ΉΔζΆ░ΒΊΧιΆ╣ΝΈΠΕ Ά╢ΒΆ╕ΙΊΨΙΈΜν.

Node.js broken ecosystem and rise of AdonisJsΈηΑ ΊΠυΉΛνΊΝΖΉζ┤ ΈπΙΉζΝΉζΕ ΉΓυΈκεΉηκΉΧαΈΜν. ΉδΡΈυ╕ Ήζ╜ΉζΕ ΉΜεΆ░ΕΉζ┤ ΉΩΗΈΜνΈσ┤ Ήζ┤ Ά╕ΑΉζα ΉγΦΉΧ╜ΉζΑ Ήζ┤ Ά▓ΔΉζ┤ΈΜν.

Why not use Express + Sequelize + Config manager + Passport + dotEnv + NodeMailer + Node Validator +30 other modules. How assembling 100+ modules manually can be better than a beautifully pre-configured framework?

ΊΧεΆ╕ΑΈκε Έ▓ΙΉΩφΊΧαΈσ┤ Ήβε ΉΧΙ ΉΥ░ΉΜεΉμι? ΉιΧΈΠΕΆ░Α ΈΡαΉΜεΆ▓ιΈΜν.
