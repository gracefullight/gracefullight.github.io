ECS лҠ” м„ңмҡё лҰ¬м јмқҙ м•„м§Ғ м—Ҷм–ҙм„ң к·ёлҹ°к°Җ, кө¬кёҖл§Ғн•ҙлҸ„ мӮ¬мҡ©н• л§Ңн•ң лҚ°мқҙн„°к°Җ л„Ҳл¬ҙ м Ғм—ҲлӢӨ. (кё°мҙҲ м„ӨлӘ…мқҖ мһҳ лҗҳмһҲлӢӨ. н•ҳм§Җл§Ң Hello World Application мқ„ мҳ¬лҰ¬л Өкі  ECS лҘј м“°лҠ” кұҙ м•„лӢҲлӢҲк№Ң..)

ClusterвҖӢ

Amazon ECS нҒҙлҹ¬мҠӨн„°лҠ” мһ‘м—…мқ„ л°°м№ҳн•  мҲҳ мһҲлҠ” м»Ён…Ңмқҙл„Ҳ мқёмҠӨн„ҙмҠӨмқҳ л…јлҰ¬м Ғ к·ёлЈ№нҷ”мһ…лӢҲлӢӨ. мқҙ кіөмӢқ м„ӨлӘ…мқ„ ліҙкі  Cluster м—җ лҢҖн•ң к°җмқ„ мһЎкё°к°Җ мүҪм§Җ м•Ҡм•ҳлӢӨ.

  • к°„лӢЁнһҲн•ҳл©ҙ Docker Container лҘј мҳ¬лҰ¬лҠ” EC2 Instance мқҙлӢӨ.
  • K8S мқҳ к·ё нҒҙлҹ¬мҠӨн„°мқҙлӢӨ.

  • Task лҠ” мһ‘м—…мқҙлқјкі  лІҲм—ӯлҗҳл©°, н•ҳлӮҳмқҳ Task Definition JSON мқҖ н•ҳлӮҳмқҳ Docker-compose YAML мқҙлқјкі  ліҙмһҗ.

ServiceвҖӢ

Amazon ECS лҠ” лӢЁмқј ECS нҒҙлҹ¬мҠӨн„°м—җм„ң мһ‘м—… м •мқҳм—җ м§Җм •лҗң мҲҳ("мӣҗн•ҳлҠ” к°ңмҲҳ")мқҳ мқёмҠӨн„ҙмҠӨлҘј лҸҷмӢңм—җ мӢӨн–ү л°Ҹ кҙҖлҰ¬н•  мҲҳ мһҲкІҢ н•ҙмӨҚлӢҲлӢӨ. м–ҙл–Ө мқҙмң лЎң мһ‘м—…мқҙ мӢӨнҢЁ лҳҗлҠ” мӨ‘м§ҖлҗҳлҠ” кІҪмҡ° Amazon ECS м„ңл№„мҠӨ мҠӨмјҖмӨ„лҹ¬к°Җ мһ‘м—… м •мқҳмқҳ лӢӨлҘё мқёмҠӨн„ҙмҠӨлҘј мӢңмһ‘н•ҳм—¬ мқҙлҘј лҢҖмІҙн•ҳкі  м„ңл№„мҠӨмқҳ мӣҗн•ҳлҠ” мһ‘м—… мҲҳлҘј мң м§Җн•©лӢҲлӢӨ.

  • Task лҘј мһҗлҸҷмңјлЎң кҙҖлҰ¬н•  мҲҳ мһҲкІҢ н•ҳлҠ” кё°лҠҘ, LB лӮҳ Auto Scaling лӘЁл‘җ м—¬кё°м„ң м Ғмҡ©мқҙ лҗңлӢӨ.
  • лЎӨл§Ғ м—…лҚ°мқҙнҠё мӢңм—җ л°°нҸ¬ л°Ҹ мҳӨнҶ  мҠӨмјҖмқјл§Ғмқ„ лӢҙлӢ№н•ңлӢӨ.
  • K8Sмқҳ м„ңл№„мҠӨ, л””н”ҢлЎңмқҙлЁјнҠё, л Ҳн”ҢлҰ¬м№ҙм„ёнҠёмқҳ м—ӯн•

ліјлҘЁ мғқм„ұ

-v лҳҗлҠ” --volume мңјлЎң Host мқҳ нҸҙлҚ”лҘј Mount н•ҳлҠ” кё°лҠҘмқҙ кјӯ н•„мҡ”н•ңлҚ°, м„Өм • м°Ҫм—җм„  м°ҫкё°к°Җ л„Ҳл¬ҙ нһҳл“Өм—ҲлӢӨ. ECS м—җм„ңлҠ” мһ‘м—… м •мқҳ мғқм„ұ мӢңм—җ н•ҳлӢЁм—җ ліјлҘЁ м¶”к°Җ лҘј кјӯ лЁјм Җ нҒҙлҰӯн•ҙ ліјлҘЁл¶Җн„° м¶”к°Җн•ҙм•јн•ңлӢӨ.

мқҙлҰ„м—” --name мҳөм…ҳ мӮ¬мҡ©н•ҳл“Ҝмқҙ лӢүл„Өмһ„мқ„ л„Јкі  мҶҢмҠӨ кІҪлЎңм—” Host directory кІҪлЎңлҘј л„Јмһҗ.

Container лҘј Task лі„лЎң мғқм„ұн•ҙм•јн•ҳлҠ”лҚ°, к·ёлҹј Task JSON м—җм„ң link мҳөм…ҳмқ„ м—°кІ°н•  мҲҳк°Җ м—ҶлӢӨ. мқҙ кІҪмҡ°м—” м–ҙл–»кІҢ Task Definition мқ„ м§ңм•јлҗҳлӮҳ? EC2 м—җ м ‘к·јн•ҙм„ң л§ӨлІҲ link лҘј мғқм„ұн•ҙм„ң лӢӨмӢң мҳ¬л Өм•јлҗҳлӮҳ?

мқҙ л¶Җл¶„мқ„ н•ҙкІ°н•ҳкё° мң„н•ҙм„  ecs-task-kiteлҘј мӮ¬мҡ©н•ҳкұ°лӮҳ VPC лҘј кө¬м„ұн•ҙ мҲҳлҸҷмңјлЎң м—°кІ°н•ҙ мЈјлҠ” л°©лІ• л°–м—җ м—ҶлӢӨ.

мү¬мҡҙ л°©лІ•мңјлЎң к°Җмһҗл©ҙ DB лҠ” (лӘЁл“  м»Ён…Ңмқҙл„Ҳк°Җ н•ҳлӮҳмқҳ лҚ°мқҙн„°лҘј л°”лқјлҙҗм•јн•ҳлҠ”) RDSлӮҳ ElastiCacheмІҳлҹј AWS мқҳ м„ңл№„мҠӨ мӮ¬мҡ©н•ҳкі  VPC лҘј кө¬м„ұн•ҙ Backend, Frontend лӢЁмқҳ м„ңлІ„л§Ң ECS Task лҘј л§Ңл“Өм–ҙм„ң к°ҖліҖм ҒмңјлЎң лҸҢлҰ¬лҠ” кІҢ мўӢм•„ліҙмқёлӢӨ.

л¬јлЎ  лҸҷкё°нҷ”лҘј н•  мҲҳлҸ„ мһҲлҠ”лҚ°... мӮҪм§Ҳн•  мӢңк°„м—җ лҚ” мһҳ лӮҳмҳЁ нҸ¬мҠӨнҢ…мқ„ кё°лӢӨл ӨліёлӢӨ.
