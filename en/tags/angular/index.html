  • м»ҙнҢҢмқјмӢң к°ҖлІјмӣҢмЎҢлӢӨ.
  • м• лӢҲл©”мқҙм…ҳ нҢЁнӮӨм§Җк°Җ angular/coreм—җм„ң angular/animationмңјлЎң л¶„лҰ¬лҗҳм—ҲлӢӨ.
  • ngIfм—җм„ң else кө¬л¬ёмқ„ м“ё мҲҳ мһҲлӢӨ.
  • angular universialмқҳ кё°лҠҘмқҙ angular/platform-serverлЎң нҶөн•©лҗҳм—ҲлӢӨ.
  • typescript 2.1, 2.2лІ„м „ нҳёнҷҳмқҙ лҗҳм—ҲлӢӨ.

м•һм„ңвҖӢ

м§ҖлӮңмӢңк°„м—җлҠ” Angular with WebpackмңјлЎң ng2 мқҳ кё°ліё мӢӨн–ү нӢҖм—җ лҢҖн•ҙ м•Ңм•„лҙӨлӢӨ. л§ӨлІҲ мқҙл ҮкІҢ м„ёнҢ…мқ„н•ҳл Өл©ҙ м•„л¬ҙлҸ„ ng2 лҘј мүҪкІҢ мӮ¬мҡ©н•ҳм§Җ лӘ»н•  кІғмқҙлӢӨ. лІ„м „лі„ м¶©лҸҢл¬ём ңлҸ„ н•ҙкІ°н•ҙм•јлҗҳкі  м»ҙнҸ¬л„ҢнҠёлҘј мғқм„ұн•  л•Ңл§ҲлӢӨ мЈјмһ…н•ҙмӨҳм•јлҗҳкі  third party лқјмқҙлёҢлҹ¬лҰ¬лҘј м“ё л•ҢлҠ” typings лҘј мӮ¬мҡ©н•ҙ нғҖмһ… мқён„°нҺҳмқҙмҠӨлҘј л„Јм–ҙмӨҳм•јн•ҳкі  мӣ№нҢ© лЎңлҚ”м—җ лҢҖн•ң м •ліҙлҸ„ м°ҫм•„лҙҗм•јн•ҳл©°... (м§ҖлӮңмӢңк°„м—җ н•ҙлҙӨлҚҳ кІғ)

мқҙкұё лӘЁл‘җ н•ҙкІ°н•ң м •л§җ л©Ӣм§„ лӘЁл“Ҳмқё Angular-cli лЎң ng2 project лҘј мӢңмһ‘н•ҙліҙмһҗ.

м„Өм№ҳвҖӢ

Angular-cliлҘј м°ёмЎ°н•ҙлҸ„ лҗҳм§Җл§Ң н•ҳлӮҳм”© н•ҙліҙмһҗ.

лЁјм Җ npm мңјлЎң angular/cli лҘј м „м—ӯмңјлЎң м„Өм№ҳн•ңлӢӨ.

npm install -g @angular/cli

м„Өм№ҳк°Җ мҷ„лЈҢлҗҳл©ҙ ng лқјлҠ” лӘ…л №м–ҙлҘј мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ.

ng --version

н”„лЎңм қнҠё мғқм„ұвҖӢ

ng new н”„лЎңм қнҠёлӘ… лӘ…л №м–ҙлЎң н”„лЎңм қнҠёлҘј мғқм„ұн•ҳл©ҙ лҗңлӢӨ.

ng new н”„лЎңм қнҠёлӘ…

ng new ng2-cli-test --routing

--routing лӘ…л №м–ҙлҠ” кё°ліёмңјлЎң angular лқјмҡ°нҢ…мқ„ app module м—җ л„Јм–ҙмӨҖлӢӨ. angular routing мқ„ мӮ¬мҡ©н•ҳм§Җ м•Ҡмқ„ кІҪмҡ° мҳөм…ҳмқ„ м ңмҷёмӢңнӮӨл©ҙ лҗңлӢӨ.

н”„лЎңм қнҠё мӢӨн–үвҖӢ

мғқм„ұн•ң н”„лЎңм қнҠёлЎң мқҙлҸҷн•ҙ н”„лЎңм қнҠёлҘј мӢӨн–үн•ҙліҙмһҗ

нҢЁнӮӨм§Җ м„Өм№ҳвҖӢ

мүҪкІҢ мӢӨн–үлҗҳм—ҲлӢӨ!

app.module.tsлҘј нҷ•мқён•ҙліҙл©ҙ мһҗлҸҷмңјлЎң import к°Җ лҗҳм–ҙмһҲлӢӨ.

м •л§җ мҳҒлЎұн•ҳлӢӨ. мһҗм„ён•ң generate component мҳөм…ҳмқҖ м—¬кё°м„ң нҷ•мқён•  мҲҳ мһҲлӢӨ.

лқјмҡ°нҢ…вҖӢ

мқҙм ң app-routing.module.ts нҢҢмқјмқ„ м—ҙм–ҙ sub.component лЎң лқјмҡ°нҢ…мқҙ лҗҳкІҢ н•ҙліҙмһҗ.

  {
    path: "",
    children: [],
  },
  { path: "sub", component: SubComponent },

  imports: [RouterModule.forRoot(routes)],
  exports: [RouterModule],
  providers: [],

export class AppRoutingModule {}

м •л§җ angular2 лҘј л°°мҡ°кі  мӢ¶м—ҲлӢӨ.

лҢҖм„ёлҠ” angular2 мҷҖ react к°Җ лҗҳм—Ҳм§Җл§Ң angular2 лҘј м„ нғқн•ңкұҙ angular1 м—җ л°ҳн–Ҳм—Ҳкі  кө¬кёҖмқҙкё° л•Ңл¬ёмқҙм—ҲлӢӨ. к·јлҚ° angular2 лҘј мӮ¬мҡ©н•ҳл Өл©ҙ typescript лҘј м•Ңм•„м•јн•ҳкі , systemjs лҳҗлҠ” webpack мқ„ м•Ңм•„м•јн•ҳл©° rxjs, corejs, zonejs, karma, e2e л“ұ мғҲлЎңмҡҙ кё°мҲ мқ„ л„Ҳл¬ҙ л§Һмқҙ м•Ңм•„м•јлҗҳм—ҲлӢӨ.

лҢҖл¶Җл¶„мқҙ м—¬кё°м„ң мўҢм Ҳ(?)н•ҙ react лӮҳ vue лЎң к°Җл Өкі  н•ҳлҠ” кІғ к°ҷлӢӨ. лҹ¬лӢқ м»ӨлёҢк°Җ мғҒлӢ№н–ҲлҚҳ мқҙмң лҠ” мқҙлҹ¬н–ҲлӢӨ.

  • angular2 мқҳ нҸ¬мҠӨнҠёлҠ” мқҙлЎ л§Ң л§Һм•ҳлӢӨ.
  • мӢӨм „мқ„ м°ҫмңјл©ҙ лІ„м „мқҙ м•ҢнҢҢ лҳҗлҠ” лІ нғҖ лІ„м „мқҙлқј нҳ„мһ¬мҷҖлҠ” нҳёнҷҳмқҙ м•ҲлҗңлӢӨ.
  • м„Өм№ҳлІ•мқҖ м•Ңл ӨмЈјм§ҖлҸ„ м•ҠлҠ”лӢӨ. (лӢӨ nodejs к°ңл°ңмһҗлқј мғқк°Ғн•ҳлҠ” кІғ к°ҷлӢӨ.)
  • м–ҙл–Ө нҢЁнӮӨм§Җк°Җ л¬ҙмҠЁ кё°лҠҘм—җ мӮ¬мҡ©лҗҳлҠ”м§Җ н•ҳлӮҳлҸ„ м•Ңл ӨмЈјм§Җ м•ҠлҠ”лӢӨ.
  • мӢ¬м§Җм–ҙ кіөмӢқ нҷҲнҺҳмқҙм§Җмқҳ starter-kit мқ„ clone н•ҳл©ҙ мҳӨл§Ңк°Җм§Җмқҳ н…ҢмҠӨнҢ… лӘЁл“ҲлҸ„ лӢӨ л”ёл ӨмҷҖ м •мӢ мқҙ нҳјлҜён•ҳлӢӨ.

2016 л…„м—җ мһҗл°”мҠӨнҒ¬лҰҪнҠёлҘј л°°мҡ°лҠ” кё°л¶„мқҖ лҢҖл¶Җл¶„мқҙ мқҙлҹ° кІғ к°ҷлӢӨ.

н•ҳлӮҳн•ҳлӮҳ м°Ёк·јм°Ёк·ј м•Ңм•„к°Җл©° angular2 лЎң л№ м ёліҙмһҗ.

npmвҖӢ

лЁјм Җ angular2 (мқҙн•ҳ ng2)лҠ” npm мңјлЎң м„Өм№ҳлҘј н•ҙм•јн•ңлӢӨ.

npm мқҙ л¬ҙм—Үмқёк°Җ? bower, composer, maven к°ҷмқҖ нҢЁнӮӨм§Җ лӢӨмҡҙлЎңл“ң л§ӨлӢҲм ҖмқҙлӢӨ. лҚ” мүҪкІҢ л§җн•ҳл©ҙ мһҗл°”мҠӨнҒ¬лҰҪнҠё лқјмқҙлёҢлҹ¬лҰ¬лҘј лӢӨмҡҙлЎңл“ңн•ҳкі  кҙҖлҰ¬н•ҙмЈјлҠ” н”„лЎңк·ёлһЁмқҙлқј мғқк°Ғн•ҳмһҗ.

nodejs лӢӨмҡҙлЎңл“ңм—җм„ң л§һлҠ” мңҲлҸ„мҡ° лІ„м „мқ„ лӢӨмҡҙлЎңл“ңн•ҙ м„Өм№ҳн•ҳмһҗ.

package.jsonвҖӢ

npm мңјлЎң мһҗл°”мҠӨнҒ¬лҰҪнҠё лқјмқҙлёҢлҹ¬лҰ¬лҘј лӢӨмҡҙл°ӣкё° мң„н•ҙм„  package.json(м„Өм •нҢҢмқј)мқҙ н•„мҡ”н•ҳлӢӨ. мӣҗн•ҳлҠ” мң„м№ҳм—җ нҸҙлҚ”лҘј л§Ңл“Өмһҗ. (D:\workspace\ng-test)

к·ёлҰ¬кі  package.json нҢҢмқјмқ„ нҸҙлҚ” н•ҳмң„м—җ л§Ңл“ лӢӨ.

  • dependencies : ng2 н”„лЎңм қнҠём—җ мӮ¬мҡ©н•  js library мқҳ мқҙлҰ„кіј лІ„м „мқ„ м ҒлҠ”лӢӨ.
    • @angular/common : ng2 мқҳ кё°ліё лӘЁл“Ҳ
    • @angular/compiler : ng2 мқҳ template мқ„ мң„н•ҙ н•„мҡ”н•ң лӘЁл“Ҳ
    • @angular/core : ng2 мқҳ кё°ліё лӘЁл“Ҳ
    • @angular/forms : ng2 лЎң form мқ„ лӢӨлЈЁкё° мң„н•ң лӘЁл“Ҳ
    • @angular/http : л№„лҸҷкё° м„ңлІ„ нҶөмӢ мқ„ мң„н•ң лӘЁл“Ҳ
    • @angular/platform-browser : ng2 лҘј лёҢлқјмҡ°м ҖлЎң н‘ңмӢңн•ҳкё° мң„н•ң лӘЁл“Ҳ
    • @angular/platform-browser-dynamic : ng2 лҘј лёҢлқјмҡ°м ҖлЎң н‘ңмӢңн•ҳкё° мң„н•ң лӘЁл“Ҳ
    • @angular/router : лқјмҡ°нҢ… кё°лҠҘ лӘЁл“Ҳ
    • core-js : js мқҳ мөңмӢ  л¬ёлІ•мқ„ мӮ¬мҡ©н•ҳкё° мң„н•Ё
    • reflect-metadata : metadata л¬ёлІ•мқ„ мӮ¬мҡ©н•ҳкё° мң„н•Ё
    • rxjs : observables кё°лҠҘмқ„ мӮ¬мҡ©н•ҳкё° мң„н•Ё
    • zone.js : async н•ЁмҲҳмқҳ лҸ„м°© м§Җм җмқ„ м•Ңкё° мң„н•Ё

ng2 мқҳ кё°лҠҘл“Өкіј к·ё кё°лҠҘмқ„ н•ҳмң„лІ„м „ лёҢлқјмҡ°м Җм—җм„ңлҸ„ мӮ¬мҡ©н•ҳкё° мң„н•ң лқјмқҙлёҢлҹ¬лҰ¬л“Өмқ„ нҸ¬н•Ён–ҲлӢӨ.

м„ӨлӘ…вҖӢ

  • imports : мқҙ лӘЁл“Ҳм—җ н•„мҡ”н•ң лӢӨлҘё лӘЁл“Ҳ
  • declarations : лӘЁл“Ҳм—җ мҶҚн•ҳлҠ” л·° нҒҙлһҳмҠӨ, м•өк·Өлҹ¬м—җлҠ” components, directives, pipes лқјлҠ” м„ёк°Җм§Җ л·° нҒҙлһҳмҠӨк°Җ мһҲлӢӨ.
  • bootstrap : л©”мқё Component м§Җм • (root лӘЁл“Ҳл§Ң м§Җм •н•ҙм•јн•Ё)

м»ҙнҸ¬л„ҢнҠёлһҖ л¬ҙм—Үмқёк°Җ? м»ҙнҸ¬л„ҢнҠёлҠ” нҷ”л©ҙ(л·°)лҘј м ңм–ҙн•ҳлҠ” мһҗл°”мҠӨнҒ¬лҰҪнҠё нҒҙлһҳмҠӨмқҙлӢӨ.

app.component.tsвҖӢ

  • template : м–ҙл–Ө н…ңн”ҢлҰҝмқ„ мӮ¬мҡ©н• м§Җ ` кё°нҳёлҘј мӮ¬мҡ©н•ҙ м ҒлҠ”лӢӨ.

м—¬кё°к№Ңм§Җк°Җ ng2 мқҳ к°ҖмһҘ кё°ліём Ғмқё нӢҖмқҙлӢӨ. мқҙм ң мӢӨн–үмқ„ н•ҳкё° мң„н•ҙ л§Ңл§Ңм№ҳ м•ҠмқҖ мһ‘м—…мқҙ лӮЁм•ҳлӢӨ.

TypeScriptвҖӢ

ng2 лҠ” typescript лҘј мЈј м–ём–ҙлЎң мӮ¬мҡ©н•ңлӢӨ. typescript лҠ” javascript мқҳ мғҒмң„ м§‘н•©мқҙлӢӨ. мғҒмң„м§‘н•©мқҙлһҖ л§җмқҙ м–ҙл өлӢӨл©ҙ.. к·ёлғҘ javascript лһ‘ лҳ‘к°ҷлӢӨкі  мғқк°Ғн•ҙлҸ„ лҗңлӢӨ. лҳ‘к°ҷмқҙ мҪ”л”©н•ҙлҸ„ лҗңлӢӨ. кұ°кё°м—җ java мІҳлҹј type мқ„ кіҒл“Өм—¬ мҪ”л“ңлҘј м§Ө мҲҳ мһҲлӢӨ.

н•ҳм§Җл§Ң лёҢлқјмҡ°м Җм—җм„ң мӢӨн–үн•ҳл Өл©ҙ javascript лЎң compile мқ„ н•ҙмӨҳм•јн•ңлӢӨ. мүҪкІҢ java(ts)лЎң м§ңкі  class(js)лЎң м»ҙнҢҢмқјн•ҙм•ј мӢӨн–үлҗҳлҠ” кө¬мЎ°лқј мқҙн•ҙн•ҳмһҗ. к·ёлҹ¬кё° мң„н•ҙ лӘҮк°Җм§Җ лқјмқҙлёҢлҹ¬лҰ¬лҘј npm м—җм„ң м¶”к°ҖлЎң м„Өм№ҳн•ҙмӨҳм•јн•ңлӢӨ.

## нғҖмһ…мҠӨнҒ¬лҰҪнҠё лӢӨмҡҙлЎңл“ң
$ npm install --save-dev typescript

мң„ лӘ…л №м–ҙлҘј мӢӨн–үн•ҳл©ҙ к°ңл°ңлІ„м „(save-dev)мңјлЎң typescript лқјмқҙлёҢлҹ¬лҰ¬к°Җ м„Өм№ҳлҗңлӢӨ. package.json мқ„ ліҙл©ҙ devDependencies мҳөм…ҳ л°‘м—җ мқҳмЎҙм„ұмқҙ м¶”к°Җ лҗң кІғмқ„ ліј мҲҳ мһҲлӢӨ.

tsconfig.jsonвҖӢ

typescript лҘј javascript лЎң м»ҙнҢҢмқјн•ҳкё° мң„н•ҙ кё°ліё мҳөм…ҳмқ„ м„Өм •н•ҙмӨҳм•јн•ңлӢӨ. root м—җ tsconfig.json нҢҢмқјмқ„ л§Ңл“Өмһҗ.

tsconfig.json
{
  "compilerOptions": {
    "target": "es5",
    "experimentalDecorators": true,
    "emitDecoratorMetadata": true
  }
}
  • target : javascript es5 лІ„м „мңјлЎң м»ҙнҢҢмқјмқ„ н•ңлӢӨ.
  • experimentalDecorators : лҚ°мҪ”л Ҳмқҙн„° кё°лҠҘмқ„ мӮ¬мҡ©н•ҳкё° мң„н•ҙ true лЎң м„Өм •н•ңлӢӨ.
  • emitDecoratorMetadata : лҚ°мҪ”л Ҳмқҙн„° кё°лҠҘмқ„ мӮ¬мҡ©н•ҳкё° мң„н•ҙ true лЎң м„Өм •н•ңлӢӨ.

м—¬кё°к№Ңм§Җк°Җ ng2 мқҳ typescript кё°ліё м„Өм •мқҙлӢӨ.

compileвҖӢ

м•„лһҳ лӘ…л №м–ҙлҘј мӢӨн–үн•ңлӢӨ.

$(npm bin)/tsc --rootDir src --outDir dist

мӢӨн–үмқҙ лҗҳкі  dist нҸҙлҚ” м•„лһҳ typescript к°Җ javascript лЎң м»ҙнҢҢмқјлҗң кІҢ ліҙмқёлӢӨ. к·јлҚ° м•„м§Ғк№Ңм§„ м—¬лҹ¬ мҳӨлҘҳк°Җ ліҙмқёлӢӨ. es6 мқҳ кё°лҠҘмқ„ мӮ¬мҡ©н•  мҲҳ м—ҶлӢӨлҠ” мҳӨлҘҳмқёлҚ° мҡ°лҰ¬м—җкІҗ core-js лқјмқҙлёҢлҹ¬лҰ¬к°Җ мһҲмңјлӢҲ typescript м—җ core-js лҘј мӮ¬мҡ©н•ҳкі  мһҲлӢӨкі  м•Ңл ӨмЈјмһҗ.

typingsвҖӢ

лқјмқҙлёҢлҹ¬лҰ¬лҘј мӮ¬мҡ©н•ҳкі  мһҲлӢӨкі  м•Ңл ӨмЈјкё° мң„н•ҙм„  typings лҘј м„Өм№ҳн•ҙм•јн•ңлӢӨ.

npm install --save-dev typings

core-jsвҖӢ

core-js м—җ type мқҙ л“Өм–ҙк°„ interface лҘј typings лЎң м¶”к°Җн•ңлӢӨ.

$(npm bin)/typings install --global --save dt~core-js

typings нҸҙлҚ”мҷҖ typings.json нҢҢмқјмқҙ м¶”к°Җлҗң кІғмқ„ нҷ•мқён•  мҲҳ мһҲлӢӨ. лӢӨмӢң м»ҙнҢҢмқјмқ„ н•ҙліҙл©ҙ мҳӨлҘҳ м—Ҷмқҙ js лЎң м»ҙнҢҢмқј лҗңлӢӨ.

custom scriptsвҖӢ

л§ӨлІҲ $(npm bin)/... лӘ…л №м–ҙлҘј м№ҳкё°лҠ” л„Ҳл¬ҙ к·Җм°®лӢӨ. package.json мқ„ м—ҙм–ҙ лӘ…л №м–ҙлҘј мӨ„мқё script кё°лҠҘмқ„ мӮ¬мҡ©н•ҙліҙмһҗ.

../dist/main.js лҘј index.html м—җ м¶”к°Җн•ҳкі  лёҢлқјмҡ°м Җм—җм„ң м—ҙм–ҙліҙмһҗ.

мқҙ мҳӨлҘҳлҠ” commonjs нҷҳкІҪмқҙ м•„лӢҲм—¬м„ң л°ңмғқн•ңлӢӨ.

commonjs лҠ” л¬ҙм—Үмқёк°Җ? nodejs мҷҖ к°ҷмқҙ require н•ЁмҲҳлҘј мӮ¬мҡ©н•ҙ javascript лҘј к°Җм ёмҳӨлҠ”(import) нҷҳкІҪмқ„ л§җн•ңлӢӨ.

н•ҙкІ°н•ҳкё° мң„н•ҙ Webpack мқ„ м„Өм№ҳн•ҳмһҗ.

WebpackвҖӢ

Webpack мқҖ л¬ҙм—Үмқёк°Җ? лӮҙк°Җ мӣҗн•ҳлҠ” лӘЁл“  нҢҢмқјмқ„ н•ҳлӮҳмқҳ javascript нҢҢмқјлЎң л¶Ҳлҹ¬мҳ¬ мҲҳ мһҲкІҢ н•ҳлҠ” лӘЁл“Ҳ лІҲл“Өлҹ¬лӢӨ.

м„Өм№ҳвҖӢ

webpack мқҙ typescript нҢҢмқјмқ„ лЎңл“ңн•ҳкё° мң„н•ҙм„  typescript loader лӘЁл“Ҳмқ„ м„Өм№ҳн•ҙмӨҳм•јн•ңлӢӨ.

typescript-loaderвҖӢ

npm install --save-dev awesome-typescript-loader

м„Өм№ҳ нӣ„м—җ tsconfig.json нҢҢмқјмқ„ м—ҙм–ҙ webpack мқ„ мӮ¬мҡ©н•ңлӢӨлҠ” мҳөм…ҳмқ„ мӨҳм•јн•ңлӢӨ.

      /angular(\\|\/)core(\\|\/)src(\\|\/)linker/,
      path.resolve(__dirname, "./src"),
      {},
    ),
  ],
  module: {
    loaders: [{ test: /\.ts$/, loaders: ["awesome-typescript-loader"] }],
  },
  resolve: {
    extensions: [".ts", ".js"],
    modules: [path.resolve(__dirname, "node_modules")],
  },
};

м„ӨлӘ…вҖӢ

  • entry : мӣ№нҢ©мқҙ мқҪмқ„ нҢҢмқј
  • output : м–ҙл””лЎң нҢҢмқјмқ„ лӮҙліҙлӮјм§Җ
  • plugins : м–ҙлҠҗ м¶”к°Җ н”Ңлҹ¬к·ёмқёмқ„ мӮ¬мҡ©н• м§Җ
  • module : нҢҢмқјмқ„ к°Җм ёмҳӨлҠ”лҚ° м–ҙл–Ө лӘЁл“Ҳмқ„ мӮ¬мҡ©н• м§Җ
  • resolve : лӘЁл“Ҳмқ„ м–ҙл””м„ң м°ҫмқ„м§Җ

plugins м—җ angular м„Өм •мқ„ мЈјм§Җ м•Ҡмңјл©ҙ мҳӨлҘҳк°Җ л°ңмғқн•ңлӢӨ.
