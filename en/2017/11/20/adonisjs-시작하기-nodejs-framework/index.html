adonisjs мӢңмһ‘н•ҳкё° (nodejs framework)

November 20, 2017 В· 8 min read
Gracefullight
Owner

nodejsлЎң м •л§җ к°„лӢЁн•ң oauth2, jwt л“ұмқҳ мқёмҰқмқ„ мӮ¬мҡ©н•ҳм§Җ м•ҠлҠ” APIлҘј л§Ңл“Өкё°м—җлҠ” expressк°Җ м •л§җ л”ұмқҙлӢӨ.

н•ҳм§Җл§Ң мЎ°кёҲ лҚ” к№”лҒ”н•ң мҪ”л“ңлҘј мӣҗн•ҳкұ°лӮҳ мғҲлЎӯм§Җл§Ң л°ҳліөм Ғмқё кё°лҠҘмқ„ л„Јкё° мң„н•ҙм„  лҚ” нҒ° н”„л Ҳмһ„мӣҢнҒ¬к°Җ н•„мҡ”н–ҲлӢӨ.

мЎ°кұҙвҖӢ

м–ём ңлӮҳ к°ңл°ңмӢңк°„мқҖ л¶ҖмЎұн•ҳкё°м—җ лӮҙк°Җ н•„мҡ”н•ң кё°лҠҘл“Өмқ„ м •лҰ¬н•ҙ ліҙм•ҳлӢӨ.

  • await, asyncлҘј м•„л¬ҙ м„ёнҢ… м—Ҷмқҙ мӮ¬мҡ©н•ҙ Promise chainingм—җм„ң лІ—м–ҙлӮҳкі  мӢ¶лӢӨ. (Babelкіј Webpackмқ„ м§Ғм ‘ м„ёнҢ…н•ҳлҠ” мӢңк°„ кұёлҰ¬лҠ” м§“мқ„ м•Ҳн•ҳкі  мӢ¶лӢӨ.)
  • кё°ліём Ғмқё мӣ№ ліҙм•Ҳкё°лҠҘ (validate, csrf, xss, injection) л°Ҹ API Throttle Request кё°лҠҘмқҙ мһҲм–ҙм•јн•ңлӢӨ.
  • мҶҢм…ң лЎңк·ёмқёмқ„ passportлҘј мӮ¬мҡ©н•ҳм§Җ м•Ҡкі  н•ң мӨ„лЎң лҒқлӮҙкі  мӢ¶м—ҲлӢӨ.
  • JWT мқёмҰқ, Log кё°лҠҘ, Bcrypt м•”нҳёнҷ”лҘј м§Ғм ‘ л¶ҷнһҲкі  мӢ¶м§Җ м•ҠлӢӨ.
  • ORMм—җ л§һкІҢ E-commerce DBлҘј кө¬мЎ°нҷ”н–Ҳкё° л•Ңл¬ём—җ ORMмқҙ кјӯ н•„мҡ”н•ҳлӢӨ.
  • м§ҒкҙҖм Ғмқҙм—¬м•ј н•ңлӢӨ. (Modelкіј Controllerк°Җ мҷ„лІҪнһҲ лӮҳлүҳлҠ” кө¬мЎ°лҘј мӣҗн–ҲлӢӨ.)
  • л¬ём„ңнҷ”к°Җ мҷ„лІҪн•ҙм•јн•ңлӢӨ.
  • м»Өл®ӨлӢҲнӢ°лӮҳ Github Issueк°Җ нҷңм„ұнҷ” лҗҳм–ҙмһҲм–ҙм•јн•ңлӢӨ.
  • кҙҖлҰ¬мһҗ кё°лҠҘмқ„ SSRлЎң н•ҙм•јн•  к°ҖлҠҘм„ұмқҙ мһҲм–ҙ н…ңн”ҢлҰҝ кё°лҠҘмқҙ мһҲм–ҙм•јн•ҳл©°, ejsлҠ” м ҲлҢҖ м“°кі  мӢ¶м§Җ м•Ҡкі  лӢӨлҘё н…ңн”ҢлҰҝмқ„ м§Ғм ‘ м„ёнҢ…н•ҙм•јн•ҳлҠ” мқјмқҖ лҚ”лҚ”мҡұ м—Ҷм–ҙм•јн•ңлӢӨ.

мқҙ мЎ°кұҙмқ„ кұ°лҘҙлӢҲ adonisjs лқјлҠ” мІҳмқҢ ліҙлҠ” н”„л Ҳмһ„мӣҢнҒ¬к°Җ лҲҲм•һм—җ мһҲм—ҲлӢӨ.

м„ұлҠҘвҖӢ

лӮҳмҒҳм§Җ м•ҠлӢӨ. м–ҙм°Ён”ј nginxм—җм„ң reverse proxy мІҳлҰ¬м—җ, auto scaling кұёкәјкі  к·ёлһҳлҸ„ лҚ” л№ лҘё мҶҚлҸ„лҘј мӣҗн•ҳл©ҙ golangмңјлЎң к°Җм•јм§Җ мҷң мқҙ нҸ¬мҠӨнҠёлҘј ліҙкі мһҲлӮҳ!

  • м—¬кё°м„ң мһҗм„ён•ң лӮҙмҡ©мқ„ нҷ•мқён•  мҲҳ мһҲлӢӨ.

нҸүнҢҗвҖӢ

мқҙ н”„л Ҳмһ„мӣҢнҒ¬лҘј м“°лҠ” лӢӨлҘё мӮ¬лһҢмқҖ м–ҙл–»кІҢ мғқк°Ғн• к№ҢлҸ„ к¶ҒкёҲн–ҲлӢӨ.

Node.js broken ecosystem and rise of AdonisJsлһҖ нҸ¬мҠӨнҢ…мқҙ л§ҲмқҢмқ„ мӮ¬лЎңмһЎм•ҳлӢӨ. мӣҗл¬ё мқҪмқ„ мӢңк°„мқҙ м—ҶлӢӨл©ҙ мқҙ кёҖмқҳ мҡ”м•ҪмқҖ мқҙ кІғмқҙлӢӨ.

Why not use Express + Sequelize + Config manager + Passport + dotEnv + NodeMailer + Node Validator +30 other modules. How assembling 100+ modules manually can be better than a beautifully pre-configured framework?

н•ңкёҖлЎң лІҲм—ӯн•ҳл©ҙ мҷң м•Ҳ м“°мӢңмЈ ? м •лҸ„к°Җ лҗҳмӢңкІ лӢӨ.

к°Ғ л©”мҶҢл“ңл“Өмқҙ м—°кІ°лҗҳлҠ” кұҙ лӢӨмқҢкіј к°ҷлӢӨ.

  • index: GET boards
  • create: GET boards/create
  • store: POST boards
  • show: GET boards/:id
  • edit: GET boards/:id/edit

  • destory: DELETE boards/:id

лӘЁлҚё мғқм„ұ нӣ„м—җ лӘҮ к°Җм§Җ м„Өм •мқ„ н•ҙмӨҳм•јн•ңлӢӨ.

class Board extends Model {
  // н…Ңмқҙлё” лӘ…мқ„ ліҖкІҪн•ҙм•јн•  кІҪмҡ°
  // (н…Ңмқҙлё” лӘ…мқҙ лӘЁлҚёлӘ…мқҳ ліөмҲҳнҳ•мқҙ м•„лӢҗ кІҪмҡ°)
  static get table() {
    return "board";
  }

  // кё°ліё м»Өл„Ҙм…ҳмқҙ ліҖкІҪлҗ  кІҪмҡ°
  static get connection() {
    return "mysql";
  }

  // PK м»¬лҹјлӘ…мқҙ idк°Җ м•„лӢҗ кІҪмҡ°
  static get primaryKey() {
    return "uid";
  }

  // PKмқҳ Auto Incrementк°Җ м•„лӢҗ кІҪмҡ°
  static get incrementing() {
    return false;
  }

  // л№„л°ҖлІҲнҳё к°ҷмқҖ м»¬лҹјмқ„ ліҙм—¬мЈјм§Җ м•Ҡм•„м•ј лҗ  кІҪмҡ°
  // мқҙ кІҪмҡ° лӘЁлҚём—җм„ң fetch лҳҗлҠ” first л©”мҶҢл“ңлЎң мҝјлҰ¬л№ҢлҚ”лҘј мӢӨн–үн•ҙм•јлҗңлӢӨ
  static get hidden() {
    return ["password"];
  }

  // мғқм„ұмқј м»¬лҹјмқҙ ліҖкІҪлҗ  кІҪмҡ°
  static get createdAtColumn() {
    return "created_at";
  }

  // м—…лҚ°мқҙнҠёмқј м»¬лҹјмқҙ ліҖкІҪлҗ  кІҪмҡ°
  static get updatedAtColumn() {
    return "updated_at";
  }
}
