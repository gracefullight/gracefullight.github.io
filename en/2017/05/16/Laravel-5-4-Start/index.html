мҷң лқјлқјлІЁмқёк°Җ?вҖӢ

1л“ұмқҙ лҗңм§Җк°Җ 14л…„ 7мӣ”мқҙлӢӨ. мқҙ кёҖмқҖ 17л…„ кёҖмқҙлӢӨ. лӢӨлҘё мһҘм җмқҙ к¶ҒкёҲн•ҳлӢӨл©ҙ лӢӨлҘё нҸ¬мҠӨнҠёлӮҳ stackoverflowм—җм„ң н•ҙкІ°н•ҙмӨ„ кІғ к°ҷлӢӨ.

мқҙ кёҖмқ„ мқҪм–ҙмЈјмӢңлҠ” л¶„л“ӨмқҖ Modern PHPм—җ лҢҖн•ҙ мқҙн•ҙн•ҳкі  мһҲлӢӨкі  мғқк°Ғн–Ҳкё° л–„л¬ём—җ л¬ёлІ•м Ғмқё кІғл“ӨмқҖ м„ӨлӘ…н•ҳм§Җ м•Ҡмқ„ кІғмқҙлӢӨ. лҚ”кө°лӢӨлӮҳ лқјлқјлІЁмқ„ лҸ„мһ…н•ҳл©ҙм„ң мӮҪм§Ҳн•ң лӮҙмҡ©мқ„ м •лҰ¬н•ҳкі  мһҲкё°м—җ, мһ…л¬ёмҡ©мқҖ м ҲлҢҖ м•„лӢҲлӢӨ. кё°ліём ҒмңјлЎң м•„лһҳмҷҖ к°ҷмқҖ PHP к°ңл…җмқ„ м•Ңкі  мһҲм–ҙм•јн•ңлӢӨ.

м„Өм№ҳвҖӢ

PHPвҖӢ

PHP 7 мқҙмғҒмқ„ м„Өм№ҳн•ҙмЈјкі , м•„лһҳ ExtensionмқҖ мӣ¬л§Ңн•ҳл©ҙ м—ҙм–ҙмЈјлҠ” кІғл“ӨмқҙлӢҲ нҒ¬кІҢ мӢ кІҪм“°м§Җ л§җмһҗ.

  • PHP >= 7.0.0
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

ComposerвҖӢ

Composer нҷҲнҺҳмқҙм§Җм—җм„ң лӢӨмҡҙл°ӣкі  м „м—ӯм„Өм •л§Ң н•ҙмЈјл©ҙ лҗңлӢӨ.

Laravel CliвҖӢ

Laravel лӘ…л №м–ҙлҘј мӮ¬мҡ©н•  мҲҳ мһҲкІҢ composerлЎң м „м—ӯ м„Өм№ҳн•ҳмһҗ.

composer global require "laravel/installer"

н”„лЎңм қнҠё мғқм„ұвҖӢ

мҙҲкё° мғқм„ұвҖӢ

laravel new н”„лЎңм қнҠёлӘ…

мң„ лӘ…л №м–ҙлҘј мӢӨн–үн•ҳл©ҙ, м•Ңм•„м„ң composer нҢЁнӮӨм§Җк№Ңм§Җ м„Өм№ҳлҗңлӢӨ.

clone н–Ҳмқ„ л•ҢвҖӢ

нҷҳкІҪ м„Өм • нҢҢмқјкіј н”„лЎңм қнҠё нӮӨк°Җ м—ҶмңјлҜҖлЎң м Ғм Ҳн•ҳкІҢ м„ёнҢ…н•ҙмӨҳм•јн•ңлӢӨ.

## нҢЁнӮӨм§Җ мқҳмЎҙм„ұ м„Өм№ҳ
$ composer install

## .env нҢҢмқј ліөмӮ¬ (м„Өм •нҢҢмқј мғқм„ұ)
## post-root-package-install scriptsлЎң л“Өм–ҙк°Җ мһҲкёҙн•ңлҚ°, ліөмӮ¬к°Җ м•Ҳ лҗҳлҠ” кІҪмҡ°
$ cp .env.example .env

## н”„лЎңм қнҠё нӮӨ мғқм„ұ
$ php artisan key:generate

мӢӨн–үвҖӢ

н”„лЎңм қнҠёлҘј мӢӨн–үн•ҙліҙмһҗ.

php artisan serve

мқҙм ң localhost:8000мңјлЎң лқјлқјлІЁ н”„лЎңм қнҠём—җ м ‘к·јмқҙ к°ҖлҠҘн•ҳлӢӨ.

php artisan лӘ…л №м–ҙлҠ” laravelм—җм„ң мӮ¬мҡ©н•ҳлҠ” м»Өл§Ёл“ң мүҳмқҙлӢӨ. php artisanлЎң лӘ…л №м–ҙ лҰ¬мҠӨнҠёк°Җ лӮҳмҳӨкі  php artisan help лӘ…л №м–ҙ лЎң н•ҙлӢ№ лӘ…л №м–ҙмқҳ мҳөм…ҳмқ„ ліј мҲҳ мһҲлӢӨ.

н”„лЎңм қнҠё кө¬мЎ°вҖӢ

лқјлқјлІЁ н•ңкёҖ л©”лүҙм–јмқҙлӮҳ кіөмӢқ л©”лүҙм–јмқ„ ліҙл©ҙ лҗҳлҠ”лҚ° л§Ҳм№ҳ мӮ¬м „мқ„ нҺҙлҶ“м•ҳлҠ”лҚ° м–ҙл–Ө лӢЁм–ҙлҘј м°ҫлҠ”м§Җ к№ҢлЁ№мқҖ лҠҗлӮҢмқҙ л“Ө кІғмқҙлӢӨ.

лӘЁл“  н”„л Ҳмһ„мӣҢнҒ¬лҠ” Model, View, Controller, Routingл§Ң м•Ңл©ҙ лҒқмқҙлқјлҠ” кІғмқ„ мһ мӢң кё°м–ө мҶҚм—җм„ң кәјлӮҙліҙкі  л”ұ 3к°Җм§Җ кІҪлЎңл§Ң кё°м–өн•ҳмһҗ.

  • Route: routes/
  • Controller: app/Http/Controllers/
  • View : resources/views/

лӘЁлҚёмқҖ м–ҙл”Ём–ҙ? лқјкі  мғқк°Ғн•  кІғ к°ҷмқҖлҚ° Modelмқ„ мҷ„лІҪн•ҳкІҢ мӮ¬мҡ©н•ҳл Өл©ҙ ORMмқ„ мҚЁм•јн•ҳкі  DBлҘј ORMм—җ л§һкІҢ м •к·ңнҷ”н•ҙм•јн•ҳл©° мқҙ мһ‘м—…мқҖ мІҳмқҢ м§„н–үн•ҳкё°м—җ кі нҶөмҠӨлҹҪлӢӨ. лӢӨмқҢ нҸ¬мҠӨнҠём—җ м§„н–үн•ҙліҙмһҗ.

RouteвҖӢ

лӘЁл“  мӣ№ н”„л Ҳмһ„мӣҢнҒ¬мқҳ кё°ліёмқҖ RoutingмқҙлӢӨ. routes нҸҙлҚ”м—җ кё°ліёмңјлЎң api, channels, console, web нҢҢмқјмқҙ ліҙмқҙлҠ”лҚ° к·ё мӨ‘ 2к°Җм§Җл§Ң м•Ңл©ҙ лҗңлӢӨ.

  • api.php: Token мқёмҰқмқҙ н•„мҡ”н•ң лқјмҡ°н„°лЎң /api/{route} лЎң м ‘к·јмқҙ к°ҖлҠҘн•ҳлӢӨ.
  • web.php: кё°ліём Ғмқё RouteмқҙлӢӨ. GETмқҙ м•„лӢҢ лӢӨлҘё л©”мҶҢл“ңлҠ” CSRF Tokenмқҙ мһҲм–ҙм•јл§Ң нҳём¶ңмқҙ к°ҖлҠҘн•ҳлӢӨ.

ControllerвҖӢ

Routing мҡ”мІӯмқ„ мІҳлҰ¬н•ҳлҠ” лЎңм§Ғмқҙ л“Өм–ҙмһҲлҠ” л¶Җл¶„мқҙлӢӨ.

ViewвҖӢ

Controllerм—җм„ң мІҳлҰ¬лҗң лҚ°мқҙн„°лҘј н”Ңлһ«нҸјм—җ ліҙм—¬мЈјлҠ” л¶Җл¶„мқҙлӢӨ.

н”„лЎңм қнҠё м„ёнҢ…вҖӢ

App м„Өм •вҖӢ
