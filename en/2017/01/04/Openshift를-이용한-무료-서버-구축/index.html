к°ңмҡ”вҖӢ

кө¬кёҖл§Ғмқ„ нҶөн•ҙ м–»мқҖ Openshift мһҗлЈҢл“ӨмқҖ мөңмӢ  лІ„м „мқҙ м•„лӢҲлқј мӮ¬мҡ©н•  мҲҳ м—ҶлӢӨ.

Google CloudлӮҳ Heroku, AWSлҘј мӮ¬мҡ©н•ҳм—¬ н…ҢмҠӨнҠёмҡ© м„ңлІ„лҘј л§Ңл“Ө мҲҳ мһҲм§Җл§Ң л¬ҙлЈҢ кё°к°„мқҙ лӘЁл‘җ м •н•ҙм ё мһҲлӢӨ. RedHatм—җм„ң мҡҙмҳҒн•ҳлҠ” 100% л¬ҙлЈҢ нҒҙлқјмҡ°л“ңмқё мҳӨн”ҲмӢңн”„нҠёмқҳ мөңмӢ  лІ„м „мқ„ мӮ¬мҡ©н•ҙліҙмһҗ.

к°Җмһ…вҖӢ

GithubлЎң лЎңк·ёмқёвҖӢ

openshiftмқҳ мғҲлЎңмҡҙ лІ„м „мқҖ Github м•„мқҙл””лЎңл§Ң к°Җмһ…мқҙ к°ҖлҠҘн•ҳлӢӨ. Github м•„мқҙл””лҘј лЁјм Җ мғқм„ұн•ң нӣ„ Login With GitHub лІ„нҠјмқ„ нҒҙлҰӯн•ңлӢӨ.

мӢ мІӯм„ң мһ‘м„ұвҖӢ

мӢ мІӯм„ң мһ‘м„ұ нӣ„ мһ…л Ҙн•ң Email мЈјмҶҢлЎң мҳӨлҠ” мҠ№мқё л©”мқјмқ„ кё°лӢӨл Өм•јн•ңлӢӨ.

мҠ№мқё л°Ҹ лЎңк·ёмқёвҖӢ

The OpenShift Teamм—җкІҢ Welcome to the OpenShift Online (Next Gen) Developer Preview лһҖ м ңлӘ©мңјлЎң л©”мқјмқ„ л°ӣмңјл©ҙ openshiftлҘј мқҙмҡ©н•  мҲҳ мһҲлӢӨ.

мғҒлӢЁ л©”лүҙмқҳ My AccountлҘј нҒҙлҰӯ нӣ„ (NEXT GEN) WEB CONSOLE лІ„нҠјмқ„ нҒҙлҰӯн•ңлӢӨ.

лӢӨмқҢ лӢЁкі„л¶Җн„°лҠ” OpenShiftмқҳ мҠ№мқёмқҙ лӮҳм•јм§Җл§Ң м§„н–үн•  мҲҳ мһҲлӢӨ. н•ҳлЈЁм •лҸ„ м•Ҳм—җ мҠ№мқё л©”мқјмқҙ мҳӨлӢҲ кё°лӢӨл Өліҙмһҗ.

н”„лЎңм қнҠё мғқм„ұвҖӢ

н”„лЎңм қнҠёлҠ” н•ҳлӮҳмқҳ нҒҙлқјмҡ°л“ңлҘј мӢ мІӯн•ңлӢӨкі  ліҙл©ҙ лҗңлӢӨ.

лЎңк·ёмқё нӣ„ New Project лІ„нҠјмқ„ нҒҙлҰӯн•ңлӢӨ. кі лҜјн•ҳм§Җ л§җкі  м Ғм–ҙлҸ„ лҗңлӢӨ.

мӣ№м„ңлІ„ мғқм„ұвҖӢ

н”„лЎңм қнҠё мғқм„ұмқҙ лҒқлӮҳл©ҙ мқҙлҜём§ҖлҘј м„ нғқн•ҳлҠ” м°Ҫмқҙ лӮҳмҳЁлӢӨ. кІҖмғүм°Ҫм—җ мӣҗн•ҳлҠ” мқҙлҜём§ҖлҘј кІҖмғү нӣ„ м„ нғқн•ңлӢӨ.

м„ңлІ„ мқҙлҜём§Җ м¶”к°ҖвҖӢ

mongodbлҘј мӮ¬мҡ©мӢң nodejs-mongo-persistent мқҙлҜём§ҖлҘј мӮ¬мҡ©н•ҳкі , nodejsл§Ң н•„мҡ”н•  кІҪмҡ° nodejs:4 мқҙлҜём§ҖлҘј мӮ¬мҡ©н•ҳл©ҙ лҗңлӢӨ. мқјлӢЁ nodejs мӣ№м„ңлІ„л§Ң н•„мҡ”н•ҳкё°м—җ н•ҙлӢ№ мқҙлҜём§ҖлҘј м„ нғқн–ҲлӢӨ.

Git RepositoryлҘј мһ…л Ҙн•ҙм•јн•ҳлҠ”лҚ°, Githubм—җ к°Җм§Җкі  мһҲлҠ” мһҗмӢ мқҳ nodejs applicationмқҙ мһҲлӢӨл©ҙ к·ё мЈјмҶҢлҘј л“ұлЎқн•ҳл©ҙлҗҳкі  к·ёл Үм§Җ м•ҠмқҖ кІҪмҡ° Try ItлІ„нҠјмқ„ нҒҙлҰӯн•ҙ Sample RepoлҘј м¶”к°Җн•ҳмһҗ.

мғҳн”Ң мҶҢмҠӨлҠ” expressмҷҖ ejsлҘј мӮ¬мҡ©н•ҳкі  мһҲлӢӨ.

Show advenced routing... лІ„нҠјмқ„ нҒҙлҰӯн•ҳл©ҙ лҸ„л©”мқё м„ёнҢ…, Webhook л“ұ кі кёү м„Өм •мқ„ н•  мҲҳ мһҲлҠ”лҚ° Resource Limitм—җм„ң л©”лӘЁлҰ¬лҘј лҠҳлҰҙ мҲҳ мһҲлӢӨ. (кё°ліё 512MB)

м¶”нӣ„ л©”лӘЁлҰ¬ мҡ©лҹү нҷ•мқёмқҖ к°ҖлҠҘн•ҳлӮҳ, нҷ•мһҘн•ҳлҠ” л¶Җл¶„мқҖ м•„м§Ғ лӘ» м°ҫм•ҳлӢӨ. (м—Ҷмқ„м§ҖлҸ„) лҜёлҰ¬ л©”лӘЁлҰ¬лҘј 1GBлЎң лҠҳл ӨмЈјкі  мӢңмһ‘н•ҳмһҗ

Fork Sample RepositoryвҖӢ

мқҙм ң node м„ңлІ„лҘј мҲҳм •н•  мҲҳ мһҲкІҢ лӮҳмқҳ GithubлЎң мҳ®кё°лҠ” мһ‘м—…мқҙ н•„мҡ”н•ҳлӢӨ. Sample Repoм—җ м ‘мҶҚн•ҙ н”„лЎңм қнҠёлҘј Forkн•ңлӢӨ.

Change Source RepositoryвҖӢ

Builds > Builds л©”лүҙлЎң л“Өм–ҙк°Җ лӮҙ applicationмқ„ м„ нғқн•ң л’Ө мҡ°мёЎ Actionsм—җм„ң Edit лІ„нҠјмқ„ нҒҙлҰӯн•ңлӢӨ.

Source Repository URLм—җ мң„м—җм„ң Forkн•ң Git RepositoryлҘј л“ұлЎқн•ңлӢӨ. к·ёлҰ¬кі  мҡ°мёЎ GitHub Webhooks URLмқ„ ліөмӮ¬н•ң л’Ө м ҖмһҘн•ңлӢӨ.

Webhooks URL л“ұлЎқвҖӢ

forkлҗң лӮҙ RepositoryлЎң мҷҖм„ң Settings > Webhooks л©”лүҙлЎң л“Өм–ҙк°Җ Add webhook лІ„нҠјмқ„ нҒҙлҰӯн•ңлӢӨ.

RebuildвҖӢ

Builds > Builds л©”лүҙлЎң лҸҢм•„к°Җ Start Build лІ„нҠјмқ„ нҒҙлҰӯн•ңлӢӨ.

л№Ңл“ңк°Җ мҷ„лЈҢлҗҳл©ҙ лІ„м „мқҙ #2лЎң ліҙмқҙкі , лӮҙ Git RepoмҷҖ м—°кІ°лҗң лІ„м „мқҙ мғқм„ұлҗңлӢӨ.
