лҰ¬м•ЎнӢ°лёҢ н”„лЎңк·ёлһҳл°ҚмқҖ к°ңмқём ҒмңјлЎң л№„лҸҷкё° н”„лЎңк·ёлһҳл°Қкіј н•ЁмҲҳнҳ• н”„лЎңк·ёлһҳл°Қмқҳ мў…м°©м§Җлқјкі  мғқк°Ғн•ңлӢӨ. Stage1 DraftлЎң м ңм•Ҳлҗҳм–ҙ лҚ” мқҙмғҒ н”јн•  мҲҳ м—ҶлҠ” Observable мқ„ м•Ңм•„ліҙмһҗ.

м •мқҳвҖӢ

мң„нӮӨн”јл””м•„м—җм„  лӢӨмқҢкіј к°ҷмқҙ м •мқҳлҗҳм–ҙмһҲлӢӨ.

reactive programming is a declarative programming paradigm concerned with data streams and the propagation of change. лҚ°мқҙн„° мҠӨнҠёлҰјкіј ліҖнҷ”мқҳ м „нҢҢм—җ мӨ‘м җмқ„ л‘” н”„лЎңк·ёлһҳл°Қ нҢЁлҹ¬лӢӨмһ„

2011л…„ нғңмҙҲм—җ RX мқҳ к°ңл…җмқ„ л§Ңл“  MS л¬ём„ңм—җм„ңлҠ” Reactive eXtensions лӢӨмқҢкіј к°ҷлӢӨ.

Reactive Extensions (Rx) is a library for composing asynchronous and event-based programs using observable sequences and LINQ-style query operators. мҳөм ёлІ„лё” мӢңнҖҖмҠӨмҷҖ л§ҒнҒҗ мҝјлҰ¬ м—°мӮ°мһҗлҘј мӮ¬мҡ©н•ҳлҠ” л№„лҸҷкё°, мқҙлІӨнҠё кё°л°ҳ н”„лЎңк·ёлһҳл°Қ лқјмқҙлёҢлҹ¬лҰ¬

мЎ°кёҲ лҚ” л””н…Ңмқјн•ҳкІҢ л§җн•ҳл©ҙ лҚ°мқҙн„°мҠӨнҠёлҰјмқ„ Pulling л°©мӢқмқҳ мқҙн„°л Ҳмқҙн„° нҢЁн„ҙмқё IEnumerable<T>/IEnumerator<T> лЎң л§Ңл“Өм–ҙ Pushing л°©мӢқмқҳ мҳөм ҖлІ„ нҢЁн„ҙмқё IObservable<T>/IObserver<T>лЎң м „нҢҢ/кө¬лҸ…н•ҳлҠ” кІғмқҙлӢӨ.

RxлҘј лӘЁл‘җ мқҙн•ҙн•ң л’Ө мқҙ м •мқҳлҘј ліҙл©ҙ м–ҙм©ң мқҙл ҮкІҢ к№”лҒ”н•ҳкІҢ н•ң мӨ„лЎң мқҙ лӮҙмҡ©мқ„ лӢӨ лӢҙм•ҳмқ„к№Ң? лһҖ мғқк°Ғмқҙ л“ңлҠ”лҚ°, мІҳмқҢліҙлҠ” мһ…мһҘм—җм„  л№„лҸҷкё°мқё кұҙ м•ҢкІ л„Ө м •лҸ„лЎңл§Ң мқҙн•ҙк°Җ лҗҳлҠ” л“Ҝн•ҳлӢӨ. м–ҙл Өмҡҙ кІҢ лӢ№м—°н•ҳлӢӨ. л©ҖнӢ°м“°л Ҳл“ң н”„лЎңк·ёлһҳл°Қмқ„ мІҳмқҢ л°°мҡё л•Ңмқҳ к°җм •мқ„ мғқк°Ғн•ҙліҙмһҗ.

мҡ©м–ҙвҖӢ

  • л№„лҸҷкё° н”„лЎңк·ёлһҳл°Қ: мқҙ л¬ём„ңлҘј ліј м •лҸ„л©ҙ м„ӨлӘ…мқҙ н•„мҡ” м—Ҷмқ„ кІғ к°ҷлӢӨ.
  • н•ЁмҲҳнҳ• н”„лЎңк·ёлһҳл°Қ: н•ЁмҲҳлҘј FirstClass лЎң м·Ёкёүн•ҙ нҢҢлқјлҜён„°, лҰ¬н„ҙ, ліҖмҲҳм—җ н• лӢ№ к°ҖлҠҘн•ҳл©° н•ЁмҲҳ н•©м„ұмқҙ к°ҖлҠҘн•ҳкі  мӣҗн•ҳлҠ” мӢңм җм—җ нҳём¶ңмқҙ мүҪлӢӨ.
  • лҚ°мқҙн„°мҠӨнҠёлҰј: file stream, event stream, http stream мқҳ м№ңкө¬лЎң data мӢқ н‘ңнҳ„
  • мқҙн„°л Ҳмқҙн„°: м„ӨлӘ… н•„мҡ” м—Ҷмқ„ л“Ҝ.
  • мҳөм ёлІ„: л””мһҗмқёнҢЁн„ҙ мұ…мқҳ мҳөм ёлІ„нҢЁн„ҙ лӢЁмӣҗ м°ёмЎ°, мһҗл°”мҠӨнҒ¬лҰҪнҠём—җм„  мқҙлҜё мқҙлІӨнҠёлҰ¬мҠӨл„Ҳк°Җ мҳөм ёлІ„ нҢЁн„ҙмқҙлӢӨ.
  • мҳөм ёлІ„лё”: нҠ№м • к°қмІҙлҘј кҙҖм°°н•ҳлҠ” мҳөм ҖлІ„м—җкІҢ м—¬лҹ¬ мқҙлІӨнҠёлӮҳ к°’мқ„ м „нҢҢн•ҳлҠ” м—ӯн• 
  • н’Җ: лҚ°мқҙн„°лҘј л°ӣмқ„м§Җ кІ°м •
  • н‘ёмү¬: лҚ°мқҙн„°лҘј ліҙлӮјм§Җ кІ°м •
  • мӢұкёҖ: н•ҳлӮҳмқҳ к°’мқҙлӮҳ мқҙлІӨнҠёлҘј лӢӨлЈё
  • л©ҖнӢ°н”Ң: м—¬лҹ¬ к°ңмқҳ к°’мқҙлӮҳ мқҙлІӨнҠёлҘј лӢӨлЈё

мқҙ лӘЁл“  мҡ©м–ҙлҘј н•ҳлӮҳмқҳ н‘ңлЎң м •лҰ¬н•ҳл©ҙ лӢӨмқҢкіј к°ҷлӢӨ.

мӢұкёҖл©ҖнӢ°н”Ң
н’Җн•ЁмҲҳмқҙн„°л Ҳмқҙн„°
н‘ёмү¬н”„лЎңлҜёмҠӨмҳөм ёлІ„лё”

нҢЁн„ҙвҖӢ

  • мҳөм ёлІ„лё”мқҳ ліҖмҲҳ мҠӨнғҖмқјмқҖ мқҙлҰ„ л’Өм—җ const click$ мІҳлҹј $мқ„ л’Өм—җ л¶ҷнҳҖмЈјлҠ”кІҢ м •нҳ•нҷ”лҗҳм–ҙмһҲлӢӨ.
  • subscribe лҠ” next, error, completeлҘј нҢҢлқјлҜён„°лЎң л°ӣлҠ”лӢӨ.

л§Ҳлё” лӢӨмқҙм–ҙк·ёлһЁвҖӢ

л§Ҳлё” лӢӨмқҙм–ҙк·ёлһЁмқҖ м—°мӮ°мһҗлҘј мүҪкІҢ мқҙн•ҙн•ҳкё° мң„н•ҙ, мҳөм ёлІ„лё”мқ„ н…ҢмҠӨнҠён•ҳкё°мң„н•ҙ лҸ„мӢқнҷ”лҗң лӢӨмқҙм–ҙк·ёлһЁмқҙлӢӨ. нғҖмқҙл°Қкіј к°’мқҳ ліҖнҷ”лҘј н•ң лҲҲм—җ нҢҢм•…н•  мҲҳ мһҲлӢӨ.

  • мң„м—җ мӨ„мқҖ input мқҙкі  м—°мӮ°мһҗлҘј л§ҢлӮҳкі лӮҳл©ҙ м•„лһҳ мӨ„м—җм„ң output мқҙ м–ҙл–»кІҢ л°”лҖҢлҠ”м§Җ ліҙм—¬м§„лӢӨ.
  • к°ҖлЎң мӨ„мқҖ н•ҳлӮҳмқҳ мҳөм ёлІ„лё”мқҙлӢӨ.
  • мӨ„мқҳ | нҢҢмқҙн”„лҠ” кө¬лҸ… мҷ„лЈҢлҘј лӮҳнғҖлӮёлӢӨ.

мҶҢмҠӨ

мғқм„ұвҖӢ

к°ҖмһҘ к°„лӢЁн•ң мҳөм ёлІ„лё”мқ„ л§Ңл“Өкі  нҷ•мқён•ҙліҙмһҗ.

const { Observable } = require("rxjs");

const test$ = Observable.create((observer) => {
  console.log("create");
  observer.next(1);
  observer.next(2);
  observer.complete();
  console.log("done");
});

test$.subscribe(
  (item) => {
    console.log(item);
  },
  (error) => {},
  () => {
    console.log("complete");
  },
);

/*
create
1
2
complete
done
*/

к°„лӢЁн•ҳм§Җл§Ң мқҙн„°л Ҳмқҙн„°мқҙл©ҙм„ң кө¬лҸ…к°ҖлҠҘн•ҳлӢӨлҠ” кұё нҷ•мқён•  мҲҳ мһҲлӢӨ.

кө¬лҸ…н•ҙм ңвҖӢ

мҳөм ёлІ„лё”мқҳ лҰ¬н„ҙн•ЁмҲҳлЎң кө¬лҸ…н•ҙм ң мҪңл°ұмқ„ м§Җмӣҗн•ңлӢӨ, мҪңл°ұмқҙ н•„мҡ”н•ҳм§Җ м•ҠлӢӨл©ҙ unsubscribe() лҘј нҳём¶ңн•ҙмЈјкё°л§Ң н•ҳл©ҙ лҗңлӢӨ.

const test$ = Observable.create((observer) => {
  const interval = setInterval(() => {
    console.log("test");
  }, 1000);

  return () => {
    clearInterval(interval);
  };
});

const subscription = test$.subscribe();
subscription.unsubscribe();

нҢҢмқҙнҚјлё” м—°мӮ°мһҗвҖӢ

Pipeable м—°мӮ°мһҗлҠ” мҳөм ёлІ„лё” мқёмҠӨн„ҙмҠӨлҘј pipe н•ЁмҲҳ м•Ҳм—җм„ң лӢӨлЈ° мҲҳ мһҲлҠ” м—°мӮ°мһҗмқҙлӢӨ. кё°ліём ҒмңјлЎң rxjs/operators лқјмқҙлёҢлҹ¬лҰ¬ м•Ҳм—җ л“Өм–ҙмһҲлӢӨ.

const { map } = require("rxjs/operators");

const test$ = Observable.create((observer) => {
  observer.next(1);
  observer.next(2);
  observer.complete();
});

test$.pipe(map((value) => value * 2)).subscribe((item) => console.log(item));

/*
2
4
*/

мҶҢмҠӨ мҳөм ёлІ„лё”м—җм„ң л°ңн–үлҗң к°’мқ„ мӣҗн•ҳлҠ”лҢҖлЎң л°”кҝҖ мҲҳ мһҲлӢӨ.

м—°мӮ°мһҗвҖӢ

ofвҖӢ

args мҲңм„ңлҢҖлЎң к°’мқ„ л°ҳнҷҳн•ңлӢӨ.

fromвҖӢ

  • Observable
  • Array
  • Promise
  • Iterable
  • String
  • ArrayLike

мң„ нғҖмһ…мқ„ мҳөм ёлІ„лё”лЎң ліҖнҷҳн•ҙмӨҖлӢӨ.

fromEventвҖӢ

EventEmitter нҒҙлһҳмҠӨмқҳ к°қмІҙмҷҖ мЎ°н•©н•ҳкұ°лӮҳ лёҢлқјмҡ°м Җмқҳ мқҙлІӨнҠёлҘј мҳөм ёлІ„лё”лЎң л°”кҝҖ л•Ң мӮ¬мҡ©н•ңлӢӨ.

deferвҖӢ

нҢ©нҶ лҰ¬ н•ЁмҲҳлЎң мҳөм ёлІ„лё”мқ„ мғқм„ұн•ң нӣ„ кө¬лҸ…н•ңлҠҗ мӢңм җм—җ нҢ©нҶ лҰ¬ н•ЁмҲҳлҘј нҳём¶ңн•ҙ мқҙлҜё мғқм„ұн•ң мҳөм ёлІ„лё”мқ„ лҰ¬н„ҙл°ӣм•„ кө¬лҸ…н•ңлӢӨ.

from кіјмқҳ м°ЁмқҙлҠ” лӢӨлҰ„кіј к°ҷлӢӨ.

  • from
    • н”„лЎңлҜёмҠӨ лӮҙл¶Җ кө¬нҳ„л¶Җк°Җ м–ём ң мӢӨн–үлҗҳлҚҳм§Җ мғҒкҙҖ м—Ҷмқ„ л•Ң
    • мқҙлҜё мӢӨн–ү мӨ‘мқҙкұ°лӮҳ мҷ„лЈҢн•ң н”„лЎңлҜёмҠӨлҘј мҳөм ёлІ„лё”лЎң л§Ңл“Ө л•Ң
  • defer
    • мҳөм ёлІ„лё”мқ„ кө¬лҸ…н•ҳлҠ” мӢңм җм—җ н”„лЎңлҜёмҠӨлҘј мғқм„ұн•ҙ кө¬нҳ„л¶Җк°Җ мӢӨн–үлҗҳм–ҙм•јн•  л•Ң
    • н”„лЎңлҜёмҠӨ к°қмІҙ мғқм„ұ мӢңм җмқҙ кө¬лҸ…н•ҳлҠ” мӢңм җмқҙм—¬м•јн•  л•Ң

rangeвҖӢ

лІ”мң„ м§Җм • нӣ„ к·ё к°’мқ„ мҲңм„ңлҢҖлЎң л°ңн–үн•ңлӢӨ. л°ҳліөл¬ёмқҙ н•„мҡ”н•  л•Ң мӮ¬мҡ©лҗңлӢӨ.

intervalвҖӢ

ms лӢЁмң„лЎң к°’мқ„ л°ңн–үн•ңлӢӨ.

timerвҖӢ

нҢҢлқјлҜён„°к°Җ н•ҳлӮҳмқј кІҪмҡ° ms мқҙнӣ„м—җ н•ң лІҲ к°’мқ„ л°ңн–үн•ҳкі , л‘җ к°ңмқј кІҪмҡ° ms мқҙнӣ„м—җ л‘җлІҲ м§ё нҢҢлқјлҜён„°л§ҢнҒј мЈјкё°м ҒмңјлЎң к°’мқ„ л°ңн–үн•ңлӢӨ.

emptyвҖӢ

к°’ л°ңн–ү нӣ„ мӨ‘к°„м—җ л©Ҳм¶°м•јн•ҳлҠ” мғҒнҷ©м—җ мӮ¬мҡ©н•ңлӢӨ. мқҙ н•ЁмҲҳл§Ң мӮ¬мҡ©н•ҳм§ҖлҠ” м•Ҡкі  лӢӨлҘё н•ЁмҲҳлӮҳ м—°мӮ°мһҗмҷҖ мЎ°н•©н•ҙм„ң complete н•ЁмҲҳлҘј нҳём¶ңн•ҙм•ј н•  л•Ң мӮ¬мҡ©лҗңлӢӨ. мҰү, л°”лЎң кө¬лҸ…мқ„ мҷ„лЈҢн•ҙм•јлҗ л•Ң мӮ¬мҡ©лҗңлӢӨ.

// мғҒмҲҳлЎң мӮ¬мҡ©лҗңлӢӨ.
const { EMPTY } = require("rxjs");

neverвҖӢ

м•„л¬ҙ кІғлҸ„ н•ҳм§Җ м•Ҡкі  мҳөм ёлІ„лё” мғқм„ұмқҙ н•„мҡ”н•  л•Ң мӮ¬мҡ©лҗңлӢӨ.

// мғҒмҲҳлЎң мӮ¬мҡ©лҗңлӢӨ.
const { NEVER } = require("rxjs");

throwErrorвҖӢ

мҳөм ёлІ„лё”лЎң к°’мқ„ л°ңн–үн•ҳлӢӨк°Җ м—җлҹ¬лҘј л°ңмғқмӢңнӮӨкі  мў…лЈҢн•ҙм•јн•ҳлҠ” мғҒнҷ©м—җ мӮ¬мҡ©н•ңлӢӨ.

filterвҖӢ

мЈјлЎң нҢҢмқҙнҚјлё” м—°мӮ°мһҗмҷҖ м—°кІ°н•ҙм„ң мӮ¬мҡ©лҗңлӢӨ.

const { filter } = require("rxjs/operators");
// 1~10 мӨ‘ м§қмҲҳ н•„н„°
range(1, 10)
  .pipe(filter((x) => x % 2 === 0))
  .subscribe((x) => console.log(x));

firstвҖӢ

мІҳмқҢмңјлЎң мқјм№ҳн•ҳлҠ” к°’мқ„ л°ңн–үн•ңлӢӨ. л‘җ лІҲм§ё мқёмһҗлЎң кё°ліё к°’мқ„ мӨ„ мҲҳ мһҲлӢӨ.

lastвҖӢ

л§Ҳм§Җл§үмңјлЎң мқјм№ҳн•ҳлҠ” к°’мқ„ л°ңн–үн•ңлӢӨ. л‘җ лІҲм§ё мқёмһҗлЎң кё°ліё к°’мқ„ мӨ„ мҲҳ мһҲлӢӨ.

takeвҖӢ

м •н•ҙм§„ к°ҜмҲҳл§ҢнҒј кө¬лҸ…н•ҳкі  кө¬лҸ…мқ„ н•ҙм ңн•ңлӢӨ. interval кіј к°ҷмқҙ л¬ҙн•ң л°ҳліөмқҙ мӢӨн–үлҗҳлҠ” м—°мӮ°мһҗмҷҖ к°ҷмқҙ м“°л©ҙ лҗңлӢӨ.

const { take } = require("rxjs/operators");

interval(1000)
  .pipe(take(3))
  .subscribe((x) => console.log(x));

/*
0
1
2
*/

takeUntilвҖӢ

нҠ№м • мқҙлІӨнҠёк°Җ л°ңмғқн•  л•Ңк№Ңм§Җ мҳөм ёлІ„лё”мқ„ кө¬лҸ…н•ҙм•јн•  л•Ң мӮ¬мҡ©н•ңлӢӨ. мҳҲмӢңлЎң ліҙлҠ” кІҢ л№ лҘҙлӢӨ.

interval(1000)
  .pipe(
    take(100),
