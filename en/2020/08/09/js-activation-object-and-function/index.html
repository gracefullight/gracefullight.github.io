мһҗл°”мҠӨнҒ¬лҰҪнҠё нҷңм„ұ к°қмІҙмҷҖ н•ЁмҲҳ

нҷңм„ұ к°қмІҙвҖӢ

  • Activation Object
  • н•ЁмҲҳк°Җ нҳём¶ңлҗҳл©ҙ нҷңм„ұ к°қмІҙк°Җ л§Ңл“Өм–ҙм§„лӢӨ. мқҙлҠ” мҲЁкІЁм§„ лҚ°мқҙн„° кө¬мЎ°лЎң нҳём¶ңлҗң н•ЁмҲҳмқҳ л°ҳнҷҳ мЈјмҶҢмҷҖ мӢӨн–үм—җ н•„мҡ”н•ң м •ліҙлҘј м ҖмһҘн•ҳкі  нҳём¶ңлҗң н•ЁмҲҳм—җ л°”мқёл”©н•ңлӢӨ.
  • CмҷҖ к°ҷмқҖ м–ём–ҙм—җм„ңлҠ” нҷңм„ұ к°қмІҙк°Җ мҠӨнғқм—җ м ҖмһҘлҗҳкі  н•ЁмҲҳк°Җ мў…лЈҢлҗҳкі  л°ҳнҷҳлҗҳл©ҙ мҠӨнғқм—җм„ң м ңкұ°н•ҳм§Җл§Ң мһҗл°”мҠӨнҒ¬лҰҪнҠём—җм„ңлҠ” нҷңм„ұ к°қмІҙлҘј лӢӨлҘё к°қмІҙмҷҖ лҸҷмқјн•ҳкІҢ нһҷм—җ м ҖмһҘн•ңлӢӨ.
  • н•ЁмҲҳк°Җ мў…лЈҢлҗңлӢӨкі  нҷңм„ұ к°қмІҙлҘј мһҗлҸҷмңјлЎң л№„нҷңм„ұнҷ”н•ҳм§Җ м•ҠлҠ”лӢӨ.
  • нҷңм„ұ к°қмІҙлҠ” н•ҙлӢ№ к°қмІҙм—җ лҢҖн•ң м°ёмЎ°к°Җ мһҲлҠ” н•ң кі„мҶҚ мӮҙм•„мһҲмңјл©° лӢӨлҘё к°қмІҙмҷҖ л§Ҳм°¬к°Җм§ҖлЎң к°Җл№„м§Җ м»¬л үн„°м—җ мқҳн•ҙ мІҳлҰ¬лҗңлӢӨ.

нҷңм„ұ к°қмІҙ лӮҙл¶Җ м •ліҙвҖӢ

  • н•ЁмҲҳ к°қмІҙм—җ лҢҖн•ң м°ёмЎ°
  • caller н•ЁмҲҳмқҳ нҷңм„ұ к°қмІҙм—җ лҢҖн•ң м°ёмЎ° (мқҙ м°ёмЎ°лҠ” return мқҙ мӢӨн–үнқҗлҰ„мқ„ caller лЎң лҸҢлҰҙ л•Ң мӮ¬мҡ©лҗңлӢӨ.)
  • н•ЁмҲҳ нҳём¶ңмқҙ лҒқлӮң л’Ө мӢӨн–үмқ„ мһ¬к°ңн•ҳкё° мң„н•ҙ н•„мҡ”н•ң м •ліҙ (лҢҖк°ң н•ЁмҲҳ нҳём¶ңл¬ё л°”лЎң лӢӨмқҢ лӘ…л №м–ҙмқҳ мЈјмҶҢ)
  • нҢҢлқјлҜён„°м—җ мқҳн•ҙ мҙҲкё°нҷ”лҗҳлҠ” н•ЁмҲҳ л§Өк°ңліҖмҲҳ
  • undefinedлЎң мҙҲкё°нҷ”лҗң н•ЁмҲҳ ліҖмҲҳл“Ө
  • н•ЁмҲҳк°Җ ліөмһЎн•ң н‘ңнҳ„мӢқмқ„ кі„мӮ°н•ҳкё° мң„н•ҙ мһ„мӢңлЎң мӮ¬мҡ©н•ҳлҠ” ліҖмҲҳл“Ө
  • н•ЁмҲҳ к°қмІҙк°Җ л©”мҶҢл“ңлЎңм„ң нҳём¶ңлҗҳм—Ҳмқ„ л•Ң мӮ¬мҡ©н•  мҲҳ мһҲлҠ” this м°ёмЎ°

н•ЁмҲҳвҖӢ

н•ЁмҲҳ к°қмІҙвҖӢ

  • н•ЁмҲҳ к°қмІҙлҠ” лӢӨлҘё мқјл°ҳм Ғмқё к°қмІҙмҷҖ л§Ҳм°¬к°Җм§ҖлЎң мҶҚм„ұмқ„ к°–кі  мһҲмңјл©° ліҖкІҪ к°ҖлҠҘн•ҳлӢӨ.
  • кіөмң лҗҳлҠ” ліҖкІҪ к°ҖлҠҘн•ң н•ЁмҲҳ к°қмІҙк°Җ м·Ём•Ҫм җмңјлЎң мӮ¬мҡ©лҗ  мҲҳ мһҲлӢӨ. (н”„лЎңнҶ нғҖмһ… мҳӨм—ј л“ұ)
  • н•ЁмҲҳ к°қмІҙлҠ” prototype мқҙлһҖ мҶҚм„ұмқ„ к°Җм§Җкі  мһҲкі , prototypeмқҖ Object.prototypeм—җ лҢҖн•ң delegation linkмҷҖ constructor мҶҚм„ұмқ„ к°Җм§„ к°қмІҙмқҳ м°ёмЎ°лҘј к°Җм§„лӢӨ.
    • constructor мҶҚм„ұмқҖ н•ЁмҲҳ к°қмІҙм—җ лҢҖн•ң м—ӯм°ёмЎ°лҘј к°Җм§Җкі  мһҲлӢӨ.
  • н•ЁмҲҳ к°қмІҙлҠ” Function.prototype м—җ лҢҖн•ң лҚёлҰ¬кІҢмқҙм…ҳ л§ҒнҒ¬лҘј к°Җм§Җкі  мһҲм–ҙ applyмҷҖ callмқ„ мғҒмҶҚл°ӣлҠ”лӢӨ.
  • н•ЁмҲҳ к°қмІҙм—җлҠ” л‘җ к°Җм§Җ мҲЁкІЁм§„ мҶҚм„ұмқҙ мһҲлҠ”лҚ° лӢӨмқҢкіј к°ҷлӢӨ.
    • н•ЁмҲҳ мӢӨн–ү мҪ”л“ңм—җ лҢҖн•ң м°ёмЎ°
    • н•ЁмҲҳ к°қмІҙк°Җ мғқм„ұлҗҳлҠ” мӢңм җм—җ нҷңм„ұнҷ”лҗң нҷңм„ұ к°қмІҙм—җ лҢҖн•ң м°ёмЎ° нҒҙлЎңм ёлҘј мӮ¬мҡ©н•  мҲҳ мһҲкІҢ н•ҙмӨҖлӢӨ.

н•ЁмҲҳ нҳём¶ңвҖӢ

  • нҢҢлқјлҜён„° н‘ңнҳ„мӢқ кі„мӮ°
  • н•ЁмҲҳмқҳ л§Өк°ңліҖмҲҳмҷҖ ліҖмҲҳлҘј м ҖмһҘн•  мҲҳ мһҲлҠ” м¶©л¶„н•ң нҒ¬кё°мқҳ нҷңм„ұ к°қмІҙ мғқм„ұ
  • нҳём¶ңлҗң н•ЁмҲҳ к°қмІҙм—җ лҢҖн•ң м°ёмЎ°лҘј мғҲлЎңмҡҙ нҷңм„ұ к°қмІҙм—җ м ҖмһҘ
  • м „лӢ¬л°ӣмқҖ нҢҢлқјлҜён„°лҘј мғҲлЎңмҡҙ нҷңм„ұ к°қмІҙмқҳ л§Өк°ңліҖмҲҳм—җ м ҖмһҘ
    • л№ м§„ нҢҢлқјлҜён„°лҠ” undefinedлЎң к°„мЈј, лӮЁлҠ” нҢҢлқјлҜён„°лҠ” лІ„лҰј
  • нҷңм„ұ к°қмІҙмқҳ лӘЁл“  ліҖмҲҳ к°’мқ„ undefinedлЎң н• лӢ№
  • н•ЁмҲҳ нҳём¶ң лӘ…л №м–ҙмқҳ л°”лЎң лӢӨмқҢ лӘ…л №м–ҙлҘј нҷңм„ұ к°қмІҙмқҳ next instruction н•„л“ң к°’мңјлЎң н• лӢ№
  • мғҲлЎңмҡҙ нҷңм„ұ к°қмІҙмқҳ caller н•„л“ң к°’м—җ нҳ„мһ¬ нҷңм„ұ к°қмІҙлҘј н• лӢ№
    • мӢӨм ң нҳём¶ң мҠӨнғқмқҙ м•„лӢҢ нҷңм„ұ к°қмІҙмқҳ м—°кІ°лҗң лӘ©лЎқмқҙлӢӨ.
  • мғҲлЎңмҡҙ нҷңм„ұ к°қмІҙлҘј нҳ„мһ¬ нҷңм„ұ к°қмІҙлЎң м§Җм •
  • нҳём¶ңлҗң н•ЁмҲҳ мӢӨн–ү

кј¬лҰ¬ мһ¬к·ҖвҖӢ

  • нҢҢлқјлҜён„° н‘ңнҳ„мӢқ кі„мӮ°
  • нҳ„мһ¬ нҷңм„ұ к°қмІҙк°Җ м¶©л¶„нһҲ нҒ¬лӢӨл©ҙ (мқјл°ҳм ҒмңјлЎң м¶©л¶„нһҲ нҒј)
    • нҳ„мһ¬ нҷңм„ұ к°қмІҙлҘј мғҲлЎңмҡҙ нҷңм„ұ к°қмІҙлЎң мӮ¬мҡ©
  • м•„лӢҢ кІҪмҡ°
    • н•ЁмҲҳ л§Өк°ңліҖмҲҳмҷҖ ліҖмҲҳлҘј м ҖмһҘн•  мҲҳ мһҲлҠ” м¶©л¶„н•ң нҒ¬кё°мқҳ нҷңм„ұ к°қмІҙ мғқм„ұ
    • мғҲлЎңмҡҙ нҷңм„ұ к°қмІҙмқҳ caller н•„л“ң к°’м—җ нҳ„мһ¬ нҷңм„ұ к°қмІҙ м ҖмһҘ
    • мғҲлЎңмҡҙ нҷңм„ұ к°қмІҙлҘј нҳ„мһ¬ нҷңм„ұ к°қмІҙлЎң м§Җм •
  • м „лӢ¬л°ӣмқҖ нҢҢлқјлҜён„°лҘј мғҲлЎңмҡҙ нҷңм„ұ к°қмІҙмқҳ л§Өк°ңліҖмҲҳм—җ м ҖмһҘ
    • л№ м§„ нҢҢлқјлҜён„°лҠ” undefinedлЎң к°„мЈј, лӮЁлҠ” нҢҢлқјлҜён„°лҠ” лІ„лҰј
  • нҷңм„ұ к°қмІҙмқҳ лӘЁл“  ліҖмҲҳ к°’мқ„ undefinedлЎң м§Җм •
  • нҳём¶ңлҗң н•ЁмҲҳлҘј мӢӨн–ү

нҷңм„ұ к°қмІҙлҘј мһ¬мӮ¬мҡ©н•ҳлҜҖлЎң мһ¬к·Җ н•ЁмҲҳ нҳём¶ңмқ„ л°ҳліөл¬ёл§ҢнҒј л№ лҘҙкІҢ мІҳлҰ¬ к°ҖлҠҘн•ҳлӢӨ. try лё”лЎқмқҳ кј¬лҰ¬ нҳём¶ңмқҖ мөңм Ғнҷ”лҗҳм§Җ м•ҠлҠ”лӢӨ.

  • tryлҠ” мӢӨн–ү нқҗлҰ„мқ„ callerмқҳ catchл¬ёмңјлЎң ліҖкІҪн•ҙм•јн•ҳлҠ”лҚ°, мқҙлҹ¬л©ҙ нҷңм„ұ к°қмІҙк°Җ мөңм Ғнҷ” лҗ  мҲҳ м—ҶмқҢ
  • лӮҙл¶Җм—җм„ң мғҲлЎңмҡҙ ліҖмҲҳлҘј к°Җм§Җкі  мһҲлӢӨл©ҙ нҷңм„ұ к°қмІҙк°Җ мөңм Ғнҷ” лҗ  мҲҳ м—ҶмқҢ

кј¬лҰ¬ нҳём¶ң мҳҲмӢңвҖӢ

; recursion
call func
return

; tail recursion
jump func
(function loop() {
  // do some
  if (done) {
    return;
  }

  // do more
  return loop();
})();
// вқҢ
function factorial(n) {
  if (n < 2) {
    return 1;
  }

  return n * factorial(n - 1);
}

// вқҢ
const value = func();
return value;

// вқҢ
// eslint-disable-next-line no-unreachable
return function () {};

// вң”пёҸ
function factorial2(n, result = 1) {
  if (n < 2) {
    return result;
  }

  return factorial2(n - 1, n * result);
}
// вң”пёҸ
// eslint-disable-next-line no-unreachable
return (function () {})();
