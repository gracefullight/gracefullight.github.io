  • Ж│авдйВё▒: Isolation, ьіИвъюВъГВЁў ВІцьќЅ ВцЉ ВцЉЖ░ё ВЌ░Вѓ░ Ж▓░Ж│╝віћ вІцвЦИ ьіИвъюВъГВЁў ВаЉЖи╝ вХѕЖ░ђ
  • ВўЂВєЇВё▒: Durability, Вё▒Ж│хВаЂВю╝вАю ВЎёвБївљю ьіИвъўВъГВЁўВЮђ Вќ┤вќц Ж│аВъЦВЌљвЈё ВєљВІц вљўВќ┤Вёювіћ ВЋѕвље

ьіИвъюВъГВЁў ВЃЂьЃю ВаёВЮ┤РђІ

  • ВІцьќЅ ВЃЂьЃю: Active
  • вХђвХё ВЎёвБї: Partially Committed
  • ВЎёвБї ВЃЂьЃю: Committed
  • ВІцьїе ВЃЂьЃю: Failed
  • В▓аьџї ВЃЂьЃю: Aborted

вЈЎВІюВё▒ ВаюВќ┤РђІ

вІцВцЉ ВѓгВџЕВъљ ьЎўЖ▓йВЮё ВДђВЏљьЋўвіћ вЇ░ВЮ┤ьё░в▓аВЮ┤Віц ВІюВіцьЁюВЌљВёю ВЌгвЪг ьіИвъюВъГВЁўвЊцВЮ┤ Вё▒Ж│хВаЂВю╝вАю вЈЎВІюВЌљ ВІцьќЅвља Вѕў ВъѕвЈёвАЮ ВДђВЏљьЋўвіћ ЖИ░віЦ

  • ьіИвъюВъГВЁў ВіцВ╝ђВЦ┤: Ж░Ђ ьіИвъюВъГВЁўВЮё ЖхгВё▒ьЋўвіћ ВЌ░Вѓ░вЊцВЮ┤ ВІюВіцьЁюВЌљВёю ВІюЖ░ёВЌљ вћ░вЮ╝ ВІцьќЅвљўвіћ ВѕюВёю
  • ВДЂвагЖ░ђвіЦВё▒: Serializability, ьіИвъюВъГВЁўвЊцВЮё в│ЉьќЅВ▓ўвдгьЋю Ж▓░Ж│╝Ж░ђ ьіИвъюВъГВЁўвЊцВЮё ВѕюВ░еВаЂВю╝вАю В▓ўвдгьЋю Ж▓░Ж│╝ВЎђ Ж░ЎВЮђ Ж▓Ѓ
  • Ж░▒ВІавХёВІц вгИВаю: Lost update problem
    • ьіИвъюВъГВЁўВЮ┤ вЈЎВЮ╝ вЇ░ВЮ┤ьё░вЦ╝ вЈЎВІюВЌљ Ж░▒ВІаьЋа Ж▓йВџ░ в░юВЃЮьЋўвіћ вгИВаю
    • ВЮ┤Ваё ьіИвъюВъГВЁўВЮ┤ вЇ░ВЮ┤ьё░вЦ╝ Ж░▒ВІаьЋю ьЏё ьіИвъюВъГВЁўВЮ┤ ВбЁвБїьЋўЖИ░ ВаёВЌљ ВЮ┤ьЏё ьіИвъюВъГВЁўВЮ┤ Ж░▒ВІа Ж░њВЮё вЇ«Вќ┤ ВЊ░віћ Ж▓йВџ░ ВўцвЦў в░юВЃЮ
  • вХѕВЮ╝В╣ў вгИВаю: Inconsistency analysis problem
    • вІцВцЉ ВѓгВџЕВъљ ьіИвъюВъГВЁўвЊцВЮ┤ вЈЎВІюВЌљ ВІцьќЅвља вЋї DBЖ░ђ ВЮ╝Ж┤ђВё▒ ВЌєвіћ вфеВѕювљю ВЃЂьЃювАю вѓевіћ вгИВаю
  • ВЌ░ВЄёв│хЖиђ вгИВаю: Cascading rollback problem
    • в│хВѕўВЮў ьіИвъюВъГВЁўВЮ┤ вЇ░ВЮ┤ьё░ Ж│хВюа ВІю ьі╣ВаЋ ьіИвъюВъГВЁўВЮ┤ В▓ўвдгвЦ╝ ВиеВєїьЋа Ж▓йВџ░ вІцвЦИ ьіИвъюВъГВЁўВЮ┤ В▓ўвдгьЋю вХђвХёВЌљ вїђьЋ┤ ВиеВєї вХѕЖ░ђвіЦ

вЈЎВІюВё▒ ВаюВќ┤ ЖИ░в▓ЋРђІ

  • Locking
    • ьіИвъюВъГВЁўВЮ┤ ВѓгВџЕьЋўвіћ ВъљВЏљВЌљ вїђьЋ┤ ВЃЂьўИв░░Ваю(Mutual Exclusive) ЖИ░віЦВЮё ВаюЖ│хьЋўвіћ ЖИ░в▓Ћ
    • ВЃЂьўИв░░Ваювіћ ьі╣ВаЋ ьіИвъюВъГВЁўВЮ┤ вЇ░ВЮ┤ьё░ ьЋГвфЕВЌљ вїђьЋ┤ вЮйВЮё ВёцВаЋьЋўвЕ┤ ВъаЖИѕВЮё ВёцВаЋьЋю ьіИвъюВъГВЁўВЮ┤ ВќИвЮйвља вЋїЖ╣їВДђ вЇ░ВЮ┤ьё░вЦ╝ вЈЁВаљВаЂВю╝вАю ВѓгВџЕ
  • Time-stamp Ordering
    • ьіИвъюВъГВЁўВЮ┤ ВІюВіцьЁюВЌљ ВъЁваЦвља вЋї ьЃђВъёВіцьЃгьћё вХђВЌг, ьіИвъЮЖ░юВЁўВЮў ВѕўьќЅ ВѕюВёювЦ╝ ВаюВќ┤
    • ьЃђВъёВіцьЃгьћё Ж░њВЮ┤ вѓўВцЉВЮИ ьіИвъюВъГВЁўВЮ┤ ьЃђВъёВіцьЃгьћё Ж░њВЮ┤ Вџ░ВёаВЮИ ьіИвъюВъГВЁўВЌљ ВаЉЖи╝ьЋўЖ│а Въѕвіћ вЇ░ВЮ┤ьё░ ьЋГвфЕВЌљ вїђьЋю ВаЉЖи╝ВЮё ВџћВ▓ГьЋўвіћ Ж▓йВџ░ вАцв░▒ вІ╣ьЋе
    • ВДЂвагВё▒ в│┤ВъЦ
    • вЇ░вЊювЮй в░ЕВДђ
  • вѓЎЖ┤ђВаЂ ЖИ░в▓Ћ
    • ьіИвъюВъГВЁўВЮ┤ ВѕўьќЅвљўвіћ вЈЎВЋѕ Вќ┤вќаьЋю Ж▓ђВѓгвЈё ВІцьќЅьЋўВДђ ВЋіВЮї
    • ьіИвъюВъГВЁў ВѕўьќЅ вДѕВДђвДЅВЌљ Ж░▒ВІа ВѓгьЋГвЊцВЮ┤ ВДЂвагЖ░ђвіЦВё▒ВЮё Вюёв░ўьЋўвіћ Ж▓йВџ░Ж░ђ ВъѕвіћВДђвДї Ж▓ђВѓгьЋўВЌг ВЮ┤ВЃЂВЮ┤ ВЌєВЮё Ж▓йВџ░ ьіИвъюВъГВЁўВЮё ьЎЋВаЋ
  • вѓЎЖ┤ђВаЂ ЖИ░в▓Ћ вІеЖ│ё
    • Read Phase: DBвЦ╝ ВЮйЖ│а Ж░юВЮИВаЂВЮИ в│хВѓгвАю Ж░▒ВІа, ьіИвъюВъГВЁў Ж░▒ВІаВЮђ вфевЉљ ВъёВІю Ж░▒ВІаьїїВЮ╝ВЌљ ЖИ░вАЮ, вІцвЦИ ьіИвъюВъГВЁўВЌљ ВЮўьЋ┤ ВаЉЖи╝ вХѕЖ░ђвіЦ
    • Validation Phase: Ж░ђьЋ┤ВДё в│ђьЎћЖ░ђ вЇ░ВЮ┤ьё░в▓аВЮ┤ВіцВЮў вг┤Ж▓░Вё▒Ж│╝ ВЮ╝Ж┤ђВё▒ВЌљ ВўЂьќЦВЮё ВБ╝ВДђ ВЋіВЮё Ж▓ЃВЮ┤вЮ╝віћ Ж▓ЃВЮё в│┤ВъЦьЋўЖИ░ ВюёьЋ┤ Ж▓ђВдЮвље, ВЮ┤ВЃЂ ВъѕВЮё Ж▓йВџ░ ьіИвъюВъГВЁў ВъгВІюВъЉ в░Ј вАцв░▒.
    • Write Phase: в│ђЖ▓йВѓгьЋГВЮ┤ ВўЂЖхгВаЂВю╝вАю DBВЌљ ВаЂВџЕвље

Locking ЖИ░в▓ЋРђІ

  • Shared Lock: Ж│хВюа ВъаЖИѕьЋю ьіИвъюВъГВЁўВЮђ вЇ░ВЮ┤ьё░ ьЋГвфЕВЌљ вїђьЋ┤ ВЮйЖИ░вДї Ж░ђвіЦ, вІцвЦИ ьіИвъюВъГВЁўвЈё ВЮйЖИ░ вДї ВІцьќЅ Ж░ђвіЦ
  • Exclusive Lock: ВаёВџЕ ВъаЖИѕьЋю ьіИвъюВъГВЁўВЮђ вЇ░ВЮ┤ьё░ ьЋГвфЕВЌљ вїђьЋ┤ ВЮйЖИ░ВЎђ ВЊ░ЖИ░ вфевЉљ Ж░ђвіЦ, вІцвЦИ ьіИвъюВъГВЁўВЮђ ВЮйЖИ░/ВЊ░ЖИ░ вХѕЖ░ђ.
  • DB -> Table -> Row -> Column: Locking overhead ВдЮЖ░ђ, ВъљвБї Ж│хВюаВё▒ вєњВЮї.

2 Phase LockingРђІ

вфевЊа ьіИвъюВъГВЁўВЮ┤ lockЖ│╝ unlock ВЌ░Вѓ░ВЮё ьЎЋВъЦвІеЖ│ёВЎђ ВѕўВХЋвІеЖ│ёвАю ЖхгвХёьЋўВЌг ВѕўьќЅ

  • ьЎЋВъЦвІеЖ│ё: ьіИвъюВъГВЁўВЮђ lockвДї ВѕўьќЅьЋа Вѕў ВъѕЖ│а, unlockВЮђ ВѕўьќЅьЋа Вѕў ВЌєвіћ вІеЖ│ё
  • ВѕўВХЋвІеЖ│ё: ьіИвъюВъГВЁўВЮђ unlockвДї ВѕўьќЅьЋа Вѕў ВъѕЖ│а, lockВЮђ ВѕўьќЅьЋа Вѕў ВЌєвіћ вІеЖ│ё
  • ВДЂвагЖ░ёВю╝Вё▒ВЮё в│┤ВъЦьЋа Вѕў Въѕвіћ ЖиюВЋйВю╝вАю Ж░ђВъЦ вДјВЮ┤ ВѓгВџЕвље.
  • ЖхљВ░ЕВЃЂьЃю в░юВЃЮвља Вѕў ВъѕВю╝вѓў Вўѕв░ЕЖ│╝ ьЃљВДђвАю ьЋ┤Ж▓░.

2 Phase CommitРђІ

вХёВѓ░ вЇ░ВЮ┤ьё░в▓аВЮ┤Віц ьЎўЖ▓йВЌљВёю ВЏљВъљВё▒ВЮё в│┤ВъЦьЋўЖИ░ ВюёьЋ┤ вХёВѓ░ ьіИвъюВъГВЁўВЌљ ьЈгьЋеьЋўвіћ вЕћВ╗цвІѕВдў PrepareВЎђ Commit вІеЖ│ёвЦ╝ ьєхьЋ┤ вфевЊа вЁИвЊюЖ░ђ В╗цв░ІьЋўВДђ ВЋіВю╝вЕ┤ вАцв░▒ьЋўвіћ ЖИ░в▓Ћ

  • Phase 1: Вцђв╣ё вІеЖ│ё
    • ьЋю ВДђВЌГвЁИвЊюВЌљВёю В╗цв░І ВџћЖхг, Global CoordinatorЖ░ђ Commit Point Site Ж▓░ВаЋ
    • Prepare Message вЦ╝ ВаёВєАьЋўЖ│а ВЏљЖ▓Е вЁИвЊювіћ Prepare Message Reply
  • Phase 2: В╗цв░І вІеЖ│ё
    • вфевЊа ВДђВЌГ вЁИвЊюЖ░ђ В╗цв░І Вцђв╣ёвљўВЌѕвІцвіћ Ж▓ЃВЮё в░ЏВЮђ Ж▓йВџ░ Ж░Ђ вЁИвЊюВЌљ CommitВЮё вфЁва╣ьЋе
    • Global CoordinatorЖ░ђвІцвЦИ ВДђВЌГ вЁИвЊювАювХђьё░ ВЌљвЪг в│┤Ж│авЦ╝ в░ЏВЮё ВІю вАцв░▒ вфЁва╣

DB ьџїв│хРђІ

ВъЦВЋа в░юВЃЮВІю вЇ░ВЮ┤ьё░в▓аВЮ┤ВіцвЦ╝ ВъЦВЋа в░юВЃЮ ВЮ┤Ваё вЇ░ВЮ┤ьё░в▓аВЮ┤ВіцВЮў вѓ┤ВџЕВЌљ вфеВѕюВЮ┤ ВЌєвіћ ВЮ╝Ж┤ђвљю ВЃЂьЃювАю в│хВЏљВІюьѓцвіћ Ж▓ЃВЮё

ВъЦВЋа ВюаьўЋРђІ

  • ьіИвъюВъГВЁў: Transation Failure, Въўвф╗вљю ВъЁваЦ, вЇ░ВЮ┤ьё░вЦ╝ В░ЙВЮё Вѕў ВЌєвіћЖ▓йВџ░, Вўцв▓ёьћївАюВџ░, Ж░ђВџЕВъљВЏљ В┤ѕЖ│╝ ВџћВ▓Г вЊ▒Вю╝вАю ьіИвъюВъГВЁў вЈЎВъЉ вХѕЖ░ђ
  • ВІюВіцьЁю: System Failure, ЖхљВ░ЕВЃЂьЃювАю ьіИвъюВъГВЁўВЮ┤ ВъЉВЌЁ вХѕЖ░ђ, ьЋўвЊюВЏеВќ┤ ВўцВъЉвЈЎВю╝вАю ьюўв░юВё▒ ЖИ░Вќ┤ВъЦВ╣ўВЮў вЇ░ВЮ┤ьё░ ВєљВІц
  • в»ИвћћВќ┤: Media Failure, в╣ёьюўв░юВё▒ ЖИ░ВќхВъЦВ╣ў(ьЋўвЊювћћВі╝) Ж│аВъЦВю╝вАю вћћВіцьЂг вИћвАЮВЮ┤ вХЋЖ┤┤

ьџїв│х ВџћЖ▒┤РђІ

  • вЇцьћё: вЇ░ВЮ┤ьё░в▓аВЮ┤Віц вѓ┤ВџЕВЮё ВЮ╝ВаЋЖИ░Ж░ёвДѕвІц вІцвЦИ ВађВъЦВєїВЌљ ВађВъЦ, в░▒ВЌЁВаЋВ▒ЁВЌљ ВЮўЖ▒░ьЋўВЌг вІцвЦИ вћћВіцьЂг вўљвіћ ьі╣ВаЋ Ж░ђВџЕ Ж│хЖ░ёВЌљ в│хВѓг
  • вАюЖиИ: вЇ░ВЮ┤ьё░в▓аВЮ┤Віц вѓ┤ВџЕВЮ┤ в│ђЖ▓йвља вЋївДѕвІц в│ђЖ▓й вѓ┤ВџЕВЮё вАюЖиИьїїВЮ╝ВЌљ ВІцВІюЖ░ё ВађВъЦ

ьџїв│х ЖИ░в▓ЋРђІ

  • ВДђВЌ░Ж░▒ВІа
    • ьіИвъюВъГВЁўВЮ┤ вХђвХёВЎёвБї ВЃЂьЃюВЌљ ВЮ┤вЦ┤ЖИ░ ВаёЖ╣їВДђ в░юВЃЮьЋю вфевЊа в│ђЖ▓йвѓ┤ВџЕВЮё вАюЖиИьїїВЮ╝ВЌљ ЖИ░вАЮьЋўЖ│а, DBВЌљ ВађВъЦьЋўвіћ Ж▓ЃВЮё ВДђВЌ░ВІюьѓцвіћ в░Ев▓Ћ
    • ьіИвъюВъГВЁў ВІцьїеВІю вАюЖиИьїїВЮ╝ВЮў вѓ┤ВџЕвДїВЮё ьЈљЖИ░ьЋўвЕ┤ вљўв»ђвАю UndoьЋёВџћВЌєВЮї
  • ВдЅВІюЖ░▒ВІа
    • в│ђЖ▓йвѓ┤ВџЕВЮё DBВЎђ вАюЖиИьїїВЮ╝ВЌљ вЈЎВІюВЌљ ВађВъЦ
    • ьіИвъюВъГВЁў ВѕўьќЅ вЈёВцЉ ВІцьїеьЋю Ж▓йВџ░ вАюЖиИьїїВЮ╝ВЮў вѓ┤ВџЕВЮё В░ИВА░ьЋўВЌг в│ђЖ▓й ВЮ┤Ваё Ж░њВю╝вАю ВѕўВаЋьЋўвіћ UndoЖ│╝ВаЋ ьЋёВџћ
  • Ж▓ђВѓгВаљ: Check Point
    • в│ђЖ▓йвѓ┤ВџЕВЮё вАюЖиИьїїВЮ╝ВЌљ ЖИ░вАЮьЋўЖ│а, ВЮ╝ВаЋВІюЖ░ёвДѕвІц В▓┤ьЂгьЈгВЮИьіИвЦ╝ в░юВЃЮВІюВ╝ю ЖиИвЋїЖ╣їВДђВЮў в│ђЖ▓йВѓгьЋГВЮё вћћВіцьЂгВЌљ ВађВъЦ, вАюЖиИьїїВЮ╝ВЌљвіћ В▓┤ьЂгьЈгВЮИьіИвЦ╝ ЖИ░вАЮ
    • ВъЦВЋаВІю В▓┤ьЂгьЈгВЮИьіИ ВЮ┤Ваё В▓ўвдгвљю ьіИвъюВъГВЁўВЮђ ьџїв│хВъЉВЌЁВЌљВёю ВаюВЎИьЋўЖ│а, ВЮ┤ьЏё В▓ўвдгвљю ьіИвъюВъГВЁўВЌљ вїђьЋ┤ВёювДї Redo вўљвіћ Undo ВѕўьќЅ.
    • Check point interval Ж┤ђвдг ьЋёВџћ
  • ЖиИвд╝ВъљьјўВЮ┤ВДђ: Shadow Page Table
