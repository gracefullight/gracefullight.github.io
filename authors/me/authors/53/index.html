мқҙмғҒнҳ„мғҒмқ„ м•јкё°н•ҳлҠ” мҶҚм„ұ к°„мқҳ мў…мҶҚм„ұмқ„ м ңкұ°н•ҳкё° мң„н•ҙ лҰҙл Ҳмқҙм…ҳмқ„ мһ‘мқҖ м—¬лҹ¬ лҰҙл Ҳмқҙм…ҳмңјлЎң л¬ҙмҶҗмӢӨ л¶„н•ҙн•ҳлҠ” кіјм • лҚ°мқҙн„° мқјкҙҖм„ұ, мӨ‘ліөмқҳ мөңмҶҢнҷ”, кө¬мЎ°мқҳ м•Ҳм •м„ұмқ„ мөңлҢҖн•ң нҷ•ліҙн•ҳкё° мң„н•ң л°©лІ•

к°Ғ лӢЁкі„к°Җ лҒқлӮҳл©ҙ Nм°Ё м •к·ңнҳ•мқҙ лҗңлӢӨ.

  • 1м°Ё м •к·ңнҷ”: лҸ„л©”мқё л¶„н•ҙ
  • 2м°Ё м •к·ңнҷ”: л¶Җл¶„н•ЁмҲҳ мў…мҶҚм„ұ м ңкұ°
  • 3м°Ё м •к·ңнҷ”: мқҙн–үн•ЁмҲҳ мў…мҶҚм„ұ м ңкұ°
  • BCNF м •к·ңнҷ”: лӮҳмӨ‘м—җ м¶”к°ҖлҗЁ
  • 4м°Ё м •к·ңнҷ”: лӢӨм№ҳмў…мҶҚм„ұ м ңкұ°
  • 5м°Ё м •к·ңнҷ”: мЎ°мқёмў…мҶҚм„ұ м ңкұ°

м—°кІ°н•Ём •вҖӢ

лҚ°мқҙн„° лӘЁлҚёл§Ғ мӢң к°ңмІҙмҷҖ к°ңмІҙ мӮ¬мқҙм—җ л¶Җм—¬н•ҳлҠ” кҙҖкі„м„ұ м§‘н•©мқҳ мқҳлҜёк°Җ лӘЁнҳён•ҳм—¬ мӣҗн•ҳлҠ” кІ°кіјлҘј м–»мқ„ мҲҳ м—Ҷкұ°лӮҳ н–Ҙнӣ„ м—…л¬ҙмІҳлҰ¬м—җ мҳҒн–Ҙмқ„ лҜём№ҳкІҢ лҗҳлҠ” ERлӘЁлҚёмқҳ л¬ём ңм җ

  • л¶Җмұ„кјҙ н•Ём •: Fan Trap, к°ңмІҙм§‘н•©(Entity Set) мӮ¬мқҙм—җ кҙҖкі„м„ұ м§‘н•©(Relation Set)мқҙ м •мқҳлҗҳм–ҙ мһҲкё°лҠ”н•ҳм§Җл§Ң кҙҖкі„м„ұ мҳҲмӢңк°Җ лӘЁнҳён•ң кІҪмҡ°
  • к· м—ҙ н•Ём •: Chasm Trap, к°ңмІҙм§‘н•© мӮ¬мқҙм—җ кҙҖкі„м„ұ м§‘н•©мқҙ м •мқҳлҗҳм–ҙ мһҲкё°лҠ” н•ҳм§Җл§Ң мқјл¶Җ к°ңмІҙм§‘н•©кіј к°ңмІҙм§‘н•© мӮ¬мқҙм—җ кҙҖкі„м„ұмқҙ мЎҙмһ¬н•ҳм§Җ м•ҠлҠ” кІҪмҡ°

л°ҳм •к·ңнҷ”вҖӢ

м •к·ңнҷ”лҗң м—”нӢ°нӢ° нғҖмһ…, мҶҚм„ұ, кҙҖкі„лҘј мӢңмҠӨн…ңмқҳ м„ұлҠҘн–ҘмғҒ, к°ңл°ңкіј мҡҙмҳҒмқҳ нҺёмқҳлҘј мң„н•ҙ лҚ°мқҙн„° лӘЁлҚёмқ„ нҶөн•©н•ҳлҠ” н”„лЎңм„ёмҠӨ

DB мҡҙмҳҒвҖӢ

нҠёлһңмһӯм…ҳвҖӢ

лҚ°мқҙн„°мқҳ м •нҷ•н•ң мқјкҙҖм„ұкіј л¬ҙкІ°м„ұмқ„ ліҙмһҘн•ҳкё° мң„н•ҙ мҷ„м „нһҲ мў…лЈҢн•ҙм•јн•ҳлҠ” лҚ°мқҙн„°лІ мқҙмҠӨ мІҳлҰ¬мқҳ л…јлҰ¬м Ғ мһ‘м—… лӢЁмң„(Logical Unit of Work)

  • мӣҗмһҗм„ұ: Atomicity, л¶„н•ҙк°Җ л¶Ҳк°ҖлҠҘн•ң мҲҳн–ү лӢЁмң„лЎң мҷ„м „нһҲ мҲҳн–ү лҳҗлҠ” мҲҳн–үлҗҳм§Җ м•ҠмқҢ
  • мқјкҙҖм„ұ: Consistency, нҠёлһңмһӯм…ҳмқҙ м„ұкіөм ҒмңјлЎң мҷ„лЈҢлҗҳл©ҙ м–ём ңлӮҳ лӘЁмҲңмқҙ м—ҶлҠ” мғҒнғң мң м§Җ, л¬ҙкІ°м„ұ м ңм•Ҫ мЎ°кұҙ
  • кі лҰҪм„ұ: Isolation, нҠёлһңмһӯм…ҳ мӢӨн–ү мӨ‘ мӨ‘к°„ м—°мӮ° кІ°кіјлҠ” лӢӨлҘё нҠёлһңмһӯм…ҳ м ‘к·ј л¶Ҳк°Җ
  • мҳҒмҶҚм„ұ: Durability, м„ұкіөм ҒмңјлЎң мҷ„лЈҢлҗң нҠёлһҳмһӯм…ҳмқҖ м–ҙл–Ө кі мһҘм—җлҸ„ мҶҗмӢӨ лҗҳм–ҙм„ңлҠ” м•ҲлҗЁ

нҠёлһңмһӯм…ҳ мғҒнғң м „мқҙвҖӢ

  • мӢӨн–ү мғҒнғң: Active
  • л¶Җл¶„ мҷ„лЈҢ: Partially Committed
  • мҷ„лЈҢ мғҒнғң: Committed
  • мӢӨнҢЁ мғҒнғң: Failed
  • мІ нҡҢ мғҒнғң: Aborted

лҸҷмӢңм„ұ м ңм–ҙвҖӢ

лӢӨмӨ‘ мӮ¬мҡ©мһҗ нҷҳкІҪмқ„ м§Җмӣҗн•ҳлҠ” лҚ°мқҙн„°лІ мқҙмҠӨ мӢңмҠӨн…ңм—җм„ң м—¬лҹ¬ нҠёлһңмһӯм…ҳл“Өмқҙ м„ұкіөм ҒмңјлЎң лҸҷмӢңм—җ мӢӨн–үлҗ  мҲҳ мһҲлҸ„лЎқ м§Җмӣҗн•ҳлҠ” кё°лҠҘ

  • нҠёлһңмһӯм…ҳ мҠӨмјҖмҘҙ: к°Ғ нҠёлһңмһӯм…ҳмқ„ кө¬м„ұн•ҳлҠ” м—°мӮ°л“Өмқҙ мӢңмҠӨн…ңм—җм„ң мӢңк°„м—җ л”°лқј мӢӨн–үлҗҳлҠ” мҲңм„ң
  • м§Ғл ¬к°ҖлҠҘм„ұ: Serializability, нҠёлһңмһӯм…ҳл“Өмқ„ лі‘н–үмІҳлҰ¬н•ң кІ°кіјк°Җ нҠёлһңмһӯм…ҳл“Өмқ„ мҲңм°Ём ҒмңјлЎң мІҳлҰ¬н•ң кІ°кіјмҷҖ к°ҷмқҖ кІғ
  • к°ұмӢ л¶„мӢӨ л¬ём ң: Lost update problem
    • нҠёлһңмһӯм…ҳмқҙ лҸҷмқј лҚ°мқҙн„°лҘј лҸҷмӢңм—җ к°ұмӢ н•  кІҪмҡ° л°ңмғқн•ҳлҠ” л¬ём ң
    • мқҙм „ нҠёлһңмһӯм…ҳмқҙ лҚ°мқҙн„°лҘј к°ұмӢ н•ң нӣ„ нҠёлһңмһӯм…ҳмқҙ мў…лЈҢн•ҳкё° м „м—җ мқҙнӣ„ нҠёлһңмһӯм…ҳмқҙ к°ұмӢ  к°’мқ„ лҚ®м–ҙ м“°лҠ” кІҪмҡ° мҳӨлҘҳ л°ңмғқ
  • л¶Ҳмқјм№ҳ л¬ём ң: Inconsistency analysis problem
    • лӢӨмӨ‘ мӮ¬мҡ©мһҗ нҠёлһңмһӯм…ҳл“Өмқҙ лҸҷмӢңм—җ мӢӨн–үлҗ  л•Ң DBк°Җ мқјкҙҖм„ұ м—ҶлҠ” лӘЁмҲңлҗң мғҒнғңлЎң лӮЁлҠ” л¬ём ң
  • м—°мҮ„ліөк·Җ л¬ём ң: Cascading rollback problem
    • ліөмҲҳмқҳ нҠёлһңмһӯм…ҳмқҙ лҚ°мқҙн„° кіөмң  мӢң нҠ№м • нҠёлһңмһӯм…ҳмқҙ мІҳлҰ¬лҘј м·ЁмҶҢн•  кІҪмҡ° лӢӨлҘё нҠёлһңмһӯм…ҳмқҙ мІҳлҰ¬н•ң л¶Җл¶„м—җ лҢҖн•ҙ м·ЁмҶҢ л¶Ҳк°ҖлҠҘ

лҸҷмӢңм„ұ м ңм–ҙ кё°лІ•вҖӢ

  • Locking
    • нҠёлһңмһӯм…ҳмқҙ мӮ¬мҡ©н•ҳлҠ” мһҗмӣҗм—җ лҢҖн•ҙ мғҒнҳёл°°м ң(Mutual Exclusive) кё°лҠҘмқ„ м ңкіөн•ҳлҠ” кё°лІ•
    • мғҒнҳёл°°м ңлҠ” нҠ№м • нҠёлһңмһӯм…ҳмқҙ лҚ°мқҙн„° н•ӯлӘ©м—җ лҢҖн•ҙ лқҪмқ„ м„Өм •н•ҳл©ҙ мһ кёҲмқ„ м„Өм •н•ң нҠёлһңмһӯм…ҳмқҙ м–ёлқҪлҗ  л•Ңк№Ңм§Җ лҚ°мқҙн„°лҘј лҸ…м җм ҒмңјлЎң мӮ¬мҡ©
  • Time-stamp Ordering
    • нҠёлһңмһӯм…ҳмқҙ мӢңмҠӨн…ңм—җ мһ…л Ҙлҗ  л•Ң нғҖмһ„мҠӨнғ¬н”„ л¶Җм—¬, нҠёлһқк°ңм…ҳмқҳ мҲҳн–ү мҲңм„ңлҘј м ңм–ҙ
    • нғҖмһ„мҠӨнғ¬н”„ к°’мқҙ лӮҳмӨ‘мқё нҠёлһңмһӯм…ҳмқҙ нғҖмһ„мҠӨнғ¬н”„ к°’мқҙ мҡ°м„ мқё нҠёлһңмһӯм…ҳм—җ м ‘к·јн•ҳкі  мһҲлҠ” лҚ°мқҙн„° н•ӯлӘ©м—җ лҢҖн•ң м ‘к·јмқ„ мҡ”мІӯн•ҳлҠ” кІҪмҡ° лЎӨл°ұ лӢ№н•Ё
    • м§Ғл ¬м„ұ ліҙмһҘ
    • лҚ°л“ңлқҪ л°©м§Җ
  • лӮҷкҙҖм Ғ кё°лІ•
    • нҠёлһңмһӯм…ҳмқҙ мҲҳн–үлҗҳлҠ” лҸҷм•Ҳ м–ҙл– н•ң кІҖмӮ¬лҸ„ мӢӨн–үн•ҳм§Җ м•ҠмқҢ
    • нҠёлһңмһӯм…ҳ мҲҳн–ү л§Ҳм§Җл§үм—җ к°ұмӢ  мӮ¬н•ӯл“Өмқҙ м§Ғл ¬к°ҖлҠҘм„ұмқ„ мң„л°ҳн•ҳлҠ” кІҪмҡ°к°Җ мһҲлҠ”м§Җл§Ң кІҖмӮ¬н•ҳм—¬ мқҙмғҒмқҙ м—Ҷмқ„ кІҪмҡ° нҠёлһңмһӯм…ҳмқ„ нҷ•м •
  • лӮҷкҙҖм Ғ кё°лІ• лӢЁкі„
    • Read Phase: DBлҘј мқҪкі  к°ңмқём Ғмқё ліөмӮ¬лЎң к°ұмӢ , нҠёлһңмһӯм…ҳ к°ұмӢ мқҖ лӘЁл‘җ мһ„мӢң к°ұмӢ нҢҢмқјм—җ кё°лЎқ, лӢӨлҘё нҠёлһңмһӯм…ҳм—җ мқҳн•ҙ м ‘к·ј л¶Ҳк°ҖлҠҘ
    • Validation Phase: к°Җн•ҙм§„ ліҖнҷ”к°Җ лҚ°мқҙн„°лІ мқҙмҠӨмқҳ л¬ҙкІ°м„ұкіј мқјкҙҖм„ұм—җ мҳҒн–Ҙмқ„ мЈјм§Җ м•Ҡмқ„ кІғмқҙлқјлҠ” кІғмқ„ ліҙмһҘн•ҳкё° мң„н•ҙ кІҖмҰқлҗЁ, мқҙмғҒ мһҲмқ„ кІҪмҡ° нҠёлһңмһӯм…ҳ мһ¬мӢңмһ‘ л°Ҹ лЎӨл°ұ.
    • Write Phase: ліҖкІҪмӮ¬н•ӯмқҙ мҳҒкө¬м ҒмңјлЎң DBм—җ м Ғмҡ©лҗЁ

Locking кё°лІ•вҖӢ

  • Shared Lock: кіөмң  мһ кёҲн•ң нҠёлһңмһӯм…ҳмқҖ лҚ°мқҙн„° н•ӯлӘ©м—җ лҢҖн•ҙ мқҪкё°л§Ң к°ҖлҠҘ, лӢӨлҘё нҠёлһңмһӯм…ҳлҸ„ мқҪкё° л§Ң мӢӨн–ү к°ҖлҠҘ
  • Exclusive Lock: м „мҡ© мһ кёҲн•ң нҠёлһңмһӯм…ҳмқҖ лҚ°мқҙн„° н•ӯлӘ©м—җ лҢҖн•ҙ мқҪкё°мҷҖ м“°кё° лӘЁл‘җ к°ҖлҠҘ, лӢӨлҘё нҠёлһңмһӯм…ҳмқҖ мқҪкё°/м“°кё° л¶Ҳк°Җ.
  • DB -> Table -> Row -> Column: Locking overhead мҰқк°Җ, мһҗлЈҢ кіөмң м„ұ лҶ’мқҢ.

2 Phase LockingвҖӢ

лӘЁл“  нҠёлһңмһӯм…ҳмқҙ lockкіј unlock м—°мӮ°мқ„ нҷ•мһҘлӢЁкі„мҷҖ мҲҳм¶•лӢЁкі„лЎң кө¬л¶„н•ҳм—¬ мҲҳн–ү

  • нҷ•мһҘлӢЁкі„: нҠёлһңмһӯм…ҳмқҖ lockл§Ң мҲҳн–үн•  мҲҳ мһҲкі , unlockмқҖ мҲҳн–үн•  мҲҳ м—ҶлҠ” лӢЁкі„
  • мҲҳм¶•лӢЁкі„: нҠёлһңмһӯм…ҳмқҖ unlockл§Ң мҲҳн–үн•  мҲҳ мһҲкі , lockмқҖ мҲҳн–үн•  мҲҳ м—ҶлҠ” лӢЁкі„
  • м§Ғл ¬к°„мңјм„ұмқ„ ліҙмһҘн•  мҲҳ мһҲлҠ” к·ңм•ҪмңјлЎң к°ҖмһҘ л§Һмқҙ мӮ¬мҡ©лҗЁ.
  • көҗм°©мғҒнғң л°ңмғқлҗ  мҲҳ мһҲмңјлӮҳ мҳҲл°©кіј нғҗм§ҖлЎң н•ҙкІ°.

2 Phase CommitвҖӢ

л¶„мӮ° лҚ°мқҙн„°лІ мқҙмҠӨ нҷҳкІҪм—җм„ң мӣҗмһҗм„ұмқ„ ліҙмһҘн•ҳкё° мң„н•ҙ л¶„мӮ° нҠёлһңмһӯм…ҳм—җ нҸ¬н•Ён•ҳлҠ” л©”м»ӨлӢҲмҰҳ PrepareмҷҖ Commit лӢЁкі„лҘј нҶөн•ҙ лӘЁл“  л…ёл“ңк°Җ м»Өл°Ӣн•ҳм§Җ м•Ҡмңјл©ҙ лЎӨл°ұн•ҳлҠ” кё°лІ•

  • Phase 1: мӨҖл№„ лӢЁкі„
    • н•ң м§Җм—ӯл…ёл“ңм—җм„ң м»Өл°Ӣ мҡ”кө¬, Global Coordinatorк°Җ Commit Point Site кІ°м •
    • Prepare Message лҘј м „мҶЎн•ҳкі  мӣҗкІ© л…ёл“ңлҠ” Prepare Message Reply
  • Phase 2: м»Өл°Ӣ лӢЁкі„
    • лӘЁл“  м§Җм—ӯ л…ёл“ңк°Җ м»Өл°Ӣ мӨҖл№„лҗҳм—ҲлӢӨлҠ” кІғмқ„ л°ӣмқҖ кІҪмҡ° к°Ғ л…ёл“ңм—җ Commitмқ„ лӘ…л №н•Ё
    • Global Coordinatorк°ҖлӢӨлҘё м§Җм—ӯ л…ёл“ңлЎңл¶Җн„° м—җлҹ¬ ліҙкі лҘј л°ӣмқ„ мӢң лЎӨл°ұ лӘ…л №

DB нҡҢліөвҖӢ

мһҘм•  л°ңмғқмӢң лҚ°мқҙн„°лІ мқҙмҠӨлҘј мһҘм•  л°ңмғқ мқҙм „ лҚ°мқҙн„°лІ мқҙмҠӨмқҳ лӮҙмҡ©м—җ лӘЁмҲңмқҙ м—ҶлҠ” мқјкҙҖлҗң мғҒнғңлЎң ліөмӣҗмӢңнӮӨлҠ” кІғмқ„
