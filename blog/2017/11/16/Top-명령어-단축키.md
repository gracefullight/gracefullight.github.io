`top` 명령어는 자주 치는 명령어지만 예쁘게 소팅하기란 쉽지 않다.

# 단축키

| 단축키   | 내용                                           |
| -------- | ---------------------------------------------- |
| spacebar | 화면 즉시 업데이트                             |
| M        | 현재 상주 메모리 사용량에 따라 분류            |
| T        | 전체 누적 CPU 사용량에 따라 분류               |
| P        | 현재 CPU 사용량(디폴트 값)에 따라 분류         |
| c        | 프로세스 경로 표시                             |
| u        | 오로지 한 사용자의 프로세스만 보여줌           |
| f        | 다른 종류의 통계 자료가 나타나도록 선택        |
| ?        | 모든 top 명령에 대한 사용법의 개요를 보여준다. |
