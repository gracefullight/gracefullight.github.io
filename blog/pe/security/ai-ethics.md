| --- | --- | --- |

| - | 프라이버시 보호 | 사생활 보호, 개인정보 오용 최소화 |

| - | 공공성 | 인류 공통 이익 중시, 순기능 극대화 |
| - | 연대성 | 이해관계자 참여 기회 보장 |
| 기술합목적성 | 데이터 관리 | 목적 외의 용도 사용 금지, 품질 관리 |
| - | 책임성 | 책임 명확화, 주체별 책임 분산 |
| - | 안전성 | 잠재적 위험 방지, 안전 보장 활동 |
| - | 투명성 | 설명 가능성 확보, 유의사항 고지 |
