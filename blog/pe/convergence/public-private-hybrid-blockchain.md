## I. 블록체인 개요

- 개념
  - 탈중앙화된 P2P 네트워크를 기반에 합의알고리즘을 통해 무결성을 제공하는 분산 원장 기술
- 배경

| 도입기        | 발전기        | 확산기        | 정착기                    |
| ------------- | ------------- | ------------- | ------------------------- |
| 디지털자산    | 스마트 계약   | 프로토콜      | 암호화폐/결제             |
| 거래 결제     | 지분증명      | 인터페이스    | 블록체인 기반 서비스 확대 |
| 비트코인 탄생 | 이더리움 탄생 | 알트코인 탄생 | -                         |
| Public        | Private       | Hybrid        | 융복합                    |
