## I. 개요

### 개념

```mermaid
graph TB;
    M[몽키테스트]
    CT["`Continuous Testing
    QCD확보`"]
    R[회귀테스트]

    CT-.신규커버리지.->M
    CT-.기존커버리지.->R
```

### 배경

애자일 개발 방법론 도입으로 소프트웨어의 잦은 변경으로 인하여 자동화되고 연속적인 테스트의 필요성이 증가.

기존 테스트로를 지속하면 살충제 패러독스가 발생하므로, 새로운 버그 발견을 위해 몽키테스트 실행.
