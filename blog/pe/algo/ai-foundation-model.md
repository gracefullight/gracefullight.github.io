## 인공지능 파운데이션 모델 개념

- 대량의 무라벨 데이터로 학습되어 목적에 맞는 다운스트림 작업에 적용될 수 있는 일반화된 기초 모델
- 데이터의 증가와 하드웨어의 연산 속도 증가로 다양한 작업에서 높은 성능을 보여주는 파운데이션 모델 등장

| 구분 | 요소 | 내용 |

| | 대규모 파라미터 | 최소 수억개 이상 파라미터 |
